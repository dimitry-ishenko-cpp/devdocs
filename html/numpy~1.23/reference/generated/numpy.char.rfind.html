numpy.char.rfind

char.rfind(a, sub, start=0, end=None)[source]

For each element in a, return the highest index in the string where substring sub is found, such that sub is contained within [start, end].

Calls str.rfind element-wise.

Parameters
aarray-like of str or unicode
substr or unicode
start, endint, optional

Optional arguments start and end are interpreted as in slice notation.

Returns
outndarray

Output array of ints. Return -1 on failure.
