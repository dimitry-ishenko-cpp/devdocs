love.graphics.hasFocus

Available since LÖVE 0.8.0 and removed in LÖVE 0.9.0
Moved to the love.window module as love.window.hasFocus.

Checks if the game window has keyboard focus.

Function

Synopsis

focus = love.graphics.hasFocus( )

Arguments

None.

Returns

boolean focus
True if the window has the focus or false if not.
