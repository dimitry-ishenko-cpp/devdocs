Thread:send

Available since LÖVE 0.7.0 and removed in LÖVE 0.8.0
It has been renamed to Thread:set.

Send a message (put it in the message box).

Function

Synopsis

Thread:send(name, value)

Arguments

string name
The name of the message.
Variant value
The contents of the message.

Returns

None.
