love.joystick.getAxis

Available since LÖVE 0.5.0 and removed in LÖVE 0.9.0
It has been moved to Joystick:getAxis.

Returns the direction of the axis.

Function

Synopsis

direction = love.joystick.getAxis( joystick, axis )

Arguments

number joystick
The joystick to be checked
number axis
The axis to be checked

Returns

number direction
Current value of the axis
