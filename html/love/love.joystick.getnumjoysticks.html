love.joystick.getNumJoysticks

Available since LÖVE 0.5.0 and removed in LÖVE 0.9.0
It has been renamed to love.joystick.getJoystickCount.


Returns how many joysticks are available.

Function

Synopsis

joysticks = love.joystick.getNumJoysticks( )

Arguments

None.

Returns

number joysticks
The number of joysticks available
