PulleyJoint:getMaxLengths

Get the maximum lengths of the rope segments.

Function

Synopsis

len1, len2 = PulleyJoint:getMaxLengths( )

Arguments

None.

Returns

number len1
The maximum length of the first rope segment.
number len2
The maximum length of the second rope segment.
