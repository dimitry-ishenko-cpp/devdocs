ParticleSystem:setSpin

Sets the spin of the sprite.

Function

Synopsis

ParticleSystem:setSpin( min, max )

Arguments

number min
The minimum spin (radians per second).
number max (min)
The maximum spin (radians per second).

Returns

Nothing.
