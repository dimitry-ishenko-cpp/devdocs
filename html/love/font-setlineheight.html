Font:setLineHeight

Sets the line height.

When rendering the font in lines the actual height will be determined by the line height multiplied by the height of the font. The default is 1.0.

Function

Synopsis

Font:setLineHeight( height )

Arguments

number height
The new line height.

Returns

Nothing.
