love.system.vibrate

Available since LÃ–VE 0.10.0
This function is not supported in earlier versions.

Causes the device to vibrate, if possible. Currently this will only work on Android and iOS devices that have a built-in vibration motor.

Function

Synopsis

love.system.vibrate( seconds )

Arguments

number seconds (0.5)
The duration to vibrate for. If called on an iOS device, it will always vibrate for 0.5 seconds due to limitations in the iOS system APIs.

Returns

Nothing.
