love.joystick.getNumHats

Available since LÖVE 0.5.0 and removed in LÖVE 0.9.0
It has been moved to Joystick:getHatCount.


Returns the number of hats on the joystick.

Function

Synopsis

hats = love.joystick.getNumHats( joystick )

Arguments

number joystick
The joystick to be checked

Returns

number hats
How many hats the joystick has
