love.thread.getThreads

Available since LÖVE 0.7.0 and removed in LÖVE 0.9.0
This function is not supported in earlier or later versions.

Get all threads.

Function

Synopsis

threads = love.thread.getThreads( )

Arguments

None.

Returns

table threads
A table containing all threads indexed by their names.
