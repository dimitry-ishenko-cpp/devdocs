Source:isStatic

Available since LÖVE 0.7.0 and removed in LÖVE 0.10.0
It has been replaced by Source:getType.

Returns whether the Source is static.

Function

Synopsis

static = Source:isStatic( )

Arguments

None.

Returns

boolean static
True if the Source is static, false otherwise.
