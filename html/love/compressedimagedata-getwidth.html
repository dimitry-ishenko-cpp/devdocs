CompressedImageData:getWidth

Available since LÃ–VE 0.9.0
This function is not supported in earlier versions.

Gets the width of the CompressedImageData.

Function

Synopsis

width = CompressedImageData:getWidth( )

Arguments

None.

Returns

number width
The width of the CompressedImageData.

Function

Synopsis

width = CompressedImageData:getWidth( level )

Arguments

number level
A mipmap level. Must be in the range of [1, CompressedImageData:getMipmapCount()].

Returns

number width
The width of a specific mipmap level of the CompressedImageData.
