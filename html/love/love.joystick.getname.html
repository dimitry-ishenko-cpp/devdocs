love.joystick.getName

Available since LÖVE 0.5.0 and removed in LÖVE 0.9.0
It has been moved to Joystick:getName.

Returns the name of a joystick.

Function

Synopsis

name = love.joystick.getName( joystick )

Arguments

number joystick
The joystick to be checked

Returns

string name
The name
