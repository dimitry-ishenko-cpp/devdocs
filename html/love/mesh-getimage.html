Mesh:getImage

Available since LÖVE 0.9.0
This function is not supported in earlier versions.

Removed in LÖVE 0.10.0
Use Mesh:getTexture instead.

Gets the Image used when drawing the Mesh.

Function

Synopsis

image = Mesh:getImage( )

Arguments

None.

Returns

Image image (nil)
The Image used to texture the Mesh when drawing. May be nil if no Image is being used with the Mesh.
