ParticleSystem:setRadialAcceleration

Set the radial acceleration (away from the emitter).

Function

Synopsis

ParticleSystem:setRadialAcceleration( min, max )

Arguments

number min
The minimum acceleration.
number max (min)
The maximum acceleration.

Returns

Nothing.
