love.joystick.getAxes

Available since LÖVE 0.5.0 and removed in LÖVE 0.9.0
It has been moved to Joystick:getAxes.

Returns the position of each axis.

Function

Synopsis

axisDir1, axisDir2, axisDirN = love.joystick.getAxes( joystick )

Arguments

number joystick
The joystick to be checked

Returns

number axisDir1
Direction of axis1
number axisDir2
Direction of axis2
number axisDirN
Direction of axisN
