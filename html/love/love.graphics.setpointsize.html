love.graphics.setPointSize

Sets the point size.

The sizes of points are not affected by love.graphics.scale â€“ they're always in pixels, or since 11.0, in DPI-scaled units.

Function

Synopsis

love.graphics.setPointSize( size )

Arguments

number size
The new point size.

Returns

Nothing.

Examples

Increase the point size by 1, by using love.graphics.getPointSize. 

function love.draw()
    local s = love.graphics.getPointSize() + 1
    love.graphics.setPointSize(s)
    love.graphics.points(100, 100)
end
