love.audio.setOrientation

Sets the orientation of the listener.

Function

Synopsis

love.audio.setOrientation( fx, fy, fz, ux, uy, uz )

Arguments

number fx, fy, fz
Forward vector of the listener orientation.
number ux, uy, uz
Up vector of the listener orientation.

Returns

Nothing.
