love.joystick.close

Available since LÖVE 0.5.0 and removed in LÖVE 0.9.0
This function is not supported in earlier or later versions.

Closes a joystick, i.e. stop using it for generating events and in query functions.

Function

Synopsis

love.joystick.close( joystick )

Arguments

number joystick
The joystick to be closed

Returns

Nothing.
