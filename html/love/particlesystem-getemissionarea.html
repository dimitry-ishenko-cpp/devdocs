ParticleSystem:getEmissionArea

Available since LÃ–VE 11.0
It has been renamed from ParticleSystem:getAreaSpread.

Gets the area-based spawn parameters for the particles.

Function

Synopsis

distribution, dx, dy, angle, directionRelativeToCenter = ParticleSystem:getEmissionArea( )

Arguments

None.

Returns

AreaSpreadDistribution distribution
The type of distribution for new particles.
number dx
The maximum spawn distance from the emitter along the x-axis for uniform distribution, or the standard deviation along the x-axis for normal distribution.
number dy
The maximum spawn distance from the emitter along the y-axis for uniform distribution, or the standard deviation along the y-axis for normal distribution.
number angle
The angle in radians of the emission area.
boolean directionRelativeToCenter
True if newly spawned particles will be oriented relative to the center of the emission area, false otherwise.
