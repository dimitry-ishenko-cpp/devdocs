Framebuffer:getImageData

Available since LÖVE 0.7.0
This function is not supported in earlier versions.

Removed in LÖVE 0.8.0
It has been renamed to Canvas:getImageData.

Returns the image data stored in the Framebuffer. Think of it as a screenshot of the hidden screen that is the framebuffer.

Function

Synopsis

data = Framebuffer:getImageData( )

Arguments

None.

Returns

ImageData data
The image data stored in the framebuffer.
