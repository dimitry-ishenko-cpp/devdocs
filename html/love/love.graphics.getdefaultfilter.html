love.graphics.getDefaultFilter

Available since LÃ–VE 0.9.0
It has been renamed from love.graphics.getDefaultImageFilter.

Returns the default scaling filters used with Images, Canvases, and Fonts.

Function

Synopsis

min, mag, anisotropy = love.graphics.getDefaultFilter( )

Arguments

None.

Returns

FilterMode min
Filter mode used when scaling the image down.
FilterMode mag
Filter mode used when scaling the image up.
number anisotropy
Maximum amount of Anisotropic Filtering used.
