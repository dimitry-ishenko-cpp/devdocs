RelativeTimeFormatOptions

package js.lib.intl

import js.lib.intl.RelativeTimeFormat

Available on js

Fields

optionalstyle:Null<RelativeTimeFormatStyle>

The length of the internationalized message. The default value is Long. The Narrow style could be similar to the short style for some locales.

optionalnumeric:Null<RelativeTimeNumeric>

The format of output message. The default value is Always. The Auto value allows to not always have to use numeric values in the output.

optionallocaleMatcher:Null<LocaleMatcher>

The locale matching algorithm to use. The default is BestFit. For information about this option, see Intl.
