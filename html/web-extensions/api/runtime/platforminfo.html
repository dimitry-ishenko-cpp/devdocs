runtime.PlatformInfo

An object containing information about the current platform.

Type

Values of this type are objects, which contain the following properties:

os

runtime.PlatformOs. The platform's operating system.

arch

runtime.PlatformArch. The platform's processor architecture.

nacl_arch

runtime.PlatformNaclArch. The native client architecture. This may be different from arch on some platforms.
