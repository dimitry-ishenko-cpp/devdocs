Command

Information about a command. This contains the information specified for the command in the commands manifest.json key.

An array of these objects is returned from commands.getAll().

Type

Values of this type are objects. They contain the following properties:

name Optional

string. Name of this command. This will be passed into the commands.onCommand event listener.

description Optional

string. Description of this command. This is primarily used to explain to the user what this command does.

shortcut Optional

string. Key(s) used to execute this command, specified as a string like "Ctrl+Shift+Y".
