ol/layer/Base~BaseLayer 


import BaseLayer from 'ol/layer/Base.js';

Abstract base class; normally only used for creating subclasses and not instantiated in apps. Note that with BaseLayer and all its subclasses, any property set in the options is set as a BaseObject property on the layer object, so is observable, and has get/set accessors.
