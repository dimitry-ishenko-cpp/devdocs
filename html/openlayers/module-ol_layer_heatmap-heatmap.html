ol/layer/Heatmap~Heatmap 


import Heatmap from 'ol/layer/Heatmap.js';

Layer for rendering vector data as a heatmap. Note that any property set in the options is set as a BaseObject property on the layer object; for example, setting title: 'My Title' in the options means that title is observable, and has get/set accessors.
