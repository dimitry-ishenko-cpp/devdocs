KeyboardZoom

Type Definitions

Options{Object}

Properties:
Name Type Description
duration number
(defaults to 100)

Animation duration in milliseconds.
condition Condition | undefined

A function that takes a MapBrowserEvent and returns a boolean to indicate whether that event should be handled. The default condition is that targetNotEditable is fulfilled and that the platform modifier key isn't pressed (!platformModifierKey).
delta number
(defaults to 1)

The zoom level delta on each key press.
