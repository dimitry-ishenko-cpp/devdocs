yarn npm tag remove

Remove a tag from a package.

Usage

$> yarn npm tag remove <package> <tag>

Examples

Remove the beta tag from package my-pkg :

yarn npm tag remove my-pkg beta

Details

This command will remove a tag from a package from the npm registry.
