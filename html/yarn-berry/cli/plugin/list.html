yarn plugin list

List the available official plugins.

Usage

$> yarn plugin list

Examples

List the official plugins :

yarn plugin list

Options

Definition
 Description
--json Format the output as an NDJSON stream

Details

This command prints the plugins available directly from the Yarn repository. Only those plugins can be referenced by name in yarn plugin import.
