LightProbeHelper

Renders a sphere to visualize a light probe in the scene.

Import

[name] is an add-on, and must be imported explicitly. See Installation / Addons. 


			import { LightProbeHelper } from 'three/addons/helpers/LightProbeHelper.js';

Code Example


		const helper = new LightProbeHelper( lightProbe, 1 );
		scene.add( helper );

Examples

WebGL / lightprobe / cubecamera

Constructor

LightProbeHelper( LightProbe lightProbe, Number size )

lightProbe -- the light probe.
size -- size of the helper sphere
