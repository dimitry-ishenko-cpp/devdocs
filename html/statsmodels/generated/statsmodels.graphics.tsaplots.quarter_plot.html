statsmodels.graphics.tsaplots.quarter_plot

statsmodels.graphics.tsaplots.quarter_plot(x, dates=None, ylabel=None, ax=None) [source]

Seasonal plot of quarterly data

Parameters:
  • x (array-like) – Seasonal data to plot. If dates is None, x must be a pandas object with a PeriodIndex or DatetimeIndex with a monthly frequency.
  • dates (array-like, optional) – If x is not a pandas object, then dates must be supplied.
  • ylabel (str, optional) – The label for the y-axis. Will attempt to use the name attribute of the Series.
  • ax (matplotlib.axes, optional) – Existing axes instance.
Returns:
Return type:

matplotlib.Figure
