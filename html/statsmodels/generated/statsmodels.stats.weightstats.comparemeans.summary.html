statsmodels.stats.weightstats.CompareMeans.summary

CompareMeans.summary(use_t=True, alpha=0.05, usevar='pooled', value=0) [source]

summarize the results of the hypothesis test

Parameters:
  • use_t (bool, optional) – if use_t is True, then t test results are returned if use_t is False, then z test results are returned
  • alpha (float) – significance level for the confidence interval, coverage is 1-alpha
  • usevar (string, 'pooled' or 'unequal') – If pooled, then the standard deviation of the samples is assumed to be the same. If unequal, then the variance of Welsh ttest will be used, and the degrees of freedom are those of Satterthwaite if use_t is True.
  • value (float) – difference between the means under the Null hypothesis.
Returns:

smry
Return type:

SimpleTable
