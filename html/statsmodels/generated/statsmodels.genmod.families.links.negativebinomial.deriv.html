NegativeBinomial.deriv(p) [source]

Derivative of the negative binomial transform

Parameters: p (array-like) – Mean parameters
Returns: g’(p) – The derivative of the negative binomial transform link function
Return type: array

Notes

g’(x) = 1/(x+alpha*x^2)
