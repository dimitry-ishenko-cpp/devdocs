statsmodels.genmod.families.links.Log

class statsmodels.genmod.families.links.Log [source]

The log transform

Notes

call and derivative call a private method _clean to trim the data by machine epsilon so that p is in (0,1). log is an alias of Log.

Methods

deriv(p) Derivative of log transform link function
deriv2(p) Second derivative of the log transform link function
inverse(z) Inverse of log transform link function
inverse_deriv(z) Derivative of the inverse of the log transform link function
