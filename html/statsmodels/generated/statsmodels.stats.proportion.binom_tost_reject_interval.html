statsmodels.stats.proportion.binom_tost_reject_interval

statsmodels.stats.proportion.binom_tost_reject_interval(low, upp, nobs, alpha=0.05) [source]

rejection region for binomial TOST

The interval includes the end points, reject if and only if r_low <= x <= r_upp.

The interval might be empty with r_upp < r_low.

Parameters:
  • upp (low,) – lower and upper limit of equivalence region
  • nobs (integer) – the number of trials or observations.
Returns:

x_low, x_upp – lower and upper bound of rejection region
Return type:

float
