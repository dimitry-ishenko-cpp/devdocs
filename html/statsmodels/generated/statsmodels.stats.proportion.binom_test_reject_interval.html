statsmodels.stats.proportion.binom_test_reject_interval

statsmodels.stats.proportion.binom_test_reject_interval(value, nobs, alpha=0.05, alternative='two-sided') [source]

rejection region for binomial test for one sample proportion

The interval includes the end points of the rejection region.

Parameters:
  • value (float) – proportion under the Null hypothesis
  • nobs (integer) – the number of trials or observations.
Returns:

x_low, x_upp – lower and upper bound of rejection region
Return type:

float
