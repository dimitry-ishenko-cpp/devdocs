statsmodels.emplike.descriptive.DescStatUV.test_kurt

DescStatUV.test_kurt(kurt0, return_weights=False) [source]

Returns -2 x log-likelihood and the p-value for the hypothesized kurtosis.

Parameters:
  • kurt0 (float) – Kurtosis value to be tested
  • return_weights (bool) – If True, function also returns the weights that maximize the likelihood ratio. Default is False.
Returns:

test_results – The log-likelihood ratio and p-value of kurt0
Return type:

tuple
