statsmodels.genmod.families.links.nbinom.inverse_deriv

nbinom.inverse_deriv(z)

Derivative of the inverse of the negative binomial transform

Parameters: z (array-like) – Usually the linear predictor for a GLM or GEE model
Returns: g^(-1)’(z) – The value of the derivative of the inverse of the negative binomial link
Return type: array
