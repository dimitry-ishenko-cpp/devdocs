statsmodels.stats.contingency_tables.StratifiedTable.summary

StratifiedTable.summary(alpha=0.05, float_format='%.3f', method='normal') [source]

A summary of all the main results.

Parameters:
  • alpha (float) – 1 - alpha is the nominal coverage probability of the confidence intervals.
  • float_format (string) – Used for formatting numeric values in the summary.
  • method (string) – The method for producing the confidence interval. Currently must be ‘normal’ which uses the normal approximation.
