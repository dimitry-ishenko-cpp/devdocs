statsmodels.robust.norms.TrimmedMean.rho

TrimmedMean.rho(z) [source]

The robust criterion function for least trimmed mean.

Parameters: z (array-like) – 1d array
Returns: rho – rho(z) = (1/2.)*z**2 for |z| <= c

rho(z) = 0 for |z| > c
Return type: array
