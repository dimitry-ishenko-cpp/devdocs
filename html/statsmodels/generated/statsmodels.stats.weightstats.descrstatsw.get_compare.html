statsmodels.stats.weightstats.DescrStatsW.get_compare

DescrStatsW.get_compare(other, weights=None) [source]

return an instance of CompareMeans with self and other

Parameters:
  • other (array_like or instance of DescrStatsW) – If array_like then this creates an instance of DescrStatsW with the given weights.
  • weights (None or array) – weights are only used if other is not an instance of DescrStatsW
Returns:

cm – the instance has self attached as d1 and other as d2.
Return type:

instance of CompareMeans
