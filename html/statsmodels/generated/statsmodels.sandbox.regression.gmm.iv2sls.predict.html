statsmodels.sandbox.regression.gmm.IV2SLS.predict

IV2SLS.predict(params, exog=None) [source]

Return linear predicted values from a design matrix.

Parameters:
  • exog (array-like) – Design / exogenous data
  • params (array-like, optional after fit has been called) – Parameters of a linear model
Returns:
Return type:

An array of fitted values

Notes

If the model as not yet been fit, params is not optional.
