statsmodels.stats.contingency_tables.SquareTable.summary

SquareTable.summary(alpha=0.05, float_format='%.3f') [source]

Produce a summary of the analysis.

Parameters:
  • alpha (float) – 1 - alpha is the nominal coverage probability of the interval.
  • float_format (string) – Used to format numeric values in the table.
  • method (string) – The method for producing the confidence interval. Currently must be ‘normal’ which uses the normal approximation.
