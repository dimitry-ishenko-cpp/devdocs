statsmodels.genmod.generalized_estimating_equations.GEEMargins

class statsmodels.genmod.generalized_estimating_equations.GEEMargins(results, args, kwargs={}) [source]

Estimated marginal effects for a regression model fit with GEE.

Parameters:
  • results (GEEResults instance) – The results instance of a fitted discrete choice model
  • args (tuple) – Args are passed to get_margeff. This is the same as results.get_margeff. See there for more information.
  • kwargs (dict) – Keyword args are passed to get_margeff. This is the same as results.get_margeff. See there for more information.
