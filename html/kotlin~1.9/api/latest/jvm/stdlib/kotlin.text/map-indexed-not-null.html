mapIndexedNotNull

kotlin-stdlib / kotlin.text / mapIndexedNotNull
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
inline fun <R : Any> CharSequence.mapIndexedNotNull(
    transform: (index: Int, Char) -> R?
): List<R>

Returns a list containing only the non-null results of applying the given transform function to each character and its index in the original char sequence.

Parameters

transform - function that takes the index of a character and the character itself and returns the result of the transform applied to the character.
