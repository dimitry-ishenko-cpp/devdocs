mapIndexedNotNullTo

kotlin-stdlib / kotlin.text / mapIndexedNotNullTo
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
inline fun <R : Any, C : MutableCollection<in R>> CharSequence.mapIndexedNotNullTo(
    destination: C, 
    transform: (index: Int, Char) -> R?
): C

Applies the given transform function to each character and its index in the original char sequence and appends only the non-null results to the given destination.

Parameters

transform - function that takes the index of a character and the character itself and returns the result of the transform applied to the character.
