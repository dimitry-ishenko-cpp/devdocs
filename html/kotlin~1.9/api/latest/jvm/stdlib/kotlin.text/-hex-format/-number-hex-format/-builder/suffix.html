suffix

kotlin-stdlib / kotlin.text / HexFormat / NumberHexFormat / Builder / suffix
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
var suffix: String

Defines NumberHexFormat.suffix of the format being built, empty string by default.

The string must not contain LF and CR characters.

Exceptions

IllegalArgumentException - if a string containing LF or CR character is assigned to this property.
