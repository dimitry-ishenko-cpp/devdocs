fun toShort(): Short

Converts this UShort value to Short.

If this value is less than or equals to Short.MAX_VALUE, the resulting Short value represents the same numerical value as this UShort. Otherwise the result is negative.

The resulting Short value has the same binary representation as this UShort value.
