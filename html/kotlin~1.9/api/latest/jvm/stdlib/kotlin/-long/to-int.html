kotlin-stdlib / kotlin / Long / toInt
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.3) 
fun toInt(): Int

Converts this Long value to Int.

If this value is in Int.MIN_VALUE..Int.MAX_VALUE, the resulting Int value represents the same numerical value as this Long.

The resulting Int value is represented by the least significant 32 bits of this Long value.
