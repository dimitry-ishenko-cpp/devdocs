Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
fun toLong(): Long

Converts this UByte value to Long.

The resulting Long value represents the same numerical value as this UByte.

The least significant 8 bits of the resulting Long value are the same as the bits of this UByte value, whereas the most significant 56 bits are filled with zeros.
