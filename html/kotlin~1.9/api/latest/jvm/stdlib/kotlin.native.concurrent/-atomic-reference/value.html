value

kotlin-stdlib / kotlin.native.concurrent / AtomicReference / value
Platform and version requirements: Native (1.3) 
var value: T

The current value. Gets the current value or sets to the given new value.

Legacy MM: if the new value value is not null, it must be frozen or permanent object.

Exceptions

InvalidMutabilityException - with legacy MM if the value is not frozen or a permanent object
