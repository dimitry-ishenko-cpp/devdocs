value

kotlin-stdlib / kotlin.native.concurrent / FreezableAtomicReference / value
Platform and version requirements: Native (1.3) 
var value: T

The referenced value. Gets the value or sets to the given new value. If the new value is not null, and this is frozen - it must be frozen or permanent object.

Exceptions

InvalidMutabilityException - if the value is not frozen or a permanent object
