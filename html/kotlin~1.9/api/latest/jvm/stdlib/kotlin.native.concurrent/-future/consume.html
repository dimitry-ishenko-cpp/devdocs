consume

kotlin-stdlib / kotlin.native.concurrent / Future / consume
Platform and version requirements: Native (1.3) 
inline fun <R> consume(code: (T) -> R): R

Blocks execution until the future is ready.

Exceptions

IllegalStateException - if future is in FutureState.INVALID, FutureState.CANCELLED or FutureState.THROWN state

Return the execution result of code consumed future's computaiton
