onPostVisitDirectory

kotlin-stdlib / kotlin.io.path / FileVisitorBuilder / onPostVisitDirectory
Platform and version requirements: JVM (1.0), JRE7 (1.0) 
abstract fun onPostVisitDirectory(
    function: (directory: Path, exception: IOException?) -> FileVisitResult)

Overrides the corresponding function of the built file visitor with the provided function.

By default, if the directory iteration completes without an I/O exception, FileVisitor.postVisitDirectory of the built file visitor returns FileVisitResult.CONTINUE; otherwise it re-throws the I/O exception that caused the iteration of the directory to terminate prematurely.
