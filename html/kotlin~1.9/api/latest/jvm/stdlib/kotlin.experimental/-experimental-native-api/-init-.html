ExperimentalNativeApi()

This annotation marks the Kotlin/Native-only standard library API that is considered experimental and is not subject to the general compatibility guarantees given for the standard library: the behavior of such API may be changed or the API may be removed completely in any further release.

Beware using the annotated API especially if you're developing a library, since your library might become binary incompatible with the future versions of the standard library.

Any usage of a declaration annotated with @ExperimentalNativeApi must be accepted either by annotating that usage with the OptIn annotation, e.g. @OptIn(ExperimentalNativeApi::class), or by using the compiler argument -opt-in=kotlin.native.ExperimentalNativeApi.
