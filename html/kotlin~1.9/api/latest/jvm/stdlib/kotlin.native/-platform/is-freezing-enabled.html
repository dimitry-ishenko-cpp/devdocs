isFreezingEnabled

kotlin-stdlib / kotlin.native / Platform / isFreezingEnabled
Platform and version requirements: Native (1.3) 
val isFreezingEnabled: Boolean

If freezing is enabled.

This value would be false, only if binary option freezing is equal to disabled. This is default when memoryModel is equal to MemoryModel.EXPERIMENTAL.
