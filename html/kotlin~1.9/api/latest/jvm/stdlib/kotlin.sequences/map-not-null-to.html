mapNotNullTo

kotlin-stdlib / kotlin.sequences / mapNotNullTo
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
inline fun <T, R : Any, C : MutableCollection<in R>> Sequence<T>.mapNotNullTo(
    destination: C, 
    transform: (T) -> R?
): C

Applies the given transform function to each element in the original sequence and appends only the non-null results to the given destination.

The operation is terminal.
