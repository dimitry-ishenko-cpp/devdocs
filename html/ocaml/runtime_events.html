Chapter 33 The runtime_events library

The runtime_events library provides an API for consuming runtime tracing and metrics information from the runtime. See chapter ‍25 for more information.

Programs that use runtime_events must be linked as follows: 

        ocamlc -I +runtime_events other options unix.cma runtime_events.cma other files
        ocamlopt -I +runtime_events other options unix.cmxa runtime_events.cmxa other files

Compilation units that use the runtime_events library must also be compiled with the -I +runtime_events option (see chapter ‍13).
