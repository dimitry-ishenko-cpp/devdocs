homepage_url

URL for the extension's home page.

If a developer key containing the "url" property and "homepage_url" are defined, Firefox uses "developer.url" while Opera uses "homepage_url". Chrome and Safari do not support the "developer" key.

This is a localizable property.

Example

"homepage_url": "https://github.com/mdn/webextensions-examples/tree/master/beastify"
