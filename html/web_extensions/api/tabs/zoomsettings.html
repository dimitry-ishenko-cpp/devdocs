tabs.ZoomSettings

Defines zoom settings for a tab: mode, scope, and default zoom factor.

Type

Values of this type are objects. They contain the following properties:

defaultZoomFactor Optional

number. The default zoom level for the current tab. Note that this is only used in tabs.getZoomSettings.

mode Optional

tabs.ZoomSettingsMode. Defines whether zoom changes are handled by the browser, by the extension, or are disabled.

scope Optional

tabs.ZoomSettingsScope. Defines whether zoom changes will persist for the page's origin, or only take effect in this tab.
