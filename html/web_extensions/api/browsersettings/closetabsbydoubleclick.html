browserSettings.closeTabsByDoubleClick

A BrowserSetting object that can be used to enable or disable the user's ability to close a tab using double-click.

The underlying value is a boolean.

By default, closeTabsByDoubleClick is false. The setting can be changed by the user in about:config.
