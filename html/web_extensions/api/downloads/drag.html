downloads.drag()

The drag() function of the downloads API initiates dragging the downloaded file to another application.

Syntax

chrome.downloads.drag(
  downloadId // integer
)

This API is also available as browser.downloads.drag().

Parameters

downloadId

An integer representing the id of the DownloadItem in question.
