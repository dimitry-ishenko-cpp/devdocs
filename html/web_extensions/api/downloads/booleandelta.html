downloads.BooleanDelta

The BooleanDelta type of the downloads API represents the difference between two booleans.

Type

Values of this type are objects. They contain the following properties:

current Optional

A boolean representing the current boolean value.

previous Optional

A boolean representing the previous boolean value.
