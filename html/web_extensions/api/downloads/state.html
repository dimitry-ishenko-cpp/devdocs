downloads.State

The State type of the downloads API defines different states that a current download can be in.

A downloads.DownloadItem's state property will contain a string taken from the values defined in this type.

Type

Values of this type are strings. Possible values are:

in_progress

The browser is currently receiving download data from the server.

interrupted

An error broke the connection with the server.

complete

The download completed successfully.

Note: These string constants will never change, but new constants may be added.
