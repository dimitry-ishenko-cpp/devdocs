management.get()

Retrieves an ExtensionInfo object containing information about the specified add-on.

This API requires the "management" API permission.

This is an asynchronous function that returns a Promise.

Syntax

let gettingInfo = browser.management.get(
  id                  // string
)

Parameters

id

string. ID of the add-on whose info you want to retrieve.

Return value

A Promise that will be fulfilled with an ExtensionInfo object, containing information about the add-on. The promise will be rejected if no extension with the given ID is installed or the extension is not allowed to be accessed by the caller.

Examples

Log the name of the add-on whose ID is "my-add-on":

let id = "my-add-on";

function got(info) {
  console.log(info.name);
}

let getting = browser.management.get(id);
getting.then(got);

Note: This API is based on Chromium's chrome.management API. This documentation is derived from management.json in the Chromium code.
