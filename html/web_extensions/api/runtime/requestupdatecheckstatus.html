runtime.RequestUpdateCheckStatus

Result of a call to runtime.requestUpdateCheck().

Type

Values of this type are strings. Possible values are:

"throttled"

Update is throttled.

"no_update"

No update is available.

"update_available"

An update of the extension is available.
