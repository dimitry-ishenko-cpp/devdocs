NAME

Test2::Event::Note - Note event type

DESCRIPTION

Notes, typically rendered to STDOUT.

SYNOPSIS

use Test2::API qw/context/;
use Test2::Event::Note;

my $ctx = context();
my $event = $ctx->Note($message);

ACCESSORS

$note->message

The message for the note.
