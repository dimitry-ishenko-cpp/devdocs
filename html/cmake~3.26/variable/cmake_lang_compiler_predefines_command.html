CMAKE_<LANG>_COMPILER_PREDEFINES_COMMAND

New in version 3.10.

Command that outputs the compiler pre definitions.

See AUTOMOC which uses CMAKE_CXX_COMPILER_PREDEFINES_COMMAND to generate the AUTOMOC_COMPILER_PREDEFINES.
