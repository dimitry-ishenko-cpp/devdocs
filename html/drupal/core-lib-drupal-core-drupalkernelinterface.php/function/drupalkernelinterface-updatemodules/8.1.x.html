public DrupalKernelInterface::updateModules(array $module_list, array $module_filenames = array())

Updates the kernel's list of modules to the new list.

The kernel needs to update its bundle list and container to match the new list.

Parameters

array $module_list: The new list of modules.

array $module_filenames: List of module filenames, keyed by module name.
