public function PagerSelectExtender::limit

public PagerSelectExtender::limit($limit = 10)

Specify the maximum number of elements per page for this query.

The default if not specified is 10 items per page.

Parameters

int|false $limit: An integer specifying the number of elements per page. If passed a false value (FALSE, 0, NULL), the pager is disabled.
