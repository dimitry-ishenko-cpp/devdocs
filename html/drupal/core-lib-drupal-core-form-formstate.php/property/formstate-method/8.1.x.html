protected property FormState::$method

The HTTP form method to use for finding the input for this form.

May be 'POST' or 'GET'. Defaults to 'POST'. Note that 'GET' method forms do not use form ids so are always considered to be submitted, which can have unexpected effects. The 'GET' method should only be used on forms that do not change data, as that is exclusively the domain of 'POST.'

This property is uncacheable.
