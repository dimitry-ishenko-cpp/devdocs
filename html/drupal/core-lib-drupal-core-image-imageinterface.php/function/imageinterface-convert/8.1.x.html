public ImageInterface::convert($extension)

Instructs the toolkit to save the image in the format specified by the extension.

Parameters

string $extension: The extension to convert to (for instance, 'jpeg' or 'png'). Allowed values depend on the current image toolkit.

Return value

bool TRUE on success, FALSE on failure.
