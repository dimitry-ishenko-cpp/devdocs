The clear() method of Set instances removes all elements from this set.

Try it

Syntax

clear()

Parameters

None.

Return value

None (undefined).

Examples

Using the clear() method

const mySet = new Set();
mySet.add(1);
mySet.add("foo");

console.log(mySet.size); // 2
console.log(mySet.has("foo")); // true

mySet.clear();

console.log(mySet.size); // 0
console.log(mySet.has("foo")); // false
