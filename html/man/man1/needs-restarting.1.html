NAME 

       needs-restarting - report running processes that have been
       updated

SYNOPSIS 

       needs-restarting [options]

DESCRIPTION 

       needs-restarting is a program that reports a list of process ids
       that started running before they or some component that they use
       were updated.

OPTIONS 

       -h, --help
              Display a help message, and then quit.

       -u, --useronly
              Show processes for my userid only.

       -r, --reboothint
              Only report whether a full reboot is required (exit code
              1) or not (exit code 0).

       -s, --services
              List the affected systemd services only.
