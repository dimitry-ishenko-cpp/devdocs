NAME 

       factor - factor numbers

SYNOPSIS 

       factor [OPTION] [NUMBER]...

DESCRIPTION 

       Print the prime factors of each specified integer NUMBER.  If
       none are specified on the command line, read them from standard
       input.

       -h, --exponents
              print repeated factors in form p^e unless e is 1

       --help display this help and exit

       --version
              output version information and exit
