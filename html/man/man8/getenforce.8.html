NAME 

       getenforce - get the current mode of SELinux

SYNOPSIS 

       getenforce

DESCRIPTION 

       getenforce reports whether SELinux is enforcing, permissive, or
       disabled.

AUTHOR 

       Dan Walsh, <dwalsh@redhat.com>
