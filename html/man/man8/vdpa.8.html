NAME 

       vdpa - vdpa management tool

SYNOPSIS 

       vdpa [ OPTIONS ] { dev|mgmtdev } { COMMAND | help }

OPTIONS 

       -V, --Version
              Print the version of the vdpa utility and exit.

       -j, --json
              Generate JSON output.

       -p, --pretty
              When combined with -j generate a pretty JSON output.

   OBJECT
       dev    - vdpa device.

       mgmtdev
              - vdpa management device.

   COMMAND
       Specifies the action to perform on the object.  The set of
       possible actions depends on the object type.  It is possible to
       show (or list ) objects. The help command is available for all
       objects. It prints out a list of available commands and argument
       syntax conventions.

       If no command is given, some default command is assumed.  Usually
       it is show or, if the objects of this class cannot be listed,
       help.

EXIT STATUS 

       Exit status is 0 if command was successful or a positive integer
       upon failure.
