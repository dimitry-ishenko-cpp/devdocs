NAME 

       lastlog2 - Y2038 safe version of lastlog library.

SYNOPSIS 

       #include <lastlog2.h>

DESCRIPTION 

       lastlog2 reports the last login of a given user or of all users
       who did ever login on a system.

       It’s using sqlite3 as database backend. Data is only collected
       via a PAM module, so that every tools can make use of it, without
       modifying existing packages. The output is as compatible as
       possible with the old lastlog implementation. By default the
       database will be written as /var/lib/lastlog/lastlog2.db. The
       size of the database depends on the amount of users, not how big
       the biggest UID is.
