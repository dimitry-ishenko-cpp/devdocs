NAME 

       freelocale — free resources allocated for a locale object

SYNOPSIS 

       #include <locale.h>

       void freelocale(locale_t locobj);

DESCRIPTION 

       The freelocale() function shall cause the resources allocated for
       a locale object returned by a call to the newlocale() or
       duplocale() functions to be released.

       The behavior is undefined if the locobj argument is the special
       locale object LC_GLOBAL_LOCALE or is not a valid locale object
       handle.

       Any use of a locale object that has been freed results in
       undefined behavior.

RETURN VALUE 

       None.

ERRORS 

       None.

       The following sections are informative.

EXAMPLES 

   Freeing Up a Locale Object
       The following example shows a code fragment to free a locale
       object created by newlocale():

           #include <locale.h>
           ...

           /* Every locale object allocated with newlocale() should be
            * freed using freelocale():
            */

           locale_t loc;

           /* Get the locale. */

           loc = newlocale (LC_CTYPE_MASK | LC_TIME_MASK, "locname", NULL);

           /* ... Use the locale object ... */
           ...

           /* Free the locale object resources. */
           freelocale (loc);
