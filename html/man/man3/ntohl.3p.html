NAME 

       ntohl, ntohs — convert values between host and network byte order

SYNOPSIS 

       #include <arpa/inet.h>

       uint32_t ntohl(uint32_t netlong);
       uint16_t ntohs(uint16_t netshort);

DESCRIPTION 

       Refer to htonl(3p).
