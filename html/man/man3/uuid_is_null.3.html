NAME 

       uuid_is_null - compare the value of the UUID to the NULL value

SYNOPSIS 

       #include <uuid.h>

       int uuid_is_null(const uuid_t uu);

DESCRIPTION 

       The uuid_is_null() function compares the value of the supplied
       UUID variable uu to the NULL value. If the value is equal to the
       NULL UUID, 1 is returned, otherwise 0 is returned.
