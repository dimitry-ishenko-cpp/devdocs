NAME 

       gnutls_session_channel_binding - API function

SYNOPSIS 

       #include <gnutls/gnutls.h>

       int gnutls_session_channel_binding(gnutls_session_t session,
       gnutls_channel_binding_t cbtype, gnutls_datum_t * cb);

ARGUMENTS 

       gnutls_session_t session
                   is a gnutls_session_t type.

       gnutls_channel_binding_t cbtype
                   an gnutls_channel_binding_t enumeration type

       gnutls_datum_t * cb
                   output buffer array with data

DESCRIPTION 

       Extract given channel binding data of the  cbtype (e.g.,
       GNUTLS_CB_TLS_UNIQUE) type.

RETURNS 

       GNUTLS_E_SUCCESS on success, GNUTLS_E_UNIMPLEMENTED_FEATURE if
       the  cbtype is unsupported,
       GNUTLS_E_CHANNEL_BINDING_NOT_AVAILABLE if the data is not
       currently available, or an error code.

SINCE 

       2.12.0
