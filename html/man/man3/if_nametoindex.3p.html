NAME 

       if_nametoindex — map a network interface name to its
       corresponding index

SYNOPSIS 

       #include <net/if.h>

       unsigned if_nametoindex(const char *ifname);

DESCRIPTION 

       The if_nametoindex() function shall return the interface index
       corresponding to name ifname.

RETURN VALUE 

       The corresponding index if ifname is the name of an interface;
       otherwise, zero.

ERRORS 

       No errors are defined.
