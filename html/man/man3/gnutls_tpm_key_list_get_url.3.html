NAME 

       gnutls_tpm_key_list_get_url - API function

SYNOPSIS 

       #include <gnutls/tpm.h>

       int gnutls_tpm_key_list_get_url(gnutls_tpm_key_list_t list,
       unsigned int idx, char ** url, unsigned int flags);

ARGUMENTS 

       gnutls_tpm_key_list_t list
                   a list of the keys

       unsigned int idx
                   The index of the key (starting from zero)

       char ** url The URL to be returned

       unsigned int flags
                   should be zero

DESCRIPTION 

       This function will return for each given index a URL of the
       corresponding key.  If the provided index is out of bounds then
       GNUTLS_E_REQUESTED_DATA_NOT_AVAILABLE is returned.

RETURNS 

       On success, GNUTLS_E_SUCCESS (0) is returned, otherwise a
       negative error value.

SINCE 

       3.1.0
