NAME 

       difftime - calculate time difference

LIBRARY 

       Standard C library (libc, -lc)

SYNOPSIS 

       #include <time.h>

       double difftime(time_t time1, time_t time0);

DESCRIPTION 

       The difftime() function returns the number of seconds elapsed
       between time time1 and time time0, represented as a double.  Each
       time is a count of seconds.

       difftime(b, a) acts like (b-a) except that the result does not
       overflow and is rounded to double.
