NAME 

       utimensat, utimes — set file access and modification times

SYNOPSIS 

       #include <sys/stat.h>

       int utimensat(int fd, const char *path, const struct timespec times[2],
           int flag);

       #include <sys/time.h>

       int utimes(const char *path, const struct timeval times[2]);

DESCRIPTION 

       Refer to futimens(3p).
