NAME 

       stapfuncs - systemtap functions

DESCRIPTION 

       Functions in the standard systemtap tapset are individually
       documented in the 3stap manual section, with the function::
       prefix.  Some built-in functions such as printf are documented on
       the stap(1) man page.

SEE ALSO 

       stap(1),
       function::*(3stap),
       tapset::*(3stap)
