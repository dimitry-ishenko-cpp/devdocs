NAME 

       gnutls_pkcs11_copy_x509_crt - API function

SYNOPSIS 

       #include <gnutls/pkcs11.h>

       int gnutls_pkcs11_copy_x509_crt(const char * token_url,
       gnutls_x509_crt_t crt, const char * label, unsigned int flags);

ARGUMENTS 

       const char * token_url
                   A PKCS 11 URL specifying a token

       gnutls_x509_crt_t crt
                   A certificate

       const char * label
                   A name to be used for the stored data

       unsigned int flags
                   One of GNUTLS_PKCS11_OBJ_FLAG_*

DESCRIPTION 

       This function will copy a certificate into a PKCS 11 token
       specified by a URL. The certificate can be marked as trusted or
       not.

RETURNS 

       On success, GNUTLS_E_SUCCESS (0) is returned, otherwise a
       negative error value.

SINCE 

       2.12.0
