NAME 

       gnutls_pk_to_sign - API function

SYNOPSIS 

       #include <gnutls/gnutls.h>

       gnutls_sign_algorithm_t gnutls_pk_to_sign(gnutls_pk_algorithm_t
       pk, gnutls_digest_algorithm_t hash);

ARGUMENTS 

       gnutls_pk_algorithm_t pk
                   is a public key algorithm

       gnutls_digest_algorithm_t hash
                   a hash algorithm

DESCRIPTION 

       This function maps public key and hash algorithms combinations to
       signature algorithms.

RETURNS 

       return a gnutls_sign_algorithm_t value, or GNUTLS_SIGN_UNKNOWN on
       error.
