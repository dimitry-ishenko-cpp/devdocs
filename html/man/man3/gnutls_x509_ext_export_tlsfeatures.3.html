NAME 

       gnutls_x509_ext_export_tlsfeatures - API function

SYNOPSIS 

       #include <gnutls/x509-ext.h>

       int gnutls_x509_ext_export_tlsfeatures(gnutls_x509_tlsfeatures_t
       f, gnutls_datum_t * ext);

ARGUMENTS 

       gnutls_x509_tlsfeatures_t f
                   The features structure

       gnutls_datum_t * ext
                   The DER-encoded extension data; must be freed using
                   gnutls_free().

DESCRIPTION 

       This function will convert the provided TLS features structure
       structure to a DER-encoded TLS features PKIX extension. The
       output data in  ext will be allocated using gnutls_malloc().

RETURNS 

       On success, GNUTLS_E_SUCCESS (0) is returned, otherwise a
       negative error value.

SINCE 

       3.5.1
