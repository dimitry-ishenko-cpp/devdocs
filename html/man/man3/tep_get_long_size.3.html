NAME 

       tep_get_long_size, tep_set_long_size - Get / set the size of a
       long integer on the machine, where the trace is generated, in
       bytes

SYNOPSIS 

       #include <event-parse.h>

       int tep_get_long_size(strucqt tep_handle *tep);
       void tep_set_long_size(struct tep_handle *tep, int long_size);

DESCRIPTION 

       The tep_get_long_size() function returns the size of a long
       integer on the machine, where the trace is generated. The tep
       argument is trace event parser context.

       The tep_set_long_size() function sets the size of a long integer
       on the machine, where the trace is generated. The tep argument is
       trace event parser context. The long_size is the size of a long
       integer, in bytes.

RETURN VALUE 

       The tep_get_long_size() function returns the size of a long
       integer on the machine, where the trace is generated, in bytes.

EXAMPLE 

           #include <event-parse.h>
           ...
           struct tep_handle *tep = tep_alloc();
           ...
           tep_set_long_size(tep, 4);
           ...
           int long_size = tep_get_long_size(tep);
           ...

FILES 

           event-parse.h
                   Header file to include in order to have access to the library APIs.
           -ltraceevent
                   Linker switch to add when building a program that uses the library.
