NAME 

       pthread_attr_setscope — set the contentionscope attribute
       (REALTIME THREADS)

SYNOPSIS 

       #include <pthread.h>

       int pthread_attr_setscope(pthread_attr_t *attr, int contentionscope);

DESCRIPTION 

       Refer to pthread_attr_getscope(3p).
