NAME 

       gnutls_handshake_write - API function

SYNOPSIS 

       #include <gnutls/gnutls.h>

       int gnutls_handshake_write(gnutls_session_t session,
       gnutls_record_encryption_level_t level, const void * data, size_t
       data_size);

ARGUMENTS 

       gnutls_session_t session
                   is a gnutls_session_t type.

       gnutls_record_encryption_level_t level
                   the current encryption level for reading a handshake
                   message

       const void * data
                   the (const) handshake data to be processed

       size_t data_size
                   the size of data

DESCRIPTION 

       This function processes a handshake message in the encryption
       level specified with  level . Prior to calling this function, a
       handshake read callback must be set on  session . Use
       gnutls_handshake_set_read_function() to do this.

SINCE 

       3.7.0
