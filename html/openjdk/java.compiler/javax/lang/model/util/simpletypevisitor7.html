@SupportedSourceVersion(RELEASE_7) public class SimpleTypeVisitor7<R,P> extends SimpleTypeVisitor6<R,P>
A simple visitor of types with default behavior appropriate for the RELEASE_7 source version. Visit methods corresponding to RELEASE_7 and earlier language constructs call defaultAction, passing their arguments to defaultAction's corresponding parameters.
