Method Details

location

Location location()
Returns the Location of this mirror, if there is executable code associated with it. Note that both Java programming language methods and native methods have executable code. Returns null for abstract methods, since abstract methods have no executable code.
Returns:
the Location of this mirror, or null if there is no executable code associated with it.
