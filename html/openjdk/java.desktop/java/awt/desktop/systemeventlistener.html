Interface SystemEventListener

All Superinterfaces:
EventListener
All Known Subinterfaces:
AppForegroundListener, AppHiddenListener, AppReopenedListener, ScreenSleepListener, SystemSleepListener, UserSessionListener 
public interface SystemEventListener extends EventListener
Common interface for all event listener sub-types. Implementors may implement multiple sub-types, but only need to call Desktop.addAppEventListener(SystemEventListener) once to receive all notifications.
