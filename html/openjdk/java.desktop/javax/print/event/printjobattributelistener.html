Interface PrintJobAttributeListener

public interface PrintJobAttributeListener
Implementations of this interface are attached to a DocPrintJob to monitor the status of attribute changes associated with the print job.

void attributeUpdate(PrintJobAttributeEvent pjae)
Notifies the listener of a change in some print job attributes. One example of an occurrence triggering this event is if the JobState attribute changed from PROCESSING to PROCESSING_STOPPED.
Parameters:
pjae - the event being notified
