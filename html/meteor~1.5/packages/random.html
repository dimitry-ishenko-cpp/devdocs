random

Documentation of Meteor's `random` package.

The random package provides several functions for generating random numbers. It uses a cryptographically strong pseudorandom number generator when possible, but falls back to a weaker random number generator when cryptographically strong randomness is not available (on older browsers or on servers that donâ€™t have enough entropy to seed the cryptographically strong generator).

Random.secret
import { Random } from 'meteor/random' (random/random.js, line 209)

Return a random string of printable characters with 6 bits of entropy per character. Use Random.secret for security-critical secrets that are intended for machine, rather than human, consumption.

Arguments

n Number

Optional length of the secret string (defaults to 43 characters, or 256 bits of entropy)
