Module: tf.compat.v1.keras.experimental

Public API for tf.keras.experimental namespace.

Classes

class CosineDecay: A LearningRateSchedule that uses a cosine decay schedule.

class SequenceFeatures: A layer for sequence input.

Functions

export_saved_model(...): Exports a tf.keras.Model as a Tensorflow SavedModel.

load_from_saved_model(...): Loads a keras Model from a SavedModel created by export_saved_model().
