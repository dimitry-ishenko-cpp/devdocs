tf.distribute.experimental.coordinator.PerWorkerValues

A container that holds a list of values, one value per worker.

View aliases

Main aliases

tf.distribute.coordinator.PerWorkerValue

tf.distribute.experimental.coordinator.PerWorkerValues(
    values
)

tf.distribute.experimental.coordinator.PerWorkerValues contains a collection of values, where each of the values is located on its corresponding worker, and upon being used as one of the args or kwargs of tf.distribute.experimental.coordinator.ClusterCoordinator.schedule(), the value specific to a worker will be passed into the function being executed at that corresponding worker.

Currently, the only supported path to create an object of tf.distribute.experimental.coordinator.PerWorkerValues is through calling iter on a ClusterCoordinator.create_per_worker_dataset-returned distributed dataset instance. The mechanism to create a custom tf.distribute.experimental.coordinator.PerWorkerValues is not yet supported.
