jQuery.effects.restoreStyle()

jQuery.effects.restoreStyle( element )version added: 1.12

Description: Restores all inline styles for an element.

  • jQuery.effects.restoreStyle( element )

    • element
      Type: jQuery
      The element to restore styles for.

Restores all inline styles for an element which have been saved using jQuery.effects.saveStyle().
