Detailed Description

The EnterKey attached property is used to manipulate the appearance and behavior of the Enter key on an on-screen keyboard.

Attached Property Documentation

EnterKey.type : enumeration

Holds the type of the Enter key.

Note: Not all of these values are supported on all platforms. For unsupported values the default key is used instead.

Constant Description
Qt.EnterKeyDefault The default Enter key. This can be either a button to accept the input and close the keyboard, or a Return button to enter a newline in case of a multi-line input field.
Qt.EnterKeyReturn Show a Return button that inserts a newline.
Qt.EnterKeyDone Show a "Done" button. Typically, the keyboard is expected to close when the button is pressed.
Qt.EnterKeyGo Show a "Go" button. Typically used in an address bar when entering a URL.
Qt.EnterKeySend Show a "Send" button.
Qt.EnterKeySearch Show a "Search" button.
Qt.EnterKeyNext Show a "Next" button. Typically used in a form to allow navigating to the next input field without the keyboard closing.
Qt.EnterKeyPrevious Show a "Previous" button.
