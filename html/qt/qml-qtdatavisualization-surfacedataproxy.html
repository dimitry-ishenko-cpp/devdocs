Detailed Description

This type handles surface data items. The data is arranged into rows and columns, and all rows must have the same number of columns.

This type is uncreatable, but contains properties that are exposed via subtypes.

For a more complete description, see QSurfaceDataProxy.

See also ItemModelSurfaceDataProxy and Qt Data Visualization Data Handling.

Property Documentation

columnCount : int

The number of columns in the data array.

rowCount : int

The number of rows in the data array.

series : Surface3DSeries

The series this proxy is attached to.
