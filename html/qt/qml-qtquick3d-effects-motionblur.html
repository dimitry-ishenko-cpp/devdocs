Detailed Description

The MotionBlur effect creates a trailing streak for rapidly moving objects in the scene.

Note: The background mode must be SceneEnvironment.Transparent for MotionBlur to work.

Property Documentation

blurQuality : real

The quality of the blur used to create the trail. Increasing quality will have impact on performance. The range is [0...1]. The default value is 0.25.

fadeAmount : real

How fast the trail fades. The range is [0...1], with 0 meaning that the object trail persists indefinitely, and 1 meaning that the object trail fades out immediately. The default value is 0.25.
