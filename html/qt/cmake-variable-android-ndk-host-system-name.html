ANDROID_NDK_HOST_SYSTEM_NAME

Android-specific architecture of the host system.

Note: This variable is in technology preview and may change in future releases.

Note: This variable is used only if targeting the Android platform.

This is normally set by the Android NDK toolchain file. It is written out as part of the deployment settings for a target.

See also qt_android_generate_deployment_settings().
