Detailed Description

The QtTextInputMethodManager corresponds to the qt-text-input-method-manager interface in the qt-text-input-method-unstable-v1 extension protocol. It is specifically designed to be used with a Qt-based input method, such as Qt Virtual Keyboard.

Instantiating this as child of a WaylandCompositor adds it to the list of interfaces available to the client. If a client binds to it, then it will be used to communciate text input to that client.
