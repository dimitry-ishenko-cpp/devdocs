int QML Basic Type

The int type refers to a whole number, e.g. 0, 10, or -20.

The possible int values range from -2147483648 to 2147483647, although most types will only accept a reduced range (which they mention in their documentation).

Example:

NumberAnimation { loops: 5 }

This basic type is provided by the QML language.

See also QML Basic Types.
