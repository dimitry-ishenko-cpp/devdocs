Detailed Description

The Flip effect flips the whole scene either horizontally, vertically, or in both directions.

Property Documentation

flipHorizontally : bool

This property determines whether the scene is flipped horizontally. The default value is true.

flipVertically : bool

This property determines whether the scene is flipped vertically. The default value is true.
