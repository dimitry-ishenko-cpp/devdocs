MenuSeparator QML Type

Detailed Description

MenuSeparator is used to visually distinguish between groups of items in a menu by separating them with a line.

import QtQuick
import QtQuick.Controls

Menu {
    id: menu
    MenuItem {
        text: qsTr("New...")
    }
    MenuItem {
        text: qsTr("Open...")
    }
    MenuItem {
        text: qsTr("Save")
    }

    MenuSeparator {
        padding: 0
        topPadding: 12
        bottomPadding: 12
        contentItem: Rectangle {
            implicitWidth: 200
            implicitHeight: 1
            color: "#1E000000"
        }
    }

    MenuItem {
        text: qsTr("Exit")
    }
}
