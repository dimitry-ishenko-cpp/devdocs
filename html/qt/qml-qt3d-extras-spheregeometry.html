Detailed Description

The SphereGeometry type is most commonly used internally by the SphereMesh type but can also be used in custom GeometryRenderer types.

Property Documentation

generateTangents : bool

Holds the value of the automatic tangent vectors generation flag. Tangent vectors are orthogonal to normal vectors.

indexAttribute : Attribute

Holds the geometry index attribute.

normalAttribute : Attribute

Holds the geometry normal attribute.

positionAttribute : Attribute

Holds the geometry position attribute.

radius : real

Holds the radius of the sphere.

rings : int

Holds the number of rings in the sphere.

slices : int

Holds the number of slices in the sphere.

tangentAttribute : Attribute

Holds the geometry tangent attribute.

texCoordAttribute : Attribute

Holds the geometry texture coordinate attribute.
