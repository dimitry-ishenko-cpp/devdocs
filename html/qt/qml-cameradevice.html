cameraDevice QML Basic Type

Methods

  • string description()
  • string id()
  • bool isDefault()
  • enumeration position()

The cameraDevice value type describes the properties of a camera device that is connected to the system.

The list of camera devices can be queried from the MediaDevices type. To select a certain camera device set it as the device on Camera.

CaptureSession {
    camera: Camera {
        cameraDevice: mediaDevices.defaultVideoInput
    }
}
MediaDevices {
    id: mediaDevices
}

Method Documentation

string description()

Returns a human readable name of the camera.

Use this string to present the device to the user.

string id()

Returns the device id of the camera

This is a unique ID to identify the camera and may not be human-readable.

bool isDefault()

Returns true if this is the default camera device.

enumeration position()

Returns the physical position of the camera on the hardware system.

The returned value can be one of the following:

Property value Description
CameraDevice.UnspecifiedPosition The physical positition of the camera is unspecified.
CameraDevice.FrontFace The camera is on the front of the hardware, facing towards the user.
CameraDevice.BackFace The camera is on the back of the hardware, facing away from the user.
