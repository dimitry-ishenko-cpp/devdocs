Detailed Description

The only difference is that the filter() method features a pointer to QProximityReading instead of QSensorReading.

Member Function Documentation

[pure virtual] bool QProximityFilter::filter(QProximityReading *reading)

Called when reading changes. Returns false to prevent the reading from propagating.
