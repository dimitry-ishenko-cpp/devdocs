tf.compat.v1.losses.add_loss

Adds a externally defined loss to the collection of losses.

tf.compat.v1.losses.add_loss(
    loss, loss_collection=ops.GraphKeys.LOSSES
)
Args
loss A loss Tensor.
loss_collection Optional collection to add the loss to.
