tf.keras.ops.log_softmax

Log-softmax activation function.

View aliases

Main aliases

tf.keras.ops.nn.log_softmax

tf.keras.ops.log_softmax(
    x, axis=-1
)

It is defined as:

f(x) = x - max(x) - log(sum(exp(x - max(x))))

Args
x Input tensor.
axis Integer, axis along which the log-softmax is applied. Defaults to -1.
Returns
A tensor with the same shape as x.

Example:

x = np.array([-1., 0., 1.])
x_log_softmax = keras.ops.log_softmax(x)
print(x_log_softmax)
array([-2.40760596, -1.40760596, -0.40760596], shape=(3,), dtype=float64)
