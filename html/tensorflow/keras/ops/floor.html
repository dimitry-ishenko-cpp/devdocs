tf.keras.ops.floor

Return the floor of the input, element-wise.

View aliases

Main aliases

tf.keras.ops.numpy.floor

tf.keras.ops.floor(
    x
)

The floor of the scalar x is the largest integer i, such that i <= x.

Args
x Input tensor.
Returns
Output tensor, element-wise floor of x.
