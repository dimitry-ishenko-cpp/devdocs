tf.keras.ops.sort

Sorts the elements of x along a given axis in ascending order.

View aliases

Main aliases

tf.keras.ops.numpy.sort

tf.keras.ops.sort(
    x, axis=-1
)
Args
x Input tensor.
axis Axis along which to sort. If None, the tensor is flattened before sorting. Defaults to -1; the last axis.
Returns
Sorted tensor.
