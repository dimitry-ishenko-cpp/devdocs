tf.keras.ops.arccosh

Inverse hyperbolic cosine, element-wise.

View aliases

Main aliases

tf.keras.ops.arccosh(
    x
)
Arguments
x Input tensor.
Returns
Output tensor of same shape as x.

Example:

x = keras.ops.convert_to_tensor([10, 100])
keras.ops.arccosh(x)
