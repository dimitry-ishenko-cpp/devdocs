tf.keras.activations.mish

Mish activation function.

tf.keras.activations.mish(
    x
)

It is defined as:

mish(x) = x * tanh(softplus(x))

where softplus is defined as:

softplus(x) = log(exp(x) + 1)

Args
x Input tensor.
