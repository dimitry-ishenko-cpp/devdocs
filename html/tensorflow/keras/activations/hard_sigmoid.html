tf.keras.activations.hard_sigmoid

Hard sigmoid activation function.

tf.keras.activations.hard_sigmoid(
    x
)

The hard sigmoid activation is defined as:

  • 0 if if x <= -3
  • 1 if x >= 3
  • (x/6) + 0.5 if -3 < x < 3

It's a faster, piecewise linear approximation of the sigmoid activation.

Args
x Input tensor.
