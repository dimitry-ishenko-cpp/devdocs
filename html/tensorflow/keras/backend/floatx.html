tf.keras.backend.floatx

Return the default float type, as a string.

View aliases

Main aliases

tf.keras.config.floatx

tf.keras.backend.floatx()

E.g. 'bfloat16', 'float16', 'float32', 'float64'.

Returns
String, the current default float type.

Example:

keras.config.floatx()
'float32'
