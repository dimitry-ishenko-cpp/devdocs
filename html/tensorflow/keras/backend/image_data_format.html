tf.keras.backend.image_data_format

Return the default image data format convention.

View aliases

Main aliases

tf.keras.config.image_data_format

tf.keras.backend.image_data_format()
Returns
A string, either 'channels_first' or 'channels_last'.

Example:

keras.config.image_data_format()
'channels_last'
