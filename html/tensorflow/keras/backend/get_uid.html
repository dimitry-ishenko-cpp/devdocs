tf.keras.backend.get_uid

Associates a string prefix with an integer counter.

tf.keras.backend.get_uid(
    prefix=''
)
Args
prefix String prefix to index.
Returns
Unique integer ID.

Example:

get_uid('dense')
1
get_uid('dense')
2
