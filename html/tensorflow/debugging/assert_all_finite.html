tf.debugging.assert_all_finite

Assert that the tensor does not contain any NaN's or Inf's.

tf.debugging.assert_all_finite(
    x, message, name=None
)

@tf.function
def f(x):
  x = tf.debugging.assert_all_finite(x, 'Input x must be all finite')
  return x + 1
