tf.saved_model.experimental.read_fingerprint

Reads the fingerprint of a SavedModel in export_dir.

tf.saved_model.experimental.read_fingerprint(
    export_dir: str
) -> tf.saved_model.experimental.Fingerprint

Returns a tf.saved_model.experimental.Fingerprint object that contains the values of the SavedModel fingerprint, which is persisted on disk in the fingerprint.pb file in the export_dir.

Read more about fingerprints in the SavedModel guide at https://www.tensorflow.org/guide/saved_model

Args
export_dir The directory that contains the SavedModel.
Returns
A tf.saved_model.experimental.Fingerprint.
Raises
FileNotFoundError If no or an invalid fingerprint is found.
