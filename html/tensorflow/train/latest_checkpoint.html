Gets the checkpoint state given the provided checkpoint_dir and looks for a corresponding TensorFlow 2 (preferred) or TensorFlow 1.x checkpoint path. The latest_filename argument is only applicable if you are saving checkpoint using v1.train.Saver.save

See the Training Checkpoints Guide for more details and examples.`

Args
checkpoint_dir Directory where the variables were saved.
latest_filename Optional name for the protocol buffer file that contains the list of most recent checkpoint filenames. See the corresponding argument to v1.train.Saver.save.
Returns
The full path to the latest checkpoint or None if no checkpoint was found.
