@babel/plugin-proposal-export-namespace-from

NOTE: This plugin is included in @babel/preset-env, in ES2020

Example​ 

export * as ns from "mod";

Installation​

  • npm
  • Yarn
  • pnpm
npm install --save-dev @babel/plugin-proposal-export-namespace-from
yarn add --dev @babel/plugin-proposal-export-namespace-from
pnpm add --save-dev @babel/plugin-proposal-export-namespace-from

Usage​

With a configuration file (Recommended)​ 

{
  "plugins": ["@babel/plugin-proposal-export-namespace-from"]
}

Via CLI​ 

babel --plugins @babel/plugin-proposal-export-namespace-from script.js

Via Node API​ 

require("@babel/core").transformSync("code", {
  plugins: ["@babel/plugin-proposal-export-namespace-from"],
});

References​
