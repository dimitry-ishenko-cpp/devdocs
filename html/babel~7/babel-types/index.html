@babel/types

This module contains methods for building ASTs manually and for checking the types of AST nodes.

Install​

  • npm
  • Yarn
  • pnpm
npm install --save-dev @babel/types
yarn add --dev @babel/types
pnpm add --save-dev @babel/types
