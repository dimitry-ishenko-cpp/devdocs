Enumerations

enum Hint:

  • HINT_NONE = 0 --- No hint used.
  • HINT_RANGE = 1 --- A range hint for scalar value, which limits possible input values between min and max. Translated to hint_range(min, max) in shader code.
  • HINT_RANGE_STEP = 2 --- A range hint for scalar value with step, which limits possible input values between min and max, with a step (increment) of step). Translated to hint_range(min, max, step) in shader code.
  • HINT_MAX = 3 --- Represents the size of the Hint enum.
