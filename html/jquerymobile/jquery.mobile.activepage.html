jQuery.mobile.activePage

jQuery.mobile.activePageReturns: jQueryversion added: 1.0, deprecated: 1.4.0

Description: Reference to the page currently in view.

Note: jQuery.mobile.activePage is deprecated as of jQuery Mobile 1.4.0 and will be removed in 1.5.0. Use the getActivePage() method from the pagecontainer widget instead.
