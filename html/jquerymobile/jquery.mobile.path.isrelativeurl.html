jQuery.mobile.path.isRelativeUrl()

jQuery.mobile.path.isRelativeUrl( url )Returns: Boolean

Description: Utility method for determining if a URL is a relative variant.

  • jQuery.mobile.path.isRelativeUrl( url )

    • url
      Type: String

Utility method for determining if a URL is relative variant. This function returns a boolean true if the URL is relative, false if it is absolute.
