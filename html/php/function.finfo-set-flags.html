finfo_set_flags -- finfo::set_flags â€” Set libmagic configuration options

Description

Procedural style

finfo_set_flags(finfo $finfo, int $flags): true

Object-oriented style

public finfo::set_flags(int $flags): bool

This function sets various Fileinfo options. Options can be set also directly in finfo_open() or other Fileinfo functions.

Parameters

finfo

An finfo instance, returned by finfo_open().

flags

One or disjunction of more Fileinfo constants.

Return Values

Always returns true.

Changelog

Version Description
8.1.0 The finfo parameter expects an finfo instance now; previously, a resource was expected.
