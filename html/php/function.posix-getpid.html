Description

posix_getpid(): int

Return the process identifier of the current process.

Parameters

This function has no parameters.

Return Values

Returns the identifier, as an int.

Examples

Example #1 Example use of posix_getpid()

<?php
echo posix_getpid(); //8805
?>
