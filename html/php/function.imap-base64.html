imap_base64

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_base64 — Decode BASE64 encoded text

Description

imap_base64(string $string): string|false

Decodes the given BASE-64 encoded string.

Parameters

string

The encoded text

Return Values

Returns the decoded message as a string, or false on failure.
