Parameters

This function has no parameters.

SolrDisMaxQuery

Examples

Example #1 SolrDisMaxQuery::useDisMaxQueryParser() example

<?php

$dismaxQuery = new SolrDisMaxQuery();
$dismaxQuery->useDisMaxQueryParser();
echo $dismaxQuery;
?>

The above example will output something similar to:

defType=dismax
