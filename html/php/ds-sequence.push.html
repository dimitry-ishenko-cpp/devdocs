Ds\Sequence::push

(PECL ds >= 1.0.0)

Ds\Sequence::push â€” Adds values to the end of the sequence

Description

abstract public Ds\Sequence::push(mixed ...$values): void

Adds values to the end of the sequence.

Parameters

values

The values to add.

Return Values

No value is returned.

Examples

Example #1 Ds\Sequence::push() example

<?php
$sequence = new \Ds\Vector();

$sequence->push("a");
$sequence->push("b");
$sequence->push("c", "d");
$sequence->push(...["e", "f"]);

print_r($sequence);
?>

The above example will output something similar to:

Ds\Vector Object
(
    [0] => a
    [1] => b
    [2] => c
    [3] => d
    [4] => e
    [5] => f
)
