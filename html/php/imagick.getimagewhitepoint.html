Description

public Imagick::getImageWhitePoint(): array

Returns the chromaticity white point as an associative array with the keys "x" and "y".

Parameters

This function has no parameters.

Return Values

Returns the chromaticity white point as an associative array with the keys "x" and "y".

Errors/Exceptions

Throws ImagickException on error.
