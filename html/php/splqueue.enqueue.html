Description

public SplQueue::enqueue(mixed $value): void

Enqueues value at the end of the queue.

Note:

SplQueue::enqueue() is an alias of SplDoublyLinkedList::push().

Parameters

value

The value to enqueue.

Return Values

No value is returned.
