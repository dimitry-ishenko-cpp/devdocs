The InternalIterator class

Introduction

(PHP 8)

Class to ease implementing IteratorAggregate for internal classes.

Class synopsis

final class InternalIterator implements Iterator {
/* Methods */
private __construct()
public current(): mixed
public key(): mixed
public next(): void
public rewind(): void
public valid(): bool
}
