Description

decbin(int $num): string

Returns a string containing a binary representation of the given num argument.

Parameters

num

Decimal value to convert

Examples

Example #1 decbin() example

<?php
echo decbin(12) . "\n";
echo decbin(26);
?>

The above example will output:

1100
11010
