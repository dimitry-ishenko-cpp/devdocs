The PDORow class

Introduction

(PHP 5 >= 5.1.0, PHP 7, PHP 8, PECL pdo >= 1.0.0)

Represents a row from a result set returned by PDOStatement::fetch() called with PDO_FETCH_LAZY fetch mode.

In addition to its default queryString property, the PDORow object creates properties that correspond to the column names returned in the result set as they are accessed.

Objects of this class cannot be instantiated.

Class synopsis

final class PDORow {
/* Properties */
public string $queryString;
}

Properties

queryString

Query string used by the PDOStatement that returned the PDORow object.
