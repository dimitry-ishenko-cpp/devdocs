Description

posix_getppid(): int

Return the process identifier of the parent process of the current process.

Parameters

This function has no parameters.

Return Values

Returns the identifier, as an int.

Examples

Example #1 Example use of posix_getppid()

<?php
echo posix_getppid(); //8259
?>
