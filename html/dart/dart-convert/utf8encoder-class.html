dart:convert

Utf8Encoder class

This class converts strings to their UTF-8 code units (a list of unsigned 8-bit integers).

Example:

final utf8Encoder = utf8.encoder;
const sample = 'Îñţérñåţîöñåļîžåţîờñ';
final encodedSample = utf8Encoder.convert(sample);
print(encodedSample);
