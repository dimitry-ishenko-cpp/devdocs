JsonEncoder encoder
override

Returns the encoder from S to T.

It may be stateful and should not be reused.

Implementation

JsonEncoder get encoder {
  if (_toEncodable == null) return const JsonEncoder();
  return JsonEncoder(_toEncodable);
}
