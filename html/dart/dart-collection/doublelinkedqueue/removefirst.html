removeFirst method

E removeFirst()
override

Removes and returns the first element of this queue.

The queue must not be empty when this method is called.

Implementation

E removeFirst() {
  // Hits sentinel's `_remove` if queue is empty.
  E result = _sentinel._nextLink!._remove();
  _elementCount--;
  return result;
}
