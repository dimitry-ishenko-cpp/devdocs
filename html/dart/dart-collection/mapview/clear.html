dart:collection

clear method

void clear()
override

Removes all entries from the map.

After this, the map is empty.

final planets = <int, String>{1: 'Mercury', 2: 'Venus', 3: 'Earth'};
planets.clear(); // {}

Implementation

void clear() {
  _map.clear();
}
