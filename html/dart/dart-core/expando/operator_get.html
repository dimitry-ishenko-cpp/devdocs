operator [] method

T? operator [](
  1. Object object
)

Gets the value of this Expando's property on the given object.

If the object hasn't been expanded, the result is the null value.

The object must not be a number, a string, a boolean, null, a dart:ffi pointer, a dart:ffi struct, or a dart:ffi union.
