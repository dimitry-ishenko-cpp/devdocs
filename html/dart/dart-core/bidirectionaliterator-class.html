BidirectionalIterator<E> class

An Iterator that allows moving backwards as well as forwards.

Deprecation note: This interface has turned out to not be valuable as a general reusable interface. There is still only one class implementing it, RuneIterator, and at least one other bidirectional iterator preferred a different name than movePrevious for the other direction. As such, this interface does not carry its own weight, and will be removed in a later release.
