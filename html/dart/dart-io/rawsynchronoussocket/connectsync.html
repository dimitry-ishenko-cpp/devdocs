connectSync method

RawSynchronousSocket connectSync(
  1. dynamic host,
  2. int port
)

Creates a new socket connection and returns a RawSynchronousSocket.

The host can either be a String or an InternetAddress. If host is a String, connectSync will perform a InternetAddress.lookup and try all returned InternetAddresses, until connected. Unless a connection was established, the error from the first failing connection is returned.
