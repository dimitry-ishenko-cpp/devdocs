write method

int write(
  1. List<int> buffer,
  2. [int offset = 0,
  3. int? count]
)

Writes up to count bytes of the buffer from offset buffer offset to the socket.

The number of successfully written bytes is returned. This function is non-blocking and will only write data if buffer space is available in the socket.

The default value for offset is 0, and the default value for count is buffer.length - offset.
