keybindings listen for platform

Get input from the user.

Signature

> keybindings listen {flags}

Input/output types:

input output
nothing nothing

Examples

Type and see key event codes

>keybindings listen

Notes

This is an internal debugging tool. For better output, try input listen --types [key]
