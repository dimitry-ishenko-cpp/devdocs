lines for filters

Converts input to lines.

Signature

> lines {flags}

Flags

  • --skip-empty, -s: skip empty lines

Input/output types:

input output
any list<string>

Examples

Split multi-line string into lines

>$"two\nlines"|lines
╭───┬───────╮
│0│two│
│1│lines│
╰───┴───────╯
