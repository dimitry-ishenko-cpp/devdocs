cd for filesystem

Change directory.

Signature

> cd {flags} (path)

Parameters

  • path: the path to change to

Input/output types:

input output
nothing nothing
string nothing

Examples

Change to your home directory

>cd~

Change to a directory via abbreviations

>cdd/s/9

Change to the previous working directory ($OLDPWD)

>cd-
