math log for math

Returns the logarithm for an arbitrary base.

Signature

> math log {flags} (base)

Parameters

  • base: Base for which the logarithm should be computed

Input/output types:

input output
list<number> list<number>
number number

Examples

Get the logarithm of 100 to the base 10

>100|math log10
2

Get the log2 of a list of values

> [1684] |math log2
╭───┬──────╮
│0│4.00│
│1│3.00│
│2│2.00│
╰───┴──────╯
