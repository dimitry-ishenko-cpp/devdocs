loop for core

Run a block in a loop.

Signature

> loop {flags} (block)

Parameters

  • block: block to loop

Input/output types:

input output
nothing nothing

Examples

Loop while a condition is true

>mutx=0;loop { if$x>10 { break };$x=$x+1 };$x
11
