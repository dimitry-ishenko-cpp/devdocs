date humanize for date

Print a 'humanized' format for the date, relative to now.

Signature

> date humanize {flags}

Input/output types:

input output
datetime string
string string

Examples

Print a 'humanized' format for the date, relative to now.

>"2021-10-22 20:00:12 +01:00"|date humanize
