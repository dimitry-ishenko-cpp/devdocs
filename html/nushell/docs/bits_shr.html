bits shr for bits

Bitwise shift right for integers.

Signature

> bits shr {flags} (bits)

Flags

  • --signed, -s: always treat input number as a signed number
  • --number-bytes, -n {string}: the word size in number of bytes, it can be 1, 2, 4, 8, auto, default value 8

Parameters

  • bits: number of bits to shift right

Input/output types:

input output
int int
list<int> list<int>

Examples

Shift right a number with 2 bits

>8|bits shr2
2

Shift right a list of numbers

> [15352] |bits shr2
╭───┬───╮
│0│3│
│1│8│
│2│0│
╰───┴───╯

Tips: Command bits shr was not included in the official binaries by default, you have to build it with --features=extra flag
