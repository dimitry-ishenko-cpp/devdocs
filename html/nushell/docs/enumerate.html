enumerate for filters

Enumerate the elements in a stream.

Signature

> enumerate {flags}

Input/output types:

input output
any table

Examples

Add an index to each element of a list

> [a, b, c] |enumerate
╭───┬──────╮
│# │ item │
├───┼──────┤
│0│a│
│1│b│
│2│c│
╰───┴──────╯
