to yaml for formats

Convert table into .yaml/.yml text.

Signature

> to yaml {flags}

Input/output types:

input output
any string

Examples

Outputs an YAML string representing the contents of this table

> [[foobar]; ["1""2"]] |to yaml
-foo:'1'
bar:'2'
