debug for debug

Debug print the value(s) piped in.

Signature

> debug {flags}

Flags

  • --raw, -r: Prints the raw value representation

Input/output types:

input output
any string
list<any> list<string>
table list<string>

Examples

Debug print a string

>'hello'|debug
hello

Debug print a list

> ['hello'] |debug
╭───┬───────╮
│0│hello│
╰───┴───────╯

Debug print a table

> [[versionpatch]; ['0.1.0'false] ['0.1.1'true] ['0.2.0'false]] |debug
╭───┬────────────────────────────────╮
│0│ {version:0.1.0, patch:false} │
│1│ {version:0.1.1, patch:true}  │
│2│ {version:0.2.0, patch:false} │
╰───┴────────────────────────────────╯
