overlay list for core

List all active overlays.

Signature

> overlay list {flags}

Input/output types:

input output
nothing list<string>

Examples

Get the last activated overlay

>modulespam { export deffoo [] { "foo" } }
overlay usespam
overlay list|last
spam

Notes

The overlays are listed in the order they were activated.
