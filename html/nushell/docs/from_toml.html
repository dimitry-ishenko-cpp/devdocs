from toml for formats

Parse text as .toml and create record.

Signature

> from toml {flags}

Input/output types:

input output
string record

Examples

Converts toml formatted string to record

>'a = 1'|from toml
╭───┬───╮
│a│1│
╰───┴───╯

Converts toml formatted string to record

>'a = 1
b = [1, 2]'|from toml
╭───┬───────────╮
│a│1│
││╭───┬───╮│
│b││0│1││
│││1│2││
││╰───┴───╯│
╰───┴───────────╯
