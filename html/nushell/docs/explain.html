explain for debug

Explain closure contents.

Signature

> explain {flags} (closure)

Parameters

  • closure: the closure to run

Input/output types:

input output
any any
nothing any

Examples

Explain a command within a closure

>explain {|| ls|sort-bynametype-i|getname } |table-e
