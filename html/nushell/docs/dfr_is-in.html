dfr is-in for expression

Creates an is-in expression.

Signature

> dfr is-in {flags} (list)

Parameters

  • list: List to check if values are in

Input/output types:

input output
any any

Examples

Creates a is-in expression

>letdf= ([[ab]; [one1] [two2] [three3]] |dfr into-df);
$df|dfr with-column (dfr cola|dfr is-in [onetwo] |dfr asa_in)
╭───┬───────┬───┬───────╮
│# │   a   │ b │ a_in  │
├───┼───────┼───┼───────┤
│0│one│1│true│
│1│two│2│true│
│2│three│3│false│
╰───┴───────┴───┴───────╯

Tips: Dataframe commands were not shipped in the official binaries by default, you have to build it with --features=dataframe flag
