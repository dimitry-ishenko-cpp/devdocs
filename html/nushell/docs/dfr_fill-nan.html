dfr fill-nan for lazyframe

Replaces NaN values with the given expression.

Signature

> dfr fill-nan {flags} (fill)

Parameters

  • fill: Expression to use to fill the NAN values

Input/output types:

input output
any any

Examples

Fills the NaN values with 0

> [12NaN3NaN] |dfr into-df|dfr fill-nan0
╭───┬───╮
│# │ 0 │
├───┼───┤
│0│1│
│1│2│
│2│0│
│3│3│
│4│0│
╰───┴───╯

Fills the NaN values of a whole dataframe

> [[ab]; [0.21] [0.1NaN]] |dfr into-df|dfr fill-nan0
╭───┬──────┬───╮
│# │  a   │ b │
├───┼──────┼───┤
│0│0.20│1│
│1│0.10│0│
╰───┴──────┴───╯

Tips: Dataframe commands were not shipped in the official binaries by default, you have to build it with --features=dataframe flag
