skip for filters

Skip the first several rows of the input. Counterpart of `drop`. Opposite of `first`.

Signature

> skip {flags} (n)

Parameters

  • n: the number of elements to skip

Input/output types:

input output
list<any> list<any>
table table

Examples

Skip the first value of a list

> [2468] |skip1
╭───┬───╮
│0│4│
│1│6│
│2│8│
╰───┴───╯

Skip two rows of a table

> [[editions]; [2015] [2018] [2021]] |skip2
╭───┬──────────╮
│# │ editions │
├───┼──────────┤
│0│2021│
╰───┴──────────╯

Notes

To skip specific numbered rows, try drop nth. To skip specific named columns, try reject.

Subcommands:

name type usage
skip until Builtin Skip elements of the input until a predicate is true.
skip while Builtin Skip elements of the input while a predicate is true.
