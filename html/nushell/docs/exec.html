exec for system

Execute a command, replacing the current process.

Signature

> exec {flags} (command)

Parameters

  • command: the command to execute

Input/output types:

input output
nothing any

Examples

Execute external 'ps aux' tool

>execpsaux

Execute 'nautilus'

>execnautilus

Notes

Currently supported only on Unix-based systems.
