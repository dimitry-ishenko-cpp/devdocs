random bool for random

Generate a random boolean value.

Signature

> random bool {flags}

Flags

  • --bias, -b {number}: Adjusts the probability of a "true" outcome

Input/output types:

input output
nothing bool

Examples

Generate a random boolean value

>random bool

Generate a random boolean value with a 75% chance of "true"

>random bool--bias0.75
