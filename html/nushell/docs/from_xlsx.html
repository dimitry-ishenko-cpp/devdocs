from xlsx for formats

Parse binary Excel(.xlsx) data and create table.

Signature

> from xlsx {flags}

Flags

  • --sheets, -s {list<string>}: Only convert specified sheets

Input/output types:

input output
binary table

Examples

Convert binary .xlsx data to a table

>open--rawtest.xlsx|from xlsx

Convert binary .xlsx data to a table, specifying the tables

>open--rawtest.xlsx|from xlsx-s [Spreadsheet1]
