from vcf for formats

Parse text as .vcf and create table.

Signature

> from vcf

Input/output types:

input output
string table

Examples

Converts ics formatted string to table

>'BEGIN:VCARD
N:Foo
FN:Bar
EMAIL:foo@bar.com
END:VCARD'|from vcf
