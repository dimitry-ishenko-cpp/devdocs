from yaml for formats

Parse text as .yaml/.yml and create table.

Signature

> from yaml {flags}

Input/output types:

input output
string any

Examples

Converts yaml formatted string to table

>'a: 1'|from yaml
╭───┬───╮
│a│1│
╰───┴───╯

Converts yaml formatted string to table

>'[ a: 1, b: [1, 2] ]'|from yaml
╭───┬────┬───────────╮
│# │ a  │     b     │
├───┼────┼───────────┤
│0│1│❎│
│1│❎│╭───┬───╮│
││││0│1││
││││1│2││
│││╰───┴───╯│
╰───┴────┴───────────╯
