metadata for debug

Get the metadata for items in the stream.

Signature

> metadata {flags} (expression)

Parameters

  • expression: the expression you want metadata for

Input/output types:

input output
any record

Examples

Get the metadata of a variable

>leta=42;metadata$a

Get the metadata of the input

>ls|metadata
