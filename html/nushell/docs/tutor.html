tutor for misc

Run the tutorial. To begin, run: tutor.

Signature

> tutor {flags} (search)

Flags

  • --find, -f {string}: Search tutorial for a phrase

Parameters

  • search: item to search for, or 'list' to list available tutorials

Input/output types:

input output
nothing string

Examples

Begin the tutorial

>tutorbegin

Search a tutorial by phrase

>tutor-f"$in"
