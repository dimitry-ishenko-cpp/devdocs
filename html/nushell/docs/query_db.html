query db for database

Query a database using SQL.

Signature

> query db {flags} (SQL)

Parameters

  • SQL: SQL to execute against the database

Input/output types:

input output
any any

Examples

Execute SQL against a SQLite database

>openfoo.db|query db"SELECT * FROM Bar"
