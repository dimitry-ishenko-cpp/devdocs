pandas.Timestamp.is_year_end

Timestamp.is_year_end

Return True if date is last day of the year.

Examples

>>> ts = pd.Timestamp(2020, 3, 14)
>>> ts.is_year_end
False

>>> ts = pd.Timestamp(2020, 12, 31)
>>> ts.is_year_end
True
