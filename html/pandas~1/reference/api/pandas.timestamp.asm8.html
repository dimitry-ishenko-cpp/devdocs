pandas.Timestamp.asm8

Timestamp.asm8

Return numpy datetime64 format in nanoseconds.

Examples

>>> ts = pd.Timestamp(2020, 3, 14, 15)
>>> ts.asm8
numpy.datetime64('2020-03-14T15:00:00.000000000')
