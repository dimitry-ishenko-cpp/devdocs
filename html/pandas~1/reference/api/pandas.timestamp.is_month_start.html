pandas.Timestamp.is_month_start

Timestamp.is_month_start

Return True if date is first day of month.

Examples

>>> ts = pd.Timestamp(2020, 3, 14)
>>> ts.is_month_start
False

>>> ts = pd.Timestamp(2020, 1, 1)
>>> ts.is_month_start
True
