pandas.Timestamp.utcfromtimestamp

classmethodTimestamp.utcfromtimestamp(ts)

Construct a naive UTC datetime from a POSIX timestamp.

Examples

>>> pd.Timestamp.utcfromtimestamp(1584199972)
Timestamp('2020-03-14 15:32:52')
