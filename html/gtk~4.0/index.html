GTK 4 Reference Manual

This document is for the GTK 4 library, version 4.0.3 . The latest versions can be found online at https://developer.gnome.org/gtk4/. If you are looking for the older GTK 3 series of libraries, see https://developer.gnome.org/gtk3/.
