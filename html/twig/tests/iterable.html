iterable

iterable checks if a variable is an array or a traversable object:

{# evaluates to true if the foo variable is iterable #}
{% if users is iterable %}
{% for user in users %}
        Hello {{ user }}!
{% endfor %}
{% else %}
{# users is probably a string #}
    Hello {{ users }}!
{% endif %}
