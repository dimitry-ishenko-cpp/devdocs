max

Return the latest of the given dates.

Description

Return the latest of the given dates.

Usage

// CommonJS
var max = require('date-fns/max')
// ES 2015
import max from 'date-fns/max'
// ESM
import { max } from 'date-fns'

Syntax

max(datesArray)

Arguments

Name Description
datesArray

the dates to compare

Returns

Description

the latest of the dates

Exceptions

Type Description
TypeError

1 argument required

Examples

// Which of these dates is the latest?
const result = max([
  new Date(1989, 6, 10),
  new Date(1987, 1, 11),
  new Date(1995, 6, 2),
  new Date(1990, 0, 1)
])
//=> Sun Jul 02 1995 00:00:00
