differenceInCalendarISOWeeks

Get the number of calendar ISO weeks between the given dates.

Description

Get the number of calendar ISO weeks between the given dates.

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var differenceInCalendarISOWeeks = require('date-fns/differenceInCalendarISOWeeks')
// ES 2015
import differenceInCalendarISOWeeks from 'date-fns/differenceInCalendarISOWeeks'
// ESM
import { differenceInCalendarISOWeeks } from 'date-fns'

Syntax

differenceInCalendarISOWeeks(dateLeft, dateRight)

Arguments

Name Description
dateLeft

the later date
dateRight

the earlier date

Returns

Description

the number of calendar ISO weeks

Exceptions

Type Description
TypeError

2 arguments required

Examples

// How many calendar ISO weeks are between 6 July 2014 and 21 July 2014?
const result = differenceInCalendarISOWeeks(
  new Date(2014, 6, 21),
  new Date(2014, 6, 6)
)
//=> 3
