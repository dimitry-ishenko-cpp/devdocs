isThisQuarter

Is the given date in the same quarter as the current date?

Description

Is the given date in the same quarter as the current date?

⚠️ Please note that this function is not present in the FP submodule as it uses Date.now() internally hence impure and can't be safely curried.

Usage

// CommonJS
var isThisQuarter = require('date-fns/isThisQuarter')
// ES 2015
import isThisQuarter from 'date-fns/isThisQuarter'
// ESM
import { isThisQuarter } from 'date-fns'

Syntax

isThisQuarter(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is in this quarter

Exceptions

Type Description
TypeError

1 argument required

Examples

// If today is 25 September 2014, is 2 July 2014 in this quarter?
const result = isThisQuarter(new Date(2014, 6, 2))
//=> true
