quartersToMonths

Convert number of quarters to months.

Description

Convert a number of quarters to a full number of months.

Usage

// CommonJS
var quartersToMonths = require('date-fns/quartersToMonths')
// ES 2015
import quartersToMonths from 'date-fns/quartersToMonths'
// ESM
import { quartersToMonths } from 'date-fns'

Syntax

quartersToMonths(quarters)

Arguments

Name Description
quarters

number of quarters to be converted

Returns

Description

the number of quarters converted in months

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 2 quarters to months
const result = quartersToMonths(2)
//=> 6
