setISODay

Set the day of the ISO week to the given date.

Description

Set the day of the ISO week to the given date. ISO week starts with Monday. 7 is the index of Sunday, 1 is the index of Monday etc.

Usage

// CommonJS
var setISODay = require('date-fns/setISODay')
// ES 2015
import setISODay from 'date-fns/setISODay'
// ESM
import { setISODay } from 'date-fns'

Syntax

setISODay(date, day)

Arguments

Name Description
date

the date to be changed
day

the day of the ISO week of the new date

Returns

Description

the new date with the day of the ISO week set

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Set Sunday to 1 September 2014:
const result = setISODay(new Date(2014, 8, 1), 7)
//=> Sun Sep 07 2014 00:00:00
