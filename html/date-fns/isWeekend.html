isWeekend

Does the given date fall on a weekend?

Description

Does the given date fall on a weekend?

Usage

// CommonJS
var isWeekend = require('date-fns/isWeekend')
// ES 2015
import isWeekend from 'date-fns/isWeekend'
// ESM
import { isWeekend } from 'date-fns'

Syntax

isWeekend(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date falls on a weekend

Exceptions

Type Description
TypeError

1 argument required

Examples

// Does 5 October 2014 fall on a weekend?
const result = isWeekend(new Date(2014, 9, 5))
//=> true
