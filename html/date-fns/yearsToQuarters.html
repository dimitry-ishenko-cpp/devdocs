yearsToQuarters

Convert years to quarters.

Description

Convert a number of years to a full number of quarters.

Usage

// CommonJS
var yearsToQuarters = require('date-fns/yearsToQuarters')
// ES 2015
import yearsToQuarters from 'date-fns/yearsToQuarters'
// ESM
import { yearsToQuarters } from 'date-fns'

Syntax

yearsToQuarters(years)

Arguments

Name Description
years

number of years to be converted

Returns

Description

the number of years converted in quarters

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 2 years to quarters
const result = yearsToQuarters(2)
//=> 8
