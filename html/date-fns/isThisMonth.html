isThisMonth

Is the given date in the same month as the current date?

Description

Is the given date in the same month as the current date?

⚠️ Please note that this function is not present in the FP submodule as it uses Date.now() internally hence impure and can't be safely curried.

Usage

// CommonJS
var isThisMonth = require('date-fns/isThisMonth')
// ES 2015
import isThisMonth from 'date-fns/isThisMonth'
// ESM
import { isThisMonth } from 'date-fns'

Syntax

isThisMonth(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is in this month

Exceptions

Type Description
TypeError

1 argument required

Examples

// If today is 25 September 2014, is 15 September 2014 in this month?
const result = isThisMonth(new Date(2014, 8, 15))
//=> true
