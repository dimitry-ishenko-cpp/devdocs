nextThursday

When is the next Thursday?

Description

When is the next Thursday?

Usage

// CommonJS
var nextThursday = require('date-fns/nextThursday')
// ES 2015
import nextThursday from 'date-fns/nextThursday'
// ESM
import { nextThursday } from 'date-fns'

Syntax

nextThursday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the next Thursday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the next Thursday after Mar, 22, 2020?
const result = nextThursday(new Date(2020, 2, 22))
//=> Thur Mar 26 2020 00:00:00
