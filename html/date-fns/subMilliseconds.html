subMilliseconds

Subtract the specified number of milliseconds from the given date.

Description

Subtract the specified number of milliseconds from the given date.

Usage

// CommonJS
var subMilliseconds = require('date-fns/subMilliseconds')
// ES 2015
import subMilliseconds from 'date-fns/subMilliseconds'
// ESM
import { subMilliseconds } from 'date-fns'

Syntax

subMilliseconds(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of milliseconds to be subtracted. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the milliseconds subtracted

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Subtract 750 milliseconds from 10 July 2014 12:45:30.000:
const result = subMilliseconds(new Date(2014, 6, 10, 12, 45, 30, 0), 750)
//=> Thu Jul 10 2014 12:45:29.250
