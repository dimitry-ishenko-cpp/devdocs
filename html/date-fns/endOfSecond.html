endOfSecond

Return the end of a second for the given date.

Description

Return the end of a second for the given date. The result will be in the local timezone.

Usage

// CommonJS
var endOfSecond = require('date-fns/endOfSecond')
// ES 2015
import endOfSecond from 'date-fns/endOfSecond'
// ESM
import { endOfSecond } from 'date-fns'

Syntax

endOfSecond(date)

Arguments

Name Description
date

the original date

Returns

Description

the end of a second

Exceptions

Type Description
TypeError

1 argument required

Examples

// The end of a second for 1 December 2014 22:15:45.400:
const result = endOfSecond(new Date(2014, 11, 1, 22, 15, 45, 400))
//=> Mon Dec 01 2014 22:15:45.999
