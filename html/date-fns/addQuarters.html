addQuarters

Add the specified number of year quarters to the given date.

Description

Add the specified number of year quarters to the given date.

Usage

// CommonJS
var addQuarters = require('date-fns/addQuarters')
// ES 2015
import addQuarters from 'date-fns/addQuarters'
// ESM
import { addQuarters } from 'date-fns'

Syntax

addQuarters(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of quarters to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the quarters added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 1 quarter to 1 September 2014:
const result = addQuarters(new Date(2014, 8, 1), 1)
//=> Mon Dec 01 2014 00:00:00
