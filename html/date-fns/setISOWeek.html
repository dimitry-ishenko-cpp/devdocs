setISOWeek

Set the ISO week to the given date.

Description

Set the ISO week to the given date, saving the weekday number.

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var setISOWeek = require('date-fns/setISOWeek')
// ES 2015
import setISOWeek from 'date-fns/setISOWeek'
// ESM
import { setISOWeek } from 'date-fns'

Syntax

setISOWeek(date, isoWeek)

Arguments

Name Description
date

the date to be changed
isoWeek

the ISO week of the new date

Returns

Description

the new date with the ISO week set

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Set the 53rd ISO week to 7 August 2004:
const result = setISOWeek(new Date(2004, 7, 7), 53)
//=> Sat Jan 01 2005 00:00:00
