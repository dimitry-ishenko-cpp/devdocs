setISOWeekYear

Set the ISO week-numbering year to the given date.

Description

Set the ISO week-numbering year to the given date, saving the week number and the weekday number.

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var setISOWeekYear = require('date-fns/setISOWeekYear')
// ES 2015
import setISOWeekYear from 'date-fns/setISOWeekYear'
// ESM
import { setISOWeekYear } from 'date-fns'

Syntax

setISOWeekYear(date, isoWeekYear)

Arguments

Name Description
date

the date to be changed
isoWeekYear

the ISO week-numbering year of the new date

Returns

Description

the new date with the ISO week-numbering year set

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Set ISO week-numbering year 2007 to 29 December 2008:
const result = setISOWeekYear(new Date(2008, 11, 29), 2007)
//=> Mon Jan 01 2007 00:00:00
