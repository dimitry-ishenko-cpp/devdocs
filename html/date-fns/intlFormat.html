intlFormat

Format the date with Intl.DateTimeFormat (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/DateTimeFormat).

Description

Return the formatted date string in the given format. The method uses Intl.DateTimeFormat inside. formatOptions are the same as Intl.DateTimeFormat options

⚠️ Please note that before Node version 13.0.0, only the locale data for en-US is available by default.

Usage

// CommonJS
var intlFormat = require('date-fns/intlFormat')
// ES 2015
import intlFormat from 'date-fns/intlFormat'
// ESM
import { intlFormat } from 'date-fns'

Syntax

intlFormat(argument, [formatOptions], [localeOptions])

Arguments

Name Description
argument

the original date.
formatOptions

an object with options.
formatOptions.localeMatcher

locale selection algorithm.
formatOptions.weekday

representation the days of the week.
formatOptions.era

representation of eras.
formatOptions.year

representation of years.
formatOptions.month

representation of month.
formatOptions.day

representation of day.
formatOptions.hour

representation of hours.
formatOptions.minute

representation of minutes.
formatOptions.second

representation of seconds.
formatOptions.timeZoneName

representation of names of time zones.
formatOptions.formatMatcher

format selection algorithm.
formatOptions.hour12

determines whether to use 12-hour time format.
formatOptions.timeZone

the time zone to use.
localeOptions

an object with locale.
localeOptions.locale

the locale code

Returns

Description

the formatted date string.

Exceptions

Type Description
TypeError

1 argument required.
RangeError

date must not be Invalid Date

Examples

// Represent 10 October 2019 in German.
// Convert the date with format's options and locale's options.
const result = intlFormat(new Date(2019, 9, 4, 12, 30, 13, 456), {
     weekday: 'long',
     year: 'numeric',
     month: 'long',
     day: 'numeric',
   }, {
     locale: 'de-DE',
 })
//=> Freitag, 4. Oktober 2019
// Represent 10 October 2019.
// Convert the date with format's options.
const result = intlFormat.default(new Date(2019, 9, 4, 12, 30, 13, 456), {
     year: 'numeric',
     month: 'numeric',
     day: 'numeric',
     hour: 'numeric',
 })
//=> 10/4/2019, 12 PM
// Represent 10 October 2019 in Korean.
// Convert the date with locale's options.
const result = intlFormat(new Date(2019, 9, 4, 12, 30, 13, 456), {
     locale: 'ko-KR',
 })
//=> 2019. 10. 4.
// Represent 10 October 2019 in middle-endian format:
const result = intlFormat(new Date(2019, 9, 4, 12, 30, 13, 456))
//=> 10/4/2019
