addMinutes

Add the specified number of minutes to the given date.

Description

Add the specified number of minutes to the given date.

Usage

// CommonJS
var addMinutes = require('date-fns/addMinutes')
// ES 2015
import addMinutes from 'date-fns/addMinutes'
// ESM
import { addMinutes } from 'date-fns'

Syntax

addMinutes(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of minutes to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the minutes added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 30 minutes to 10 July 2014 12:00:00:
const result = addMinutes(new Date(2014, 6, 10, 12, 0), 30)
//=> Thu Jul 10 2014 12:30:00
