differenceInCalendarISOWeekYears

Get the number of calendar ISO week-numbering years between the given dates.

Description

Get the number of calendar ISO week-numbering years between the given dates.

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var differenceInCalendarISOWeekYears = require('date-fns/differenceInCalendarISOWeekYears')
// ES 2015
import differenceInCalendarISOWeekYears from 'date-fns/differenceInCalendarISOWeekYears'
// ESM
import { differenceInCalendarISOWeekYears } from 'date-fns'

Syntax

differenceInCalendarISOWeekYears(dateLeft, dateRight)

Arguments

Name Description
dateLeft

the later date
dateRight

the earlier date

Returns

Description

the number of calendar ISO week-numbering years

Exceptions

Type Description
TypeError

2 arguments required

Examples

// How many calendar ISO week-numbering years are 1 January 2010 and 1 January 2012?
const result = differenceInCalendarISOWeekYears(
  new Date(2012, 0, 1),
  new Date(2010, 0, 1)
)
//=> 2
