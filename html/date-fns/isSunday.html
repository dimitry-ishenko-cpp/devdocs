isSunday

Is the given date Sunday?

Description

Is the given date Sunday?

Usage

// CommonJS
var isSunday = require('date-fns/isSunday')
// ES 2015
import isSunday from 'date-fns/isSunday'
// ESM
import { isSunday } from 'date-fns'

Syntax

isSunday(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is Sunday

Exceptions

Type Description
TypeError

1 argument required

Examples

// Is 21 September 2014 Sunday?
const result = isSunday(new Date(2014, 8, 21))
//=> true
