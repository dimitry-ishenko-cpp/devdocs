previousTuesday

When is the previous Tuesday?

Description

When is the previous Tuesday?

Usage

// CommonJS
var previousTuesday = require('date-fns/previousTuesday')
// ES 2015
import previousTuesday from 'date-fns/previousTuesday'
// ESM
import { previousTuesday } from 'date-fns'

Syntax

previousTuesday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the previous Tuesday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the previous Tuesday before Jun, 18, 2021?
const result = previousTuesday(new Date(2021, 5, 18))
//=> Tue June 15 2021 00:00:00
