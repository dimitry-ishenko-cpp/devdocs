subISOWeekYears

Subtract the specified number of ISO week-numbering years from the given date.

Description

Subtract the specified number of ISO week-numbering years from the given date.

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var subISOWeekYears = require('date-fns/subISOWeekYears')
// ES 2015
import subISOWeekYears from 'date-fns/subISOWeekYears'
// ESM
import { subISOWeekYears } from 'date-fns'

Syntax

subISOWeekYears(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of ISO week-numbering years to be subtracted. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the ISO week-numbering years subtracted

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Subtract 5 ISO week-numbering years from 1 September 2014:
const result = subISOWeekYears(new Date(2014, 8, 1), 5)
//=> Mon Aug 31 2009 00:00:00
