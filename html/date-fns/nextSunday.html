nextSunday

When is the next Sunday?

Description

When is the next Sunday?

Usage

// CommonJS
var nextSunday = require('date-fns/nextSunday')
// ES 2015
import nextSunday from 'date-fns/nextSunday'
// ESM
import { nextSunday } from 'date-fns'

Syntax

nextSunday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the next Sunday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the next Sunday after Mar, 22, 2020?
const result = nextSunday(new Date(2020, 2, 22))
//=> Sun Mar 29 2020 00:00:00
