yearsToMonths

Convert years to months.

Description

Convert a number of years to a full number of months.

Usage

// CommonJS
var yearsToMonths = require('date-fns/yearsToMonths')
// ES 2015
import yearsToMonths from 'date-fns/yearsToMonths'
// ESM
import { yearsToMonths } from 'date-fns'

Syntax

yearsToMonths(years)

Arguments

Name Description
years

number of years to be converted

Returns

Description

the number of years converted in months

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 2 years into months
const result = yearsToMonths(2)
//=> 24
