previousSaturday

When is the previous Saturday?

Description

When is the previous Saturday?

Usage

// CommonJS
var previousSaturday = require('date-fns/previousSaturday')
// ES 2015
import previousSaturday from 'date-fns/previousSaturday'
// ESM
import { previousSaturday } from 'date-fns'

Syntax

previousSaturday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the previous Saturday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the previous Saturday before Jun, 20, 2021?
const result = previousSaturday(new Date(2021, 5, 20))
//=> Sat June 19 2021 00:00:00
