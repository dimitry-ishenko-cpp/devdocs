nextFriday

When is the next Friday?

Description

When is the next Friday?

Usage

// CommonJS
var nextFriday = require('date-fns/nextFriday')
// ES 2015
import nextFriday from 'date-fns/nextFriday'
// ESM
import { nextFriday } from 'date-fns'

Syntax

nextFriday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the next Friday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the next Friday after Mar, 22, 2020?
const result = nextFriday(new Date(2020, 2, 22))
//=> Fri Mar 27 2020 00:00:00
