weeksToDays

Convert weeks to days.

Description

Convert a number of weeks to a full number of days.

Usage

// CommonJS
var weeksToDays = require('date-fns/weeksToDays')
// ES 2015
import weeksToDays from 'date-fns/weeksToDays'
// ESM
import { weeksToDays } from 'date-fns'

Syntax

weeksToDays(weeks)

Arguments

Name Description
weeks

number of weeks to be converted

Returns

Description

the number of weeks converted in days

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 2 weeks into days
const result = weeksToDays(2)
//=> 14
