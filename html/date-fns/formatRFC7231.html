formatRFC7231

Format the date according to the RFC 7231 standard (https://tools.ietf.org/html/rfc7231#section-7.1.1.1).

Description

Return the formatted date string in RFC 7231 format. The result will always be in UTC timezone.

Usage

// CommonJS
var formatRFC7231 = require('date-fns/formatRFC7231')
// ES 2015
import formatRFC7231 from 'date-fns/formatRFC7231'
// ESM
import { formatRFC7231 } from 'date-fns'

Syntax

formatRFC7231(date)

Arguments

Name Description
date

the original date

Returns

Description

the formatted date string

Exceptions

Type Description
TypeError

1 argument required
RangeError

date must not be Invalid Date

Examples

// Represent 18 September 2019 in RFC 7231 format:
const result = formatRFC7231(new Date(2019, 8, 18, 19, 0, 52))
//=> 'Wed, 18 Sep 2019 19:00:52 GMT'
