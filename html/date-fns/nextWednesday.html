nextWednesday

When is the next Wednesday?

Description

When is the next Wednesday?

Usage

// CommonJS
var nextWednesday = require('date-fns/nextWednesday')
// ES 2015
import nextWednesday from 'date-fns/nextWednesday'
// ESM
import { nextWednesday } from 'date-fns'

Syntax

nextWednesday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the next Wednesday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the next Wednesday after Mar, 22, 2020?
const result = nextWednesday(new Date(2020, 2, 22))
//=> Wed Mar 25 2020 00:00:00
