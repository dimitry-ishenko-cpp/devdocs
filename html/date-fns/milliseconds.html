milliseconds

Returns the number of milliseconds in the specified, years, months, weeks, days, hours, minutes and seconds.

Description

Returns the number of milliseconds in the specified, years, months, weeks, days, hours, minutes and seconds.

One years equals 365.2425 days according to the formula:

Leap year occures every 4 years, except for years that are divisable by 100 and not divisable by 400. 1 mean year = (365+1/4-1/100+1/400) days = 365.2425 days

One month is a year divided by 12.

Usage

// CommonJS
var milliseconds = require('date-fns/milliseconds')
// ES 2015
import milliseconds from 'date-fns/milliseconds'
// ESM
import { milliseconds } from 'date-fns'

Syntax

milliseconds(duration)

Arguments

Name Description
duration

the object with years, months, weeks, days, hours, minutes and seconds to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the milliseconds

Exceptions

Type Description
TypeError

1 argument required

Examples

// 1 year in milliseconds
milliseconds({ years: 1 })
//=> 31556952000

// 3 months in milliseconds
milliseconds({ months: 3 })
//=> 7889238000
