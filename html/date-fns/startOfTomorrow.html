startOfTomorrow

Return the start of tomorrow.

Description

Return the start of tomorrow.

⚠️ Please note that this function is not present in the FP submodule as it uses new Date() internally hence impure and can't be safely curried.

Usage

// CommonJS
var startOfTomorrow = require('date-fns/startOfTomorrow')
// ES 2015
import startOfTomorrow from 'date-fns/startOfTomorrow'
// ESM
import { startOfTomorrow } from 'date-fns'

Returns

Description

the start of tomorrow

Examples

// If today is 6 October 2014:
const result = startOfTomorrow()
//=> Tue Oct 7 2014 00:00:00
