previousWednesday

When is the previous Wednesday?

Description

When is the previous Wednesday?

Usage

// CommonJS
var previousWednesday = require('date-fns/previousWednesday')
// ES 2015
import previousWednesday from 'date-fns/previousWednesday'
// ESM
import { previousWednesday } from 'date-fns'

Syntax

previousWednesday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the previous Wednesday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the previous Wednesday before Jun, 18, 2021?
const result = previousWednesday(new Date(2021, 5, 18))
//=> Wed June 16 2021 00:00:00
