isValid

Is the given date valid?

Description

Returns false if argument is Invalid Date and true otherwise. Argument is converted to Date using toDate. See [toDate]{@link https://date-fns.org/docs/toDate} Invalid Date is a Date, whose time value is NaN.

Time value of Date: http://es5.github.io/#x15.9.1.1

Usage

// CommonJS
var isValid = require('date-fns/isValid')
// ES 2015
import isValid from 'date-fns/isValid'
// ESM
import { isValid } from 'date-fns'

Syntax

isValid(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is valid

Exceptions

Type Description
TypeError

1 argument required

Examples

// For the valid date:
const result = isValid(new Date(2014, 1, 31))
//=> true
// For the value, convertable into a date:
const result = isValid(1393804800000)
//=> true
// For the invalid date:
const result = isValid(new Date(''))
//=> false
