isSaturday

Is the given date Saturday?

Description

Is the given date Saturday?

Usage

// CommonJS
var isSaturday = require('date-fns/isSaturday')
// ES 2015
import isSaturday from 'date-fns/isSaturday'
// ESM
import { isSaturday } from 'date-fns'

Syntax

isSaturday(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is Saturday

Exceptions

Type Description
TypeError

1 argument required

Examples

// Is 27 September 2014 Saturday?
const result = isSaturday(new Date(2014, 8, 27))
//=> true
