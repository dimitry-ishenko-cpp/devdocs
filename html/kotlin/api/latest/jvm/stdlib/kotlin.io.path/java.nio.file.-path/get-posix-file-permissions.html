getPosixFilePermissions

fun Path.getPosixFilePermissions(
    vararg options: LinkOption
): Set<PosixFilePermission>

Returns the POSIX file permissions of the file located by this path.

Exceptions

UnsupportedOperationException - if the associated file system does not support the PosixFileAttributeView.
