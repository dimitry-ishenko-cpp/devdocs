fold

kotlin-stdlib / kotlin.sequences / fold
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
inline fun <T, R> Sequence<T>.fold(
    initial: R, 
    operation: (acc: R, T) -> R
): R

Accumulates value starting with initial value and applying operation from left to right to current accumulator value and each element.

Returns the specified initial value if the sequence is empty.

Parameters

operation -

function that takes current accumulator value and an element, and calculates the next accumulator value.

The operation is terminal.
