ln

kotlin-stdlib / kotlin.math / ln
Platform and version requirements: JVM (1.2), JS (1.2), Native (1.2) 
fun ln(x: Double): Double
fun ln(x: Float): Float

Computes the natural logarithm (base E) of the value x.

Special cases:

  • ln(NaN) is NaN
  • ln(x) is NaN when x < 0.0
  • ln(+Inf) is +Inf
  • ln(0.0) is -Inf
