sinh

kotlin-stdlib / kotlin.math / sinh
Platform and version requirements: JVM (1.2), JS (1.2), Native (1.2) 
fun sinh(x: Double): Double
fun sinh(x: Float): Float

Computes the hyperbolic sine of the value x.

Special cases:

  • sinh(NaN) is NaN
  • sinh(+Inf) is +Inf
  • sinh(-Inf) is -Inf
