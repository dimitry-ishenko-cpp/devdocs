tanh

kotlin-stdlib / kotlin.math / tanh
Platform and version requirements: JVM (1.2), JS (1.2), Native (1.2) 
fun tanh(x: Double): Double
fun tanh(x: Float): Float

Computes the hyperbolic tangent of the value x.

Special cases:

  • tanh(NaN) is NaN
  • tanh(+Inf) is 1.0
  • tanh(-Inf) is -1.0
