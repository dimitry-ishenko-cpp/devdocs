kotlin-stdlib / kotlin / ULong / toFloat
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
fun toFloat(): Float

Converts this ULong value to Float.

The resulting value is the closest Float to this ULong value. In case when this ULong value is exactly between two Floats, the one with zero at least significant bit of mantissa is selected.
