ensureNeverFrozen

kotlin-stdlib / kotlin.native.concurrent / ensureNeverFrozen
Platform and version requirements: Native (1.3) 
fun Any.ensureNeverFrozen()

This function ensures that if we see such an object during freezing attempt - freeze fails and FreezingException is thrown.

Exceptions

FreezingException - thrown immediately if this object is already frozen
