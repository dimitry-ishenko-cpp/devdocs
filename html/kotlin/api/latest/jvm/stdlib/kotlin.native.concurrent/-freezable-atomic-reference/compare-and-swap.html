compareAndSwap

kotlin-stdlib / kotlin.native.concurrent / FreezableAtomicReference / compareAndSwap
Platform and version requirements: Native (1.3) 
fun compareAndSwap(expected: T, newValue: T): T

Atomically sets the value to the given new value if the current value equals the expected value and returns the old value in any case.

Legacy MM: If the new value value is not null and object is frozen, it must be frozen or permanent object.

Parameters

expected - the expected value

newValue - the new value

Exceptions

InvalidMutabilityException - with legacy MM if the value is not frozen or a permanent object

Return the old value
