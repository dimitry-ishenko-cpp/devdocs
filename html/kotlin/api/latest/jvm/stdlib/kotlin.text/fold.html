fold

kotlin-stdlib / kotlin.text / fold
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
inline fun <R> CharSequence.fold(
    initial: R, 
    operation: (acc: R, Char) -> R
): R

Accumulates value starting with initial value and applying operation from left to right to current accumulator value and each character.

Returns the specified initial value if the char sequence is empty.

Parameters

operation - function that takes current accumulator value and a character, and calculates the next accumulator value.
