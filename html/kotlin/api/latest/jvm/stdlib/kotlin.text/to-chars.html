toChars

kotlin-stdlib / kotlin.text / toChars
Platform and version requirements: Native (1.3) 
@ExperimentalNativeApi fun Char.Companion.toChars(
    codePoint: Int
): CharArray

Converts the codepoint specified to a char array. If the codepoint is not supplementary, the method will return an array with one element otherwise it will return an array A with a high surrogate in A0 and a low surrogate in A1.

Note that this function is unstable. In the future it could be deprecated in favour of an overload that would accept a CodePoint type.
