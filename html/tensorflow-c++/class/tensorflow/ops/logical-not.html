tensorflow::ops::LogicalNot

#include <math_ops.h>

Returns the truth value of NOT x element-wise.

Summary

Args:

  • scope: A Scope object
  • x: A Tensor of type bool.

Returns:

  • Output: A Tensor of type bool with the same shape as x. The logical negation of x.
