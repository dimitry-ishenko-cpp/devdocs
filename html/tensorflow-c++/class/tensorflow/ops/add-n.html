tensorflow::ops::AddN

#include <math_ops.h>

Add all input tensors element wise.

Summary

Inputs must be of same size and shape.

x = [9, 7, 10]
tf.math.add_n(x) ==> 26

Args:

  • scope: A Scope object

Returns:

  • Output: The sum tensor.
