tensorflow::ops::IsFinite

#include <math_ops.h>

Returns which elements of x are finite.

Summary

(numpy) Equivalent to np.isfinite

Example:

x = tf.constant([5.0, 4.8, 6.8, np.inf, np.nan])
tf.math.is_finite(x) ==> [True, True, True, False, False]

Args:

  • scope: A Scope object

Returns:

  • Output: The y tensor.
