tensorflow::ops::ZerosLike

#include <array_ops.h>

Returns a tensor of zeros with the same shape and type as x.

Summary

Args:

  • scope: A Scope object
  • x: a tensor of type T.

Returns:

  • Output: a tensor of the same shape and type as x but filled with zeros.
