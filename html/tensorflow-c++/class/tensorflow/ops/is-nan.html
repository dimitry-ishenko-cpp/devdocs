tensorflow::ops::IsNan

#include <math_ops.h>

Returns which elements of x are NaN.

Summary

(numpy) Equivalent to np.isnan

Example:

x = tf.constant([5.0, np.nan, 6.8, np.nan, np.inf])
tf.math.is_nan(x) ==> [False, True, False, True, False]

Args:

  • scope: A Scope object

Returns:

  • Output: The y tensor.
