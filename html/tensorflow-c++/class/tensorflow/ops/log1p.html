tensorflow::ops::Log1p

#include <math_ops.h>

Computes natural logarithm of (1 + x) element-wise.

Summary

I.e., \(y = \log_e (1 + x)\).

Example:

x = tf.constant([0, 0.5, 1, 5])
tf.math.log1p(x) ==> [0., 0.4054651, 0.6931472, 1.7917595]

Args:

  • scope: A Scope object

Returns:

  • Output: The y tensor.
