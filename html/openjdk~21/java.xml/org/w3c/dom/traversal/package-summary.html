Package org.w3c.dom.traversal

package org.w3c.dom.traversal

Provides interfaces for DOM Level 2 Traversal. Refer to the Document Object Model (DOM) Level 2 Traversal and Range Specification, the Traversal module contains specialized interfaces dedicated to traversing the document structure.

Since:
1.5
Package Description
org.w3c.dom
Provides the interfaces for the Document Object Model (DOM).
Class Description
DocumentTraversal
DocumentTraversal contains methods that create NodeIterators and TreeWalkers to traverse a node and its children in document order (depth first, pre-order traversal, which is equivalent to the order in which the start tags occur in the text representation of the document).

Filters are objects that know how to "filter out" nodes.
