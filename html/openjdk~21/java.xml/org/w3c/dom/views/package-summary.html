Package org.w3c.dom.views

package org.w3c.dom.views
Provides interfaces for DOM Level 2 Views. Refer to the Document Object Model (DOM) Level 2 Views Specification, the Views module allows programs and scripts to dynamically access and update the content of a representation of a document.
Since:
1.8
Package Description
org.w3c.dom
Provides the interfaces for the Document Object Model (DOM).
Class Description
AbstractView
A base interface that all views shall derive from.
DocumentView
The DocumentView interface is implemented by Document objects in DOM implementations supporting DOM Views.
