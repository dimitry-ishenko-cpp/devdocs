Package javax.xml.crypto.dom

package javax.xml.crypto.dom
DOM-specific classes for the javax.xml.crypto package. Only users who are using DOM-based XML cryptographic implementations (ex: XMLSignatureFactory or KeyInfoFactory) should need to make direct use of this package.
