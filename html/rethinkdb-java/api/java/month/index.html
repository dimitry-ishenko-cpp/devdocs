ReQL command: month

Command syntax

time.month() â†’ number

Description

Return the month of a time object as a number between 1 and 12. For your convenience, the terms r.january, r.february, etc. are defined and map to the appropriate integer.

Example: Retrieve all the users who were born in November.

r.table("users").filter(row -> row.g("birthdate").month().eq(11)).run(conn);

Example: Retrieve all the users who were born in September.

r.table("users").filter(
    row -> row.g("birthdate").month().eq(r.september())
).run(conn);
