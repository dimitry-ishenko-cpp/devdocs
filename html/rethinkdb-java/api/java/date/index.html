ReQL command: date

Command syntax

time.date() → time

Description

Return a new OffsetDateTime object only based on the day, month and year (ie. the same day at 00:00).

Example: Retrieve all the users whose birthday is today.

r.table("users").filter(
    user -> user.g("birthdate").date().eq(r.now().date())
).run(conn);

Note that the now command always returns UTC time, so the comparison may fail if user.g("birthdate") isn’t also in UTC. You can use the inTimezone command to adjust for this:

r.table("users").filter(
    user -> user.g("birthdate").date().eq(r.now().inTimezone("-08:00").date())
).run(conn);
