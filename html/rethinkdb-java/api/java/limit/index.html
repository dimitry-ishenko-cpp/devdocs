ReQL command: limit

Command syntax

sequence.limit(n) → stream
array.limit(n) → array

Description

End the sequence after the given number of elements.

Example: Only so many can fit in our Pantheon of heroes.

r.table("marvel").orderBy("belovedness").limit(10).run(conn);
