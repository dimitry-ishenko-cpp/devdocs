WP_Styles::add_inline_style( string $handle, string $code ): bool

Adds extra CSS styles to a registered stylesheet.

Parameters

$handle string Required
The style's registered handle.
$code string Required
String containing the CSS styles to be added.

Return

bool True on success, false on failure.

More Information

This method adds style rules ($code) to the given stylesheet to be output as embedded style (i.e. in a <style> element).

public function add_inline_style( $handle, $code ) {
	if ( ! $code ) {
		return false;
	}

	$after = $this->get_data( $handle, 'after' );
	if ( ! $after ) {
		$after = array();
	}

	$after[] = $code;

	return $this->add_data( $handle, 'after', $after );
}
