WP_Theme_JSON::get_settings(): array

Returns the existing settings for each block.

Description

Example:

{
  'root': {
    'color': {
      'custom': true
    }
  },
  'core/paragraph': {
    'spacing': {
      'customPadding': true
    }
  }
}

Return

array Settings per block.
