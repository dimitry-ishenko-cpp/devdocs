WP_Theme_JSON::scope_selector( string $scope, string $selector ): string

Function that scopes a selector with another one. This works a bit like SCSS nesting except the &amp; operator isnâ€™t supported.

Description

$scope = '.a, .b .c';
$selector = '> .x, .y';
$merged = scope_selector( $scope, $selector );
// $merged is '.a > .x, .a .y, .b .c > .x, .b .c .y'

Parameters

$scopestringrequired
Selector to scope to.
$selectorstringrequired
Original selector.

Return

string Scoped selector.

	$block_rules = '';

		}
