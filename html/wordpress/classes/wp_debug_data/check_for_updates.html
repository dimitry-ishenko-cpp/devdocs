WP_Debug_Data::check_for_updates()

Calls all core functions to check for updates.

Source

File: wp-admin/includes/class-wp-debug-data.php. View all references 

public static function check_for_updates() {
	wp_version_check();
	wp_update_plugins();
	wp_update_themes();
}
