WP_Customize_Selective_Refresh::handle_error( int $errno, string $errstr, string $errfile = null, int $errline = null ): true

Handles PHP errors triggered during rendering the partials.

Description

These errors will be relayed back to the client in the Ajax response.

Parameters

$errno int Required
Error number.
$errstr string Required
Error string.
$errfile string Optional
Error file.

Default: null

$errline int Optional
Error line.

Default: null

Return

true Always true.

public function handle_error( $errno, $errstr, $errfile = null, $errline = null ) {
	$this->triggered_errors[] = array(
		'partial'      => $this->current_partial_id,
		'error_number' => $errno,
		'error_string' => $errstr,
		'error_file'   => $errfile,
		'error_line'   => $errline,
	);
	return true;
}
