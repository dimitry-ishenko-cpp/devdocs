WP_REST_Widgets_Controller::retrieve_widgets()

Looks for “lost” widgets once per request.

Description

protected function retrieve_widgets() {
	if ( ! $this->widgets_retrieved ) {
		retrieve_widgets();
		$this->widgets_retrieved = true;
	}
}

Uses

Uses Description
retrieve_widgets() wp-includes/widgets.php

Validates and remaps any “orphaned” widgets to wp_inactive_widgets sidebar, and saves the widget settings. This has to run at least on each theme change.
