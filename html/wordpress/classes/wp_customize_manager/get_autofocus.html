WP_Customize_Manager::get_autofocus(): string[]

Gets the autofocused constructs.

Return

string[] Mapping of ‘panel’, ‘section’, ‘control’ to the ID which should be autofocused.
  • control string
    ID for control to be autofocused.
  • section string
    ID for section to be autofocused.
  • panel string
    ID for panel to be autofocused.
