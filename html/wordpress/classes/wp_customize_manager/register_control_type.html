WP_Customize_Manager::register_control_type( string $control )

Registers a customize control type.

Description

Registered types are eligible to be rendered via JS and created dynamically.

Parameters

$control string Required
Name of a custom control which is a subclass of WP_Customize_Control.

Source

File: wp-includes/class-wp-customize-manager.php. View all references 

public function register_control_type( $control ) {
	$this->registered_control_types[] = $control;
}
