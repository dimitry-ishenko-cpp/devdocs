Theme_Upgrader::theme_info( string $theme = null ): WP_Theme|false

Get the WP_Theme object for a theme.

Parameters

$theme string Optional
The directory name of the theme. This is optional, and if not supplied, the directory name from the last result will be used.

Default: null

Return

WP_Theme|false The theme's info object, or false $theme is not supplied and the last result isn't set.

Source

File: wp-admin/includes/class-theme-upgrader.php. View all references 

public function theme_info( $theme = null ) {
	if ( empty( $theme ) ) {
		if ( ! empty( $this->result['destination_name'] ) ) {
			$theme = $this->result['destination_name'];
		} else {
			return false;
		}
	}

	$theme = wp_get_theme( $theme );
	$theme->cache_delete();

	return $theme;
}
