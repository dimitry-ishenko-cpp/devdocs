WP_Screen::set_parentage( string $parent_file )

Sets the parent information for the screen.

Description

This is called in admin-header.php after the menu parent for the screen has been determined.

Parameters

$parent_file string Required
The parent file of the screen. Typically the $parent_file global.

Source

File: wp-admin/includes/class-wp-screen.php. View all references 

public function set_parentage( $parent_file ) {
	$this->parent_file         = $parent_file;
	list( $this->parent_base ) = explode( '?', $parent_file );
	$this->parent_base         = str_replace( '.php', '', $this->parent_base );
}
