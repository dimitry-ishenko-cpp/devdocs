WP_User::translate_level_to_cap( int $level ): string

Converts numeric level to level capability name.

Description

Prepends ‘level_’ to level number.

Parameters

$levelintrequired
Level number, 1 to 10.

Return

string

public function translate_level_to_cap( $level ) {
	return 'level_' . $level;
