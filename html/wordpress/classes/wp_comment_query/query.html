WP_Comment_Query::query( string|array $query ): array|int

Sets up the WordPress query for retrieving comments.

Parameters

$query string|array Required
Array or URL query string of parameters.

Return

array|int List of comments, or number of comments when 'count' is passed as a query var.

Source

File: wp-includes/class-wp-comment-query.php. View all references 

public function query( $query ) {
	$this->query_vars = wp_parse_args( $query );
	return $this->get_comments();
}
