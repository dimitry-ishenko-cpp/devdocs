WP_Query::rewind_comments()

Rewinds the comments, resets the comment index and comment to first.

Source

public function rewind_comments() {
	$this->current_comment = -1;
	if ( $this->comment_count > 0 ) {
		$this->comment = $this->comments[0];
	}
}
