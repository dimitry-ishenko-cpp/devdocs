WP_REST_Autosaves_Controller::create_item_permissions_check( WP_REST_Request $request ): true|WP_Error

Checks if a given request has access to create an autosave revision.

Description

Autosave revisions inherit permissions from the parent post, check if the current user has permission to edit the post.

Parameters

$request WP_REST_Request Required
Full details about the request.

Return

true|WP_Error True if the request has access to create the item, WP_Error object otherwise.

Source

File: wp-includes/rest-api/endpoints/class-wp-rest-autosaves-controller.php. View all references 

public function create_item_permissions_check( $request ) {
	$id = $request->get_param( 'id' );

	if ( empty( $id ) ) {
		return new WP_Error(
			'rest_post_invalid_id',
			__( 'Invalid item ID.' ),
			array( 'status' => 404 )
		);
	}

	return $this->parent_controller->update_item_permissions_check( $request );
}
