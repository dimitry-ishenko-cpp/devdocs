WP_Customize_Setting::json(): array

Retrieves the data to export to the client via JSON.

Return

array Array of parameters passed to JavaScript.

Source

public function json() {
	return array(
		'value'     => $this->js_value(),
		'transport' => $this->transport,
		'dirty'     => $this->dirty,
		'type'      => $this->type,
	);
}
