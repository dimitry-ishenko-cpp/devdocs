WP_Rewrite::get_author_permastruct(): string|false

Retrieves the author permalink structure.

Description

The permalink structure is front property, author base, and finally ‘/%author%’. Will set the author_structure property and then return it without attempting to set the value again.

Return

string|false Author permalink structure on success, false on failure.

Source

File: wp-includes/class-wp-rewrite.php. View all references 

public function get_author_permastruct() {
	if ( isset( $this->author_structure ) ) {
		return $this->author_structure;
	}

	if ( empty( $this->permalink_structure ) ) {
		$this->author_structure = '';
		return false;
	}

	$this->author_structure = $this->front . $this->author_base . '/%author%';

	return $this->author_structure;
}
