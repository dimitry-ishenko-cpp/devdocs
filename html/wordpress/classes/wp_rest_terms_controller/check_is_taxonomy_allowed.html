WP_REST_Terms_Controller::check_is_taxonomy_allowed( string $taxonomy ): bool

Checks that the taxonomy is valid.

Parameters

$taxonomy string Required
Taxonomy to check.

Return

bool Whether the taxonomy is allowed for REST management.

Source

File: wp-includes/rest-api/endpoints/class-wp-rest-terms-controller.php. View all references 

protected function check_is_taxonomy_allowed( $taxonomy ) {
	$taxonomy_obj = get_taxonomy( $taxonomy );
	if ( $taxonomy_obj && ! empty( $taxonomy_obj->show_in_rest ) ) {
		return true;
	}
	return false;
}
