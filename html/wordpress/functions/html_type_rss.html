html_type_rss()

Displays the HTML type based on the blog setting.

Description

The two possible values are either ‘xhtml’ or ‘html’.

Source

function html_type_rss() {
	$type = get_bloginfo( 'html_type' );
	if ( str_contains( $type, 'xhtml' ) ) {
		$type = 'xhtml';
	} else {
		$type = 'html';
	}
	echo $type;
}
