get_user_id_from_string( string $string ): int

This function has been deprecated. Use get_user_by() instead.

Get a numeric user ID from either an email address or a login.

Description

A numeric string is considered to be an existing user ID and is simply returned as such.

See also

  • get_user_by()

Parameters

$string string Required
Either an email address or a login.

Return

int

function get_user_id_from_string( $string ) {
	_deprecated_function( __FUNCTION__, '3.6.0', 'get_user_by()' );

	if ( is_email( $string ) )
		$user = get_user_by( 'email', $string );
	elseif ( is_numeric( $string ) )
		return $string;
	else
		$user = get_user_by( 'login', $string );

	if ( $user )
		return $user->ID;
	return 0;
}
