wp_create_thumbnail( mixed $file, int $max_side, mixed $deprecated = '' ): string

This function has been deprecated. Use image_resize() instead.

This was once used to create a thumbnail from an Image given a maximum side size.

Description

See also

  • image_resize()

Parameters

$file mixed Required
Filename of the original image, Or attachment ID.
$max_side int Required
Maximum length of a single side for the thumbnail.
$deprecated mixed Optional
Never used.

Default: ''

Return

string Thumbnail path on success, Error string on failure.

Source

File: wp-admin/includes/deprecated.php. View all references 

function wp_create_thumbnail( $file, $max_side, $deprecated = '' ) {
	_deprecated_function( __FUNCTION__, '3.5.0', 'image_resize()' );
	return apply_filters( 'wp_create_thumbnail', image_resize( $file, $max_side, $max_side ) );
}
