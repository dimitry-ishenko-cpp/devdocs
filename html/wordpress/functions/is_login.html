is_login(): bool

Determines whether the current request is for the login screen.

Description

See also

  • wp_login_url()

Return

bool True if inside WordPress login screen, false otherwise.

Source

function is_login() {
	return false !== stripos( wp_login_url(), $_SERVER['SCRIPT_NAME'] );
}
