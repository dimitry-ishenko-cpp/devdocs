do_activate_header()

Adds an action hook specific to this page.

Description

Fires on ‘wp_head’.

Source

function do_activate_header() {
	/**
	 * Fires within the `<head>` section of the Site Activation page.
	 *
	 * Fires on the 'wp_head' action.
	 *
	 * @since 3.0.0
	 */
	do_action( 'activate_wp_head' );
}
