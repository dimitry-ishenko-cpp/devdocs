wp_ajax_date_format()

Ajax handler for date formatting.

Source

File: wp-admin/includes/ajax-actions.php. View all references 

function wp_ajax_date_format() {
	wp_die( date_i18n( sanitize_option( 'date_format', wp_unslash( $_POST['date'] ) ) ) );
}
