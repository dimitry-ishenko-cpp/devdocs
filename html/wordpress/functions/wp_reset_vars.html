wp_reset_vars( array $vars )

Resets global variables based on $_GET and $_POST.

Description

This function resets global variables based on the names passed in the $vars array to the value of $_POST[$var] or $_GET[$var] or ‚Äù if neither is defined.

Parameters

$vars array Required
An array of globals to reset.

function wp_reset_vars( $vars ) {
	foreach ( $vars as $var ) {
		if ( empty( $_POST[ $var ] ) ) {
			if ( empty( $_GET[ $var ] ) ) {
				$GLOBALS[ $var ] = '';
			} else {
				$GLOBALS[ $var ] = $_GET[ $var ];
			}
		} else {
			$GLOBALS[ $var ] = $_POST[ $var ];
		}
	}
}
