is_gd_image( resource|GdImage|false $image ): bool

Determines whether the value is an acceptable type for GD image functions.

Description

In PHP 8.0, the GD extension uses GdImage objects for its data structures.
This function checks if the passed value is either a resource of type gd or a GdImage object instance. Any other type will return false.

Parameters

$image resource|GdImage|false Required
A value to check the type for.

Return

bool True if $image is either a GD image resource or GdImage instance, false otherwise.

Source

function is_gd_image( $image ) {
	if ( is_resource( $image ) && 'gd' === get_resource_type( $image )
		|| is_object( $image ) && $image instanceof GdImage
	) {
		return true;
	}

	return false;
}
