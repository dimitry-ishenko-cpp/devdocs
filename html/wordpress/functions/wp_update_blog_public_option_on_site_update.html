wp_update_blog_public_option_on_site_update( int $site_id, string $is_public )

Updates the blog_public option for a given site ID.

Parameters

$site_idintrequired
Site ID.
$is_publicstringrequired
Whether the site is public. A numeric string, for compatibility reasons. Accepts '1' or '0'.

Source

function wp_update_blog_public_option_on_site_update( $site_id, $is_public ) {

	// Bail if the site's database tables do not exist (yet).
	if ( ! wp_is_site_initialized( $site_id ) ) {
		return;
	}

	update_blog_option( $site_id, 'blog_public', $is_public );
}
