wp_get_update_php_annotation(): string

Returns the default annotation for the web hosting altering the “Update PHP” page URL.

Description

This function is to be used after wp_get_update_php_url() to return a consistent annotation if the web host has altered the default "Update PHP" page URL.

Return

string Update PHP page annotation. An empty string if no custom URLs are provided.

Source

File: wp-includes/functions.php. View all references 

function wp_get_update_php_annotation() {
	$update_url  = wp_get_update_php_url();
	$default_url = wp_get_default_update_php_url();

	if ( $update_url === $default_url ) {
		return '';
	}

	$annotation = sprintf(
		/* translators: %s: Default Update PHP page URL. */
		__( 'This resource is provided by your web host, and is specific to your site. For more information, <a href="%s.html" target="_blank">see the official WordPress documentation</a>.' ),
		esc_url( $default_url )
	);

	return $annotation;
}
