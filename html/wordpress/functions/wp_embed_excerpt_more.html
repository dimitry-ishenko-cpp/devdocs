wp_embed_excerpt_more( string $more_string ): string

Filters the string in the ‘more’ link displayed after a trimmed excerpt.

Description

Replaces ‘[…]’ (appended to automatically generated excerpts) with an ellipsis and a "Continue reading" link in the embed template.

Parameters

$more_string string Required
Default 'more' string.

Return

string 'Continue reading' link prepended with an ellipsis.

Source

File: wp-includes/embed.php. View all references 

function wp_embed_excerpt_more( $more_string ) {
	if ( ! is_embed() ) {
		return $more_string;
	}

	$link = sprintf(
		'<a href="%1$s.html" class="wp-embed-more" target="_top">%2$s</a>',
		esc_url( get_permalink() ),
		/* translators: %s: Post title. */
		sprintf( __( 'Continue reading %s' ), '<span class="screen-reader-text">' . get_the_title() . '</span>' )
	);
	return ' &hellip; ' . $link;
}
