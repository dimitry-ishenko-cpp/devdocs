calendar_week_mod( int $num ): float

Gets number of days since the start of the week.

Parameters

$numintrequired
Number of day.

Return

float Days since the start of the week.

Source

function calendar_week_mod( $num ) {
	$base = 7;
	return ( $num - $base * floor( $num / $base ) );
}
