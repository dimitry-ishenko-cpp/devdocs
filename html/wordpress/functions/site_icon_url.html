site_icon_url( int $size = 512, string $url = '', int $blog_id )

Displays the Site Icon URL.

Parameters

$size int Optional
Size of the site icon. Default 512 (pixels).

Default: 512

$url string Optional
Fallback url if no site icon is found.

Default: ''

$blog_id int Optional
ID of the blog to get the site icon for. Default current blog.

Source

function site_icon_url( $size = 512, $url = '', $blog_id = 0 ) {
	echo esc_url( get_site_icon_url( $size, $url, $blog_id ) );
}
