has_custom_header(): bool

Checks whether a custom header is set or not.

Return

bool True if a custom header is set. False if not.

Source

function has_custom_header() {
	if ( has_header_image() || ( has_header_video() && is_header_video_active() ) ) {
		return true;
	}

	return false;
}
