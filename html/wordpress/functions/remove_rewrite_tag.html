remove_rewrite_tag( string $tag )

Removes an existing rewrite tag (like %postname%).

Parameters

$tag string Required
Name of the rewrite tag.

function remove_rewrite_tag( $tag ) {
	global $wp_rewrite;
	$wp_rewrite->remove_rewrite_tag( $tag );
