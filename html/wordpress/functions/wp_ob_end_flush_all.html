wp_ob_end_flush_all()

Flushes all output buffers for PHP 5.2.

Description

Make sure all output buffers are flushed before our singletons are destroyed.

Source

File: wp-includes/functions.php. View all references 

function wp_ob_end_flush_all() {
	$levels = ob_get_level();
	for ( $i = 0; $i < $levels; $i++ ) {
		ob_end_flush();
	}
}
