write_post(): int|void

Calls wp_write_post() and handles the errors.

Return

int|void Post ID on success, void on failure.

Source

function write_post() {
	$result = wp_write_post();
	if ( is_wp_error( $result ) ) {
		wp_die( $result->get_error_message() );
	} else {
		return $result;
	}
}
