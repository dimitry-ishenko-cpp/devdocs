do_action( ‘edit_comment’, int $comment_id, array $data )

Fires immediately after a comment is updated in the database.

Description

The hook also fires immediately before comment status transition hooks are fired.

Parameters

$comment_idint
The comment ID.
$dataarray
Comment data.
