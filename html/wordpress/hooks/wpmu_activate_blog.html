do_action( 'wpmu_activate_blog', int $blog_id, int $user_id, string $password, string $signup_title, array $meta )

Fires immediately after a site is activated.

Parameters

$blog_id int
Blog ID.
$user_id int
User ID.
$password string
User password.
$signup_title string
Site title.
$meta array
Signup meta data. By default, contains the requested privacy setting and lang_id.
