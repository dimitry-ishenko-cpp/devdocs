apply_filters( ‘wpmu_signup_user_notification’, string $user_login, string $user_email, string $key, array $meta )

Filters whether to bypass the email notification for new user sign-up.

Parameters

$user_loginstring
User login name.
$user_emailstring
User email address.
$keystring
Activation key created in wpmu_signup_user() .
$metaarray
Signup meta data. Default empty array.
