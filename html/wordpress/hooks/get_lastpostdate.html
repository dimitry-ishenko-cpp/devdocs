apply_filters( 'get_lastpostdate', string|false $lastpostdate, string $timezone, string $post_type )

Filters the most recent time that a post on the site was published.

Parameters

$lastpostdate string|false
The most recent time that a post was published, in 'Y-m-d H:i:s' format. False on failure.
$timezone string
Location to use for getting the post published date.
See get_lastpostdate() for accepted $timezone values.
More Arguments from get_lastpostdate( ... $timezone ) The timezone for the timestamp. Accepts 'server', 'blog', or 'gmt'.
'server' uses the server's internal timezone.
'blog' uses the post_date field, which proxies to the timezone set for the site.
'gmt' uses the post_date_gmt field.
Default 'server'.
$post_type string
The post type to check.
