apply_filters( 'oembed_ttl', int $time, string $url, array $attr, int $post_ID )

Filters the oEmbed TTL value (time to live).

Parameters

$time int
Time to live (in seconds).
$url string
The attempted embed URL.
$attr array
An array of shortcode attributes.
$post_ID int
Post ID.
