apply_filters( 'teeny_mce_buttons', array $mce_buttons, string $editor_id )

Filters the list of teenyMCE buttons (Text tab).

Parameters

$mce_buttons array
An array of teenyMCE buttons.
$editor_id string
Unique editor identifier, e.g. 'content'.
