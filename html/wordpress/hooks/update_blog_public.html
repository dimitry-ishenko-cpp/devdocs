do_action( ‘update_blog_public’, int $site_id, string $is_public )

Fires after the current blog’s ‘public’ setting is updated.

Parameters

$site_idint
Site ID.
$is_publicstring
Whether the site is public. A numeric string, for compatibility reasons. Accepts '1' or '0'.
