apply_filters( ‘list_terms_exclusions’, string $exclusions, array $args, string[] $taxonomies )

Filters the terms to exclude from the terms query.

Parameters

$exclusionsstring
NOT IN clause of the terms query.
$argsarray
An array of terms query arguments.
