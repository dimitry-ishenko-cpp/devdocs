apply_filters( 'search_feed_link', string $link, string $feed, string $type )

Filters the search feed link.

Parameters

$link string
Search feed link.
$feed string
Feed type. Possible values include 'rss2', 'atom'.
$type string
The search type. One of 'posts' or 'comments'.
