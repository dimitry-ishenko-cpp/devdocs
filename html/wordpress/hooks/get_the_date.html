apply_filters( 'get_the_date', string|int $the_date, string $format, WP_Post $post )

Filters the date a post was published.

Parameters

$the_date string|int
Formatted date string or Unix timestamp if $format is 'U' or 'G'.
$format string
PHP date format.
$post WP_Post
The post object.
