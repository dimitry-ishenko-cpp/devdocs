do_action( ‘heartbeat_tick’, array $response, string $screen_id )

Fires when Heartbeat ticks in logged-in environments.

Description

Allows the transport to be easily replaced with long-polling.

Parameters

$responsearray
The Heartbeat response.
$screen_idstring
The screen ID.
