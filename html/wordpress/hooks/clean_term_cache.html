do_action( ‘clean_term_cache’, array $ids, string $taxonomy, bool $clean_taxonomy )

Fires once after each taxonomy’s term cache has been cleaned.

Parameters

$idsarray
An array of term IDs.
$taxonomystring
Taxonomy slug.
$clean_taxonomybool
Whether or not to clean taxonomy-wide caches
