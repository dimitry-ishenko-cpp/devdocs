apply_filters( 'comment_status_links', string[] $status_links )

Filters the comment status links.

Parameters

$status_links string[]
An associative array of fully-formed comment status links. Includes 'All', 'Mine', 'Pending', 'Approved', 'Spam', and 'Trash'.
