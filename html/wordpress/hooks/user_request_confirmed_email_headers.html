apply_filters( 'user_request_confirmed_email_headers', string|array $headers, string $subject, string $content, int $request_id, array $email_data )

Filters the headers of the user request confirmation email.

Parameters

$headers string|array
The email headers.
$subject string
The email subject.
$content string
The email content.
$request_id int
The request ID.
$email_data array
Data relating to the account action email.
  • requestWP_User_Request
    User request object.
  • user_emailstring
    The email address confirming a request
  • descriptionstring
    Description of the action being performed so the user knows what the email is for.
  • manage_urlstring
    The link to click manage privacy requests of this type.
  • sitenamestring
    The site name sending the mail.
  • siteurlstring
    The site URL sending the mail.
  • admin_emailstring
    The administrator email receiving the mail.
