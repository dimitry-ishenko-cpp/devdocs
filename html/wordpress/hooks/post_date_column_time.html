apply_filters( 'post_date_column_time', string $t_time, WP_Post $post, string $column_name, string $mode )

Filters the published time of the post.

Parameters

$t_time string
The published time.
$post WP_Post
Post object.
$column_name string
The column name.
$mode string
The list display mode ('excerpt' or 'list').
