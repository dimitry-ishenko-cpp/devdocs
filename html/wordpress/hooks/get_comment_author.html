apply_filters( 'get_comment_author', string $author, string $comment_ID, WP_Comment $comment )

Filters the returned comment author name.

Parameters

$author string
The comment author's username.
$comment_ID string
The comment ID as a numeric string.
$comment WP_Comment
The comment object.
