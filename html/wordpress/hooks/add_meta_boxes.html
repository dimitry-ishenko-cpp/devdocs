do_action( 'add_meta_boxes', string $post_type, WP_Post $post )

Fires after all built-in meta boxes have been added.

Parameters

$post_type string
Post type.
$post WP_Post
Post object.

More Information

The hook allows meta box registration for any post type.

Passes two parameters: $post_type and $post.

Note: You can also use add_meta_boxes_{post_type} for best practice, so your hook will only run when editing a specific post type. This will only receive 1 parameter â€“ $post.
