apply_filters( 'term_link', string $termlink, WP_Term $term, string $taxonomy )

Filters the term link.

Parameters

$termlink string
Term link URL.
$term WP_Term
Term object.
$taxonomy string
Taxonomy slug.

More Information

This filter is applied to the link URL for a term prior to printing by the function get_term_link() .
