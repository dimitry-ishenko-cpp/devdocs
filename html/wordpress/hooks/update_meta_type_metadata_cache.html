apply_filters( "update_{$meta_type}_metadata_cache", mixed $check, int[] $object_ids )

Short-circuits updating the metadata cache of a specific type.

Description

The dynamic portion of the hook name, $meta_type, refers to the meta object type (post, comment, term, user, or any other type with an associated meta table).
Returning a non-null value will effectively short-circuit the function.

Possible hook names include:

  • update_post_metadata_cache
  • update_comment_metadata_cache
  • update_term_metadata_cache
  • update_user_metadata_cache

Parameters

$check mixed
Whether to allow updating the meta cache of the given type.
$object_ids int[]
Array of object IDs to update the meta cache for.
