apply_filters( 'get_comment_time', string|int $date, string $format, bool $gmt, bool $translate, WP_Comment $comment )

Filters the returned comment time.

Parameters

$date string|int
The comment time, formatted as a date string or Unix timestamp.
$format string
PHP date format.
$gmt bool
Whether the GMT date is in use.
$translate bool
Whether the time is translated.
$comment WP_Comment
The comment object.
