do_action( 'attachment_updated', int $post_ID, WP_Post $post_after, WP_Post $post_before )

Fires once an existing attachment has been updated.

Parameters

$post_ID int
Post ID.
$post_after WP_Post
Post object following the update.
$post_before WP_Post
Post object before the update.
