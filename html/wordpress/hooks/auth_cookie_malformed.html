do_action( ‘auth_cookie_malformed’, string $cookie, string $scheme )

Fires if an authentication cookie is malformed.

Parameters

$cookiestring
Malformed auth cookie.
$schemestring
Authentication scheme. Values include 'auth', 'secure_auth', or 'logged_in'.
