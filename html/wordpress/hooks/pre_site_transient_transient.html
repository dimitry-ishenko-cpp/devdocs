apply_filters( “pre_site_transient_{$transient}”, mixed $pre_site_transient, string $transient )

Filters the value of an existing site transient before it is retrieved.

Description

The dynamic portion of the hook name, $transient, refers to the transient name.

Returning a value other than boolean false will short-circuit retrieval and return that value instead.

Parameters

$pre_site_transientmixed
The default value to return if the site transient does not exist.
Any value other than false will short-circuit the retrieval of the transient, and return that value.
$transientstring
Transient name.

Source

$pre = apply_filters( "pre_site_transient_{$transient}", false, $transient );
