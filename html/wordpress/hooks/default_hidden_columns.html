apply_filters( 'default_hidden_columns', string[] $hidden, WP_Screen $screen )

Filters the default list of hidden columns.

Parameters

$hidden string[]
Array of IDs of columns hidden by default.
$screen WP_Screen
WP_Screen object of the current screen.
