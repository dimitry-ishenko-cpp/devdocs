apply_filters( ‘wp_mime_type_icon’, string $icon, string $mime, int $post_id )

Filters the mime type icon.

Parameters

$iconstring
Path to the mime type icon.
$mimestring
Mime type.
$post_idint
Attachment ID. Will equal 0 if the function passed the mime type.
