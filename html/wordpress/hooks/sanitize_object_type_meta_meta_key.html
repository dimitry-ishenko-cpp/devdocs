apply_filters( "sanitize_{$object_type}_meta_{$meta_key}", mixed $meta_value, string $meta_key, string $object_type )

Filters the sanitization of a specific meta key of a specific meta type.

Description

The dynamic portions of the hook name, $meta_type, and $meta_key, refer to the metadata object type (comment, post, term, or user) and the meta key value, respectively.

Parameters

$meta_value mixed
Metadata value to sanitize.
$meta_key string
Metadata key.
$object_type string
Type of object metadata is for. Accepts 'post', 'comment', 'term', 'user', or any other object type with an associated meta table.
