apply_filters( 'pre_update_option', mixed $value, string $option, mixed $old_value )

Filters an option before its value is (maybe) serialized and updated.

Parameters

$value mixed
The new, unserialized option value.
$option string
Name of the option.
$old_value mixed
The old option value.
