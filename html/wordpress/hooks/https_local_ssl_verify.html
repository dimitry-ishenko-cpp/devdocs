apply_filters( 'https_local_ssl_verify', bool $ssl_verify, string $url )

Filters whether SSL should be verified for local HTTP API requests.

Parameters

$ssl_verify bool
Whether to verify the SSL connection. Default true.
$url string
The request URL.
