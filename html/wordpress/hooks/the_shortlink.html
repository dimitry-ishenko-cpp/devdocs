apply_filters( ‘the_shortlink’, string $link, string $shortlink, string $text, string $title )

Filters the short link anchor tag for a post.

Parameters

$linkstring
Shortlink anchor tag.
$shortlinkstring
Shortlink URL.
$textstring
Shortlink’s text.
$titlestring
Shortlink’s title attribute.
