do_action( 'network_admin_menu', string $context )

Fires before the administration menu loads in the Network Admin.

Parameters

$context string
Empty context.

More Information

This action is used to add extra submenus and menu options to the network admin panelâ€™s menu structure. The network admin panel is only seen by a super admin in a multisite installation.

The function associated with this action runs after the basic network admin panel menu structure is in place. For example, the function might call add_submenu_page() to add a submenu page.
