apply_filters( “{$field_no_prefix}_edit_pre”, mixed $value, int $post_id )

Filters the value of a specific post field to edit.

Description

Only applied to post fields with a name which is prefixed with post_.

The dynamic portion of the hook name, $field_no_prefix, refers to the post field name minus the post_ prefix. Possible filter names include:

  • author_edit_pre
  • date_edit_pre
  • date_gmt_edit_pre
  • content_edit_pre
  • title_edit_pre
  • excerpt_edit_pre
  • status_edit_pre
  • password_edit_pre
  • name_edit_pre
  • modified_edit_pre
  • modified_gmt_edit_pre
  • content_filtered_edit_pre
  • parent_edit_pre
  • type_edit_pre
  • mime_type_edit_pre

Parameters

$valuemixed
Value of the post field.
$post_idint
Post ID.
