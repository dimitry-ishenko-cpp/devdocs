apply_filters( ‘pre_months_dropdown_query’, object[]|false $months, string $post_type )

Filters whether to short-circuit performing the months dropdown query.

Parameters

$monthsobject[]|false
'Months' drop-down results. Default false.
$post_typestring
The post type.

Source
