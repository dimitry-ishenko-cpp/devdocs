apply_filters( 'get_term', WP_Term $_term, string $taxonomy )

Filters a taxonomy term object.

Description

The ‘get_$taxonomy’ hook is also available for targeting a specific taxonomy.

Parameters

$_term WP_Term
Term object.
$taxonomy string
The taxonomy slug.
