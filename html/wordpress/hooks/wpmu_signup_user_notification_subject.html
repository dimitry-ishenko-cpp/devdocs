apply_filters( 'wpmu_signup_user_notification_subject', string $subject, string $user_login, string $user_email, string $key, array $meta )

Filters the subject of the notification email of new user signup.

Parameters

$subject string
Subject of the notification email.
$user_login string
User login name.
$user_email string
User email address.
$key string
Activation key created in wpmu_signup_user() .
$meta array
Signup meta data. Default empty array.

Source

File: wp-includes/ms-functions.php. View all references 

apply_filters(
	'wpmu_signup_user_notification_subject',
	/* translators: New user notification email subject. 1: Network title, 2: New user login. */
	_x( '[%1$s] Activate %2$s', 'New user notification email subject' ),
	$user_login,
	$user_email,
	$key,
	$meta
