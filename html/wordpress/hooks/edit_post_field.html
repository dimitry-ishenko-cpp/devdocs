apply_filters( “edit_post_{$field}”, mixed $value, int $post_id )

Filters the value of a specific post field to edit.

Description

Only applied to post fields not prefixed with post_.

The dynamic portion of the hook name, $field, refers to the post field name. Possible filter names include:

  • edit_post_ID
  • edit_post_ping_status
  • edit_post_pinged
  • edit_post_to_ping
  • edit_post_comment_count
  • edit_post_comment_status
  • edit_post_guid
  • edit_post_menu_order

Parameters

$valuemixed
Value of the post field.
$post_idint
Post ID.
