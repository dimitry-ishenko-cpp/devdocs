apply_filters( ‘wp_edit_nav_menu_walker’, string $class, int $menu_id )

Filters the Walker class used when adding nav menu items.

Parameters

$classstring
The walker class to use. Default ‘Walker_Nav_Menu_Edit‘.
$menu_idint
ID of the menu being rendered.
