apply_filters( ‘widget_pages_args’, array $args, array $instance )

Filters the arguments for the Pages widget.

Description

See also

  • wp_list_pages()

Parameters

$argsarray
An array of arguments to retrieve the pages list.
$instancearray
Array of settings for the current widget.

Source

apply_filters(
	'widget_pages_args',
	array(
		'title_li'    => '',
		'echo'        => 0,
		'sort_column' => $sortby,
		'exclude'     => $exclude,
	),
	$instance
)
