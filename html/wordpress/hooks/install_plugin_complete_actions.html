apply_filters( ‘install_plugin_complete_actions’, string[] $install_actions, object $api, string $plugin_file )

Filters the list of action links available following a single plugin installation.

Parameters

$install_actionsstring[]
Array of plugin action links.
$apiobject
Object containing WordPress.org API plugin data. Empty for non-API installs, such as when a plugin is installed via upload.
$plugin_filestring
Path to the plugin file relative to the plugins directory.
