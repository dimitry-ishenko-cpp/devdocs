apply_filters( ‘wp_handle_upload’, array $upload, string $context )

Filters the data array for the uploaded file.

Parameters

$uploadarray
Array of upload data.
  • file string
    Filename of the newly-uploaded file.
  • url string
    URL of the newly-uploaded file.
  • type string
    Mime type of the newly-uploaded file.
$contextstring
The type of upload action. Values include 'upload' or 'sideload'.

Source

return apply_filters(
	'wp_handle_upload',
	array(
		'file' => $new_file,
		'url'  => $url,
		'type' => $type,
	),
	'wp_handle_sideload' === $action ? 'sideload' : 'upload'
);
