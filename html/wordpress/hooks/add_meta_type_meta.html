do_action( “add_{$meta_type}_meta”, int $object_id, string $meta_key, mixed $_meta_value )

Fires immediately before meta of a specific type is added.

Description

The dynamic portion of the hook name, $meta_type, refers to the meta object type (post, comment, term, user, or any other type with an associated meta table).

Possible hook names include:

  • add_post_meta
  • add_comment_meta
  • add_term_meta
  • add_user_meta

Parameters

$object_idint
ID of the object metadata is for.
$meta_keystring
Metadata key.
$_meta_valuemixed
Metadata value.
