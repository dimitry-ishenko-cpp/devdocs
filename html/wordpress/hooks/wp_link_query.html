apply_filters( 'wp_link_query', array $results, array $query )

Filters the link query results.

Description

Allows modification of the returned link query results.

See also

  • ‘wp_link_query_args’: filter

Parameters

$results array
An array of associative arrays of query results.
  • ...$0array
    • IDint
      Post ID.
    • titlestring
      The trimmed, escaped post title.
    • permalinkstring
      Post permalink.
    • infostring
      A 'Y/m/d'-formatted date for 'post' post type, the 'singular_name' post type label otherwise.
    $query array
    An array of WP_Query arguments.
