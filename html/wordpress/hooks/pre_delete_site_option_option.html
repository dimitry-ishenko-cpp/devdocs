do_action( “pre_delete_site_option_{$option}”, string $option, int $network_id )

Fires immediately before a specific network option is deleted.

Description

The dynamic portion of the hook name, $option, refers to the option name.

Parameters

$optionstring
Option name.
$network_idint
ID of the network.
