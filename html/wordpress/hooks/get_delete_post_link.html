apply_filters( 'get_delete_post_link', string $link, int $post_id, bool $force_delete )

Filters the post delete link.

Parameters

$link string
The delete link.
$post_id int
Post ID.
$force_delete bool
Whether to bypass the Trash and force deletion. Default false.
