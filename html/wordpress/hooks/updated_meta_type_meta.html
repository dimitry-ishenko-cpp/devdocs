do_action( "updated_{$meta_type}_meta", int $meta_id, int $object_id, string $meta_key, mixed $_meta_value )

Fires immediately after updating metadata of a specific type.

Description

The dynamic portion of the hook name, $meta_type, refers to the meta object type (post, comment, term, user, or any other type with an associated meta table).

Possible hook names include:

  • updated_post_meta
  • updated_comment_meta
  • updated_term_meta
  • updated_user_meta

Parameters

$meta_id int
ID of updated metadata entry.
$object_id int
ID of the object metadata is for.
$meta_key string
Metadata key.
$_meta_value mixed
Metadata value.
