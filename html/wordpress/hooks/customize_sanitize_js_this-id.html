apply_filters( "customize_sanitize_js_{$this->id}", mixed $value, WP_Customize_Setting $setting )

Filters a Customize setting value for use in JavaScript.

Description

The dynamic portion of the hook name, $this->id, refers to the setting ID.

Parameters

$value mixed
The setting value.
