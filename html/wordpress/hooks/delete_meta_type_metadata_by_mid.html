apply_filters( “delete_{$meta_type}_metadata_by_mid”, null|bool $delete, int $meta_id )

Short-circuits deleting metadata of a specific type by meta ID.

Description

The dynamic portion of the hook name, $meta_type, refers to the meta object type (post, comment, term, user, or any other type with an associated meta table).
Returning a non-null value will effectively short-circuit the function.

Possible hook names include:

  • delete_post_metadata_by_mid
  • delete_comment_metadata_by_mid
  • delete_term_metadata_by_mid
  • delete_user_metadata_by_mid

Parameters

$deletenull|bool
Whether to allow metadata deletion of the given type.
$meta_idint
Meta ID.
