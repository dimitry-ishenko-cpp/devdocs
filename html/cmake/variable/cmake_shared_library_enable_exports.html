CMAKE_SHARED_LIBRARY_ENABLE_EXPORTS

Added in version 3.27.

Specify whether shared library generates an import file.

This variable is used to initialize the ENABLE_EXPORTS target property for shared library targets when they are created by calls to the add_library() command. See the property documentation for details.
