CMAKE_BUILD_WITH_INSTALL_NAME_DIR

New in version 3.9.

Whether to use INSTALL_NAME_DIR on targets in the build tree.

This variable is used to initialize the BUILD_WITH_INSTALL_NAME_DIR property on all targets.
