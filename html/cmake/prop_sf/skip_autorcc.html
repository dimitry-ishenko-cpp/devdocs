SKIP_AUTORCC

Added in version 3.8.

Exclude the source file from AUTORCC processing (for Qt projects).

For broader exclusion control see SKIP_AUTOGEN.

EXAMPLE

# ...
set_property(SOURCE file.qrc PROPERTY SKIP_AUTORCC ON)
# ...
