append

Concatenates two strings and returns the concatenated value.

Input

{{ "/my/fancy/url" | append: ".html" }}

Output

/my/fancy/url.html

append can also be used with variables:

Input

{% assign filename = "/index.html" %}
{{ "website.com" | append: filename }}

Output

website.com/index.html
