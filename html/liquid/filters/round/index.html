round

Rounds a number to the nearest integer or, if a number is passed as an argument, to that number of decimal places.

Input

{{ 1.2 | round }}

Output

1

Input

{{ 2.7 | round }}

Output

3

Input

{{ 183.357 | round: 2 }}

Output

183.36
