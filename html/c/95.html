C95

ISO/IEC 9899/AMD1:1995, a.k.a. C95, is a previous revision of the C standard.

New language features

  • Digraphs
  • __STDC_VERSION__

New library features

New headers

  • <iso646.h>
  • <wctype.h>
  • <wchar.h>
