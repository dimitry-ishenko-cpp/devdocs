CSSMathInvert: CSSMathInvert() constructor

The CSSMathInvert() constructor creates a new CSSMathInvert object which represents a CSS calc() used as calc(1 / value)

Syntax

js

new CSSMathInvert(arg)

Parameters

arg

A CSSNumericValue.

Exceptions

RangeError

Raised if the arg is 0 or -0.
