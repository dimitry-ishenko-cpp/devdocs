ResizeObserver: disconnect() method

The disconnect() method of the ResizeObserver interface unobserves all observed Element or SVGElement targets.

Syntax

js

disconnect()

Parameters

None.

Return value

None (undefined).

Exceptions

None.

Examples

js

btn.addEventListener("click", () => {
  resizeObserver.disconnect();
});
