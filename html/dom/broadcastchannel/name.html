BroadcastChannel: name property

The read-only BroadcastChannel.name property returns a string, which uniquely identifies the given channel with its name. This name is passed to the BroadcastChannel() constructor at creation time and is therefore read-only.

Note: This feature is available in Web Workers

Values

A string.

Examples

js

// Connect to a channel
const bc = new BroadcastChannel("test_channel");

// More operations (like postMessage, â€¦)

// Log the channel name to the console
console.log(bc.name); // "test_channel"

// When done, disconnect from the channel
bc.close();
