SVGRect: height property

Baseline Widely available

This feature is well established and works across many devices and browser versions. It’s been available across browsers since January 2020.

  • Learn more
  • See full compatibility
  • Report feedback

The height property of the SVGRect interface is an alias for the DOMRect.height property. It describes the vertical size of the element. It reflects the SVG element's height attribute and the CSS height property.

The height is a length; it is the distance from the top of element to the bottom of the element in the the user coordinate system. Its syntax is the same as that for <length>.
