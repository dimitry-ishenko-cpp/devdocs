SVGAnimatedLength

The SVGAnimatedLength interface represents attributes of type <length> which can be animated.

Instance properties

baseVal Read only

A SVGLength representing the base value of the given attribute before applying any animations.

animVal Read only

If the given attribute or property is being animated, a SVGLength containing the current animated value of the attribute or property. If the given attribute or property is not currently being animated, a SVGLength containing the same value as baseVal.

Instance methods

This interface does not implement any specific methods.
