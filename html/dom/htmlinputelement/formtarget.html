HTMLInputElement: formTarget property

Baseline Widely available

This feature is well established and works across many devices and browser versions. It’s been available across browsers since July 2015.

  • Learn more
  • See full compatibility
  • Report feedback

The formTarget property of the HTMLInputElement interface is the tab, window, or iframe where the response of the submitted <form> is to be displayed. It reflects the value of the <input> element's formtarget attribute.

This property is valid only for submit and image <input> elements.

Its value overrides the target property of the HTMLFormElement interface if the form is submitted via the input. This property can be retrieved or set. If not set, the value is the empty string ("").

Value

A string.

Examples

inputElement.formTarget = "_blank";
