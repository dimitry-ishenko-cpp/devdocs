MediaKeySession: update() method

The MediaKeySession.update() method loads messages and licenses to the CDM, and then returns a Promise .

Syntax

js

update(response)

Parameters

response

An instance that is either an ArrayBuffer, a TypedArray, or a DataView.

Return value

A Promise.
