VTTCue: text property

Baseline Widely available

This feature is well established and works across many devices and browser versions. It’s been available across browsers since July 2015.

  • Learn more
  • See full compatibility
  • Report feedback

The text property of the VTTCue interface represents the text contents of the cue.

Value

A string containing the raw text of the cue.

Examples

In the following example a new VTTCue is created, then the value of size is set to the string "new cue value". The value is then printed to the console.

js
let video = document.querySelector("video");
let track = video.addTextTrack("captions", "Captions", "en");
track.mode = "showing";

let cue1 = new VTTCue(0, 0.9, "Hildy!");
cue1.text = "new cue value";
console.log(cue1.text); // 'new cue value';

track.addCue(cue1);
