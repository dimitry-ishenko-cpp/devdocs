WebGLRenderingContext: flush() method

The WebGLRenderingContext.flush() method of the WebGL API empties different buffer commands, causing all commands to be executed as quickly as possible.

Syntax

js
flush()

Parameters

None.

Return value

None (undefined).

Specifications

Specification
WebGL Specification
# 5.14.11
