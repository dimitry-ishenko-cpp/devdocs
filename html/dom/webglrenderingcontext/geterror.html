WebGLRenderingContext: getError() method

The WebGLRenderingContext.getError() method of the WebGL API returns error information.

Syntax

js
getError()

Parameters

None.

Return value

Examples

js
gl.getError(); // gl.NO_ERROR (0)

gl.enable(gl.FOOBAR);
gl.getError(); // gl.INVALID_ENUM;
