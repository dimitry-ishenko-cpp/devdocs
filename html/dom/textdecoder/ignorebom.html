TextDecoder: ignoreBOM property

The ignoreBOM read-only property of the TextDecoder interface is a Boolean indicating whether the byte order mark will be included in the output or skipped over.

Value

true if the byte order mark will be included in the decoded text; false if it will be skipped over when decoding and omitted.
