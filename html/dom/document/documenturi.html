Document: documentURI property

The documentURI read-only property of the Document interface returns the document location as a string.

Value

A string.

Examples

JavaScript

js

document.getElementById("url").textContent = document.documentURI;

HTML

html

<p id="urlText">
  URL:<br />
  <span id="url">URL goes here</span>
</p>
