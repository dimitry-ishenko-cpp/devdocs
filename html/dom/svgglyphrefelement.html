SVGGlyphRefElement

Deprecated: This feature is no longer recommended. Though some browsers might still support it, it may have already been removed from the relevant web standards, may be in the process of being dropped, or may only be kept for compatibility purposes. Avoid using it, and update existing code if possible; see the compatibility table at the bottom of this page to guide your decision. Be aware that this feature may cease to work at any time.

The SVGGlyphRefElement interface corresponds to the <glyphRef> elements.

Instance properties

This interface also inherits properties from its parent, SVGElement.

SVGGlyphRefElement.glyphRef Deprecated

A string corresponding to the glyphRef attribute of the given element.

SVGGlyphRefElement.format Deprecated

A string corresponding to the format attribute of the given element.

SVGGlyphRefElement.x Deprecated

A float corresponding to the x attribute of the given element.

SVGGlyphRefElement.y Deprecated

A float corresponding to the y attribute of the given element.

SVGGlyphRefElement.dx Deprecated

A float corresponding to the dx attribute of the given element.

SVGGlyphRefElement.dy Deprecated

A float corresponding to the dy attribute of the given element.

Instance methods

This interface has no methods but inherits methods from its parent, SVGElement.
