DeviceMotionEventRotationRate: alpha property

Secure context: This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The alpha read-only property of the DeviceMotionEventRotationRate interface indicates the rate of rotation around the Z axis, in degrees per second.

Value

A double indicating the rate of rotation around the Z axis, in degrees per second. See Accelerometer values explained for details.
