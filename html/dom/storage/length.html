Storage: length property

Baseline Widely available

This feature is well established and works across many devices and browser versions. It’s been available across browsers since July 2015.

  • Learn more
  • See full compatibility
  • Report feedback

The length read-only property of the Storage interface returns the number of data items stored in a given Storage object.

Value

The number of items stored in the Storage object.

Examples

The following function adds three data items to the local storage for the current domain, then returns the number of items in the storage:

function populateStorage() {
  localStorage.setItem("bgcolor", "yellow");
  localStorage.setItem("font", "Helvetica");
  localStorage.setItem("image", "cats.png");

  return localStorage.length; // Should return 3
}

Note: For a real-world example, see our Web Storage Demo.
