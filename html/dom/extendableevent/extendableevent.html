ExtendableEvent: ExtendableEvent() constructor

The ExtendableEvent() constructor creates a new ExtendableEvent object.

Syntax

js

new ExtendableEvent(type)
new ExtendableEvent(type, options)

Parameters

type

A string with the name of the event. It is case-sensitive.

options Optional

An object that, in addition of the properties defined in Event(), can contain any custom settings that you want to apply to the event object. Currently no possible options are mandatory, but this has been defined for forward compatibility across the different derived events.

Return value

A new ExtendableEvent object.
