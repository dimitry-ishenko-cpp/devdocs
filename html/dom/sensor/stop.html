Sensor: stop() method

Secure context: This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The stop() method of the Sensor interface deactivates the current sensor.

Syntax

js
stop()

Parameters

None.

Return value

None (undefined).
