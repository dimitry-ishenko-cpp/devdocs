HTMLLinkElement: sheet property

The read-only sheet property of the HTMLLinkElement interface contains the stylesheet associated with that element.

A stylesheet is associated to an HTMLLinkElement if rel="stylesheet" is used with <link>.

Value

A StyleSheet object, or null if none is associated with the element.

Examples

html

<html>
  <header>
    <link rel="stylesheet" href="styles.css" />
    â€¦
  </header>
</html>

The sheet property of the HTMLLinkElement object will return the StyleSheet object describing styles.css.
