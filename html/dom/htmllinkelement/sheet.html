HTMLLinkElement: sheet property

Baseline Widely available

This feature is well established and works across many devices and browser versions. It’s been available across browsers since September 2015.

  • Learn more
  • See full compatibility
  • Report feedback

The sheet read-only property of the HTMLLinkElement interface contains the stylesheet associated with that element.

A stylesheet is associated to an HTMLLinkElement if rel="stylesheet" is used with <link>.

Value

A StyleSheet object, or null if none is associated with the element.

Examples

<html>
  <header>
    <link rel="stylesheet" href="styles.css.html" />
    …
  </header>
</html>

The sheet property of the HTMLLinkElement object will return the StyleSheet object describing styles.css.
