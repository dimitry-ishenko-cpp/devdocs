TransformStream: writable property

The writable read-only property of the TransformStream interface returns the WritableStream instance controlled by this TransformStream.

Value

A WritableStream.

Examples

The following example creates a new TransformStream as a textEncoderStream, and prints the value of writable to the console.

js

const textEncoderStream = new TransformStream();
console.log(textEncoderStream.writable); // a WritableStream
