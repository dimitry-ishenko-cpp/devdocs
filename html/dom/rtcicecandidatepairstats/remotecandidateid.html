RTCIceCandidatePairStats: remoteCandidateId property

The RTCIceCandidatePairStats property remoteCandidateId is a string that uniquely identifies the remote ICE candidate which was analyzed to generate the RTCIceCandidateStats used to compute the statistics for this pair of candidates.

Value

A string uniquely identifies the remote ICE candidate—that is, the candidate describing a configuration for the remote peer—which is represented by the remote end of these statistics.
