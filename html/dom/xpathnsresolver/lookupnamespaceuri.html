XPathNSResolver: lookupNamespaceURI() method

The lookupNamespaceURI method looks up the namespace URI associated to the given namespace prefix within an XPath expression evaluated by the XPathEvaluator interface.

Syntax

js

lookupNamespaceURI(prefix)

Parameters

prefix

A string representing the prefix to look for.

Return value

A string representing the associated namespace URI or null if none is found.
