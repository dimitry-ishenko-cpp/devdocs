location: hostname property

The hostname property of the Location interface is a string containing the domain of the URL.

Value

A string.

Examples

js

const anchor = document.createElement("a");
anchor.href = "https://developer.mozilla.org:4097/en-US/docs/Location.hostname";

console.log(anchor.hostname); // developer.mozilla.org
