CSSMathMin: CSSMathMin() constructor

Experimental: This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

The CSSMathMin() constructor creates a new CSSMathMin object which represents the CSS min() function.

Syntax

js
new CSSMathMin(args)

Parameters

args

A list of values for the CSSMathProduct object to be either a double integer or a CSSNumericValue.

Exceptions

TypeError

Thrown if there is a failure when adding all of the values in args.
