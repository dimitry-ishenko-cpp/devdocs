MediaList: length property

The read-only length property of the MediaList interface returns the number of media queries in the list.

Value

A positive integer.

Examples

The following would log to the console the number of media queries stored in the MediaList associated with the first stylesheet applied to the current document.

js

const stylesheets = document.styleSheets;
const stylesheet = stylesheets[0];
console.log(stylesheet.media.length);
