FontFaceSet: entries() method

The entries() method of the FontFaceSet interface returns a new Iterator object, containing an array of [value,value] for each element in the FontFaceSet.

Syntax

js
entries()

Parameters

None.

Return value

A new iterator object that contains an array of [value, value] for each element in the CustomStateSet, in insertion order.
