FontFaceSet: clear() method

The clear() method of the FontFaceSet interface removes all fonts added via this interface. Fonts added with the @font-face rule are not removed.

Syntax

js

clear()

Parameters

None.

Return value

None (undefined).

Specifications

Specification
CSS Font Loading Module Level 3
