The transferIn() method of the USBDevice interface returns a promise that resolves with a USBInTransferResult when bulk or interrupt data is received from the USB device.

Syntax

js
transferIn(endpointNumber, length)

Parameters

endpointNumber

The number of a device-specific endpoint (buffer).

length

The maximum number of bytes that will be read back from the device. The actual data is in the USBInTransferResult in the resolved Promise.

Return value

A promise that resolves with a USBInTransferResult.
