HTMLTimeElement: dateTime property

The HTMLTimeElement.dateTime property is a string that reflects the datetime HTML attribute, containing a machine-readable form of the element's date and time value.

The format of the string must follow one of the following HTML microsyntaxes:

Value

A string.

Examples

js

// Assumes there is <time id="t"> element in the HTML

const t = document.getElementById("t");
t.dateTime = "6w 5h 34m 5s";
