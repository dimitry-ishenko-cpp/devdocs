FontFace: stretch property

The stretch property of the FontFace interface retrieves or sets how the font stretches.

This property is equivalent to the font-stretch descriptor.

Value

A string containing a descriptor as it would be defined in a style sheet's @font-face rule.

Specifications

Specification
