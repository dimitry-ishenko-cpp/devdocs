CSSGroupingRule: cssRules property

The cssRules property of the CSSGroupingRule interface returns a CSSRuleList containing a collection of CSSRule objects.

Value

a CSSRuleList.

Examples

js
let myRules = document.styleSheets[0].cssRules;
console.log(myRules);
