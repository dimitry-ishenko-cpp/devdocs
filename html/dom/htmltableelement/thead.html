HTMLTableElement: tHead property

The HTMLTableElement.tHead represents the <thead> element of a <table>. Its value will be null if there is no such element.

Value

A HTMLTableSectionElement.

Examples

js

if (table.tHead === my_head_el) {
  // …
}
