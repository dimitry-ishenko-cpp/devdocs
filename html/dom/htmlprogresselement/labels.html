HTMLProgressElement: labels property

Baseline Widely available

This feature is well established and works across many devices and browser versions. It’s been available across browsers since July 2015.

  • Learn more
  • See full compatibility
  • Report feedback

The HTMLProgressElement.labels read-only property returns a NodeList of the <label> elements associated with the <progress> element.

Value

A NodeList containing the <label> elements associated with the <progress> element.

Examples

HTML

<label id="label1" for="test">Label 1</label>
<progress id="test" value="70" max="100">70%</progress>
<label id="label2" for="test">Label 2</label>

JavaScript

window.addEventListener("DOMContentLoaded", () => {
  const progress = document.getElementById("test");
  for (const label of progress.labels) {
    console.log(label.textContent); // "Label 1" and "Label 2"
  }
});
