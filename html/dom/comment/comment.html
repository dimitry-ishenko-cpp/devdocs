Comment: Comment() constructor

The Comment() constructor returns a newly created Comment object with the optional string given in parameter as its textual content.

Syntax

js
new Comment()
new Comment(aString)

Parameters

  • aString Optional

Return value

A new Comment containing aString, or the empty string if no parameter was given.

Example

js
let comment = new Comment("Test");
