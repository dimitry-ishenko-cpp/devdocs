CSSTranslate

The CSSTranslate interface of the CSS_Object_Model#css_typed_object_model represents the translate() value of the individual transform property in CSS. It inherits properties and methods from its parent CSSTransformValue.

Constructor

CSSTranslate()

Creates a new CSSTranslate object.

Instance properties

x

Returns or sets the x-axis value.

y

Returns or sets the y-axis value.

z

Returns or sets the z-axis value.
