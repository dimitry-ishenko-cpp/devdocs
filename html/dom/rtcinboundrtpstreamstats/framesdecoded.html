RTCInboundRtpStreamStats: framesDecoded property

The framesDecoded property of the RTCInboundRtpStreamStats dictionary indicates the total number of frames which have been decoded successfully for this media source.

Value

An integer value indicating the total number of video frames which have been decoded for this stream so far. This represents the number of frames that would have been displayed assuming no frames were skipped.
