WebGL2RenderingContext: getUniformIndices() method

The WebGL2RenderingContext.getUniformIndices() method of the WebGL 2 API retrieves the indices of a number of uniforms within a WebGLProgram.

Syntax

js

getUniformIndices(program, uniformNames)

Parameters

program

A WebGLProgram containing uniforms whose indices to query.

uniformNames

An Array of string specifying the names of the uniforms to query.

Return value

An Array of GLuint containing the uniform indices.

Examples

js

const uniformIndices = gl.getUniformIndices(program, [
  "UBORed",
  "UBOGreen",
  "UBOBlue",
]);
