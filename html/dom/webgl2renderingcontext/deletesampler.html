WebGL2RenderingContext: deleteSampler() method

The WebGL2RenderingContext.deleteSampler() method of the WebGL 2 API deletes a given WebGLSampler object.

Syntax

js

deleteSampler(sampler)

Parameters

sampler

A WebGLSampler object to delete.

Return value

None (undefined).

Examples

gl must be a WebGL2RenderingContext. WebGLSampler objects are not available in WebGL 1.

js

const sampler = gl.createSampler();

// â€¦

gl.deleteSampler(sampler);
