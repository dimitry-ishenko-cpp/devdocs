WebGL2RenderingContext: deleteSync() method

The WebGL2RenderingContext.deleteSync() method of the WebGL 2 API deletes a given WebGLSync object.

Syntax

js
deleteSync(sync)

Parameters

sync

A WebGLSync object to delete.

Return value

None (undefined).

Examples

gl must be a WebGL2RenderingContext. WebGLSync objects are not available in WebGL 1.

js
const sync = gl.fenceSync(gl.SYNC_GPU_COMMANDS_COMPLETE, 0);

// â€¦

gl.deleteSync(sync);
