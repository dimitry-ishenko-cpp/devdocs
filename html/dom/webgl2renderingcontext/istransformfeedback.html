WebGL2RenderingContext: isTransformFeedback() method

The WebGL2RenderingContext.isTransformFeedback() method of the WebGL 2 API returns true if the passed object is a valid WebGLTransformFeedback object.

Syntax

js

isTransformFeedback(transformFeedback)

Parameters

transformFeedback

A WebGLTransformFeedback object to test.

Return value

A GLboolean indicating whether the given object is a valid WebGLTransformFeedback object (true) or not (false).

Examples

gl must be a WebGL2RenderingContext. WebGLTransformFeedback objects are not available in WebGL 1.

js

const transformFeedback = gl.createTransformFeedback();

// â€¦

gl.isTransformFeedback(transformFeedback);
