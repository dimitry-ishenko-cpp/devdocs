Headers: values() method

The Headers.values() method returns an iterator allowing to go through all values contained in this object. The values are String objects.

Note: This method is available in Web Workers.

Syntax

js

values()

Parameters

None.

Return value

Returns an iterator.

Examples

js

// Create a test Headers object
const myHeaders = new Headers();
myHeaders.append("Content-Type", "text/xml");
myHeaders.append("Vary", "Accept-Language");

// Display the values
for (const value of myHeaders.values()) {
  console.log(value);
}

The result is:

text/xml
Accept-Language
