CSSScale: CSSScale() constructor

The CSSScale() constructor creates a new CSSScale object representing the scale() and scale3d() values of the individual transform property in CSS.

Syntax

js

new CSSScale(x, y)
new CSSScale(x, y, z)

Parameters

x

A value for the x-axis of the CSSScale object to be constructed. This must either be a double integer or a CSSNumericValue.

y

A value for the y-axis of the CSSScale object to be constructed. This must either be a double integer or a CSSNumericValue.

z Optional

A value for the z-axis of the CSSScale object to be constructed. This must either be a double integer or a CSSNumericValue. If a value is passed for the z-axis this is a 3d transform. The value of is2D will be set to false.
