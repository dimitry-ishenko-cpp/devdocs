The containerName read-only property of the TaskAttributionTiming interface returns the container's name attribute. A container is the iframe, embed or object etc. that is being implicated, on the whole, for a long task.

Value

A string containing the container's name HTML content attribute (e.g. <iframe name="myIframe"> or <object name="myObject">).
