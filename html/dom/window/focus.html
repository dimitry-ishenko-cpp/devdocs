Window: focus() method

Makes a request to bring the window to the front. It may fail due to user settings and the window isn't guaranteed to be frontmost before this method returns.

Syntax

js

focus()

Parameters

None.

Return value

None (undefined).

Examples

js

if (clicked) {
  window.focus();
}
