void trace_setlogfilename(string name);

Set the file path for profile reports (-profile)

This function is a public API, exposed in core.runtime.

Since we are calling C functions under the hood, and we might need to open and close files during the runtime tear-down we copy the parameter via malloc to ensure NUL-termination.

Parameters:
string name Path to the output file. Empty means stdout.
