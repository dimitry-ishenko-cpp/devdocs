5.2.5 OPEN( ... NAME=) Â¶

Some Fortran compilers, including g77, let the user declare OPEN( ... NAME=). This is invalid standard Fortran syntax and is not supported by gfortran. OPEN( ... NAME=) should be replaced with OPEN( ... FILE=).
