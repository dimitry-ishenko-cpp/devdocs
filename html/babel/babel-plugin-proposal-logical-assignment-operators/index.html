Installation​

  • npm
  • Yarn
  • pnpm
npm install --save-dev @babel/plugin-proposal-logical-assignment-operators
yarn add --dev @babel/plugin-proposal-logical-assignment-operators
pnpm add --save-dev @babel/plugin-proposal-logical-assignment-operators

Usage​

With a configuration file (Recommended)​ 

{
  "plugins": ["@babel/plugin-proposal-logical-assignment-operators"]
}

Via CLI​ 

babel --plugins @babel/plugin-proposal-logical-assignment-operators script.js

Via Node API​ 

require("@babel/core").transformSync("code", {
  plugins: ["@babel/plugin-proposal-logical-assignment-operators"],
});
