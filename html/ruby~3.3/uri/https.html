class URI::HTTPS

Parent:
URI::HTTP

The default port for HTTPS URIs is 443, and the scheme is ‘https:’ rather than ‘http:’. Other than that, HTTPS URIs are identical to HTTP URIs; see URI::HTTP.

Constants

DEFAULT_PORT

A Default port of 443 for URI::HTTPS
