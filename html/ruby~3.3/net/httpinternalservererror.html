class Net::HTTPInternalServerError

Parent:
Net::HTTPServerError

Response class for Internal Server Error responses (status code 500).

An unexpected condition was encountered and no more specific message is suitable.

This class also includes (indirectly) module Net::HTTPHeader, which gives access to its methods for getting headers.
