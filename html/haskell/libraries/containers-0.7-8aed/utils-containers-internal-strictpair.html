Utils.Containers.Internal.StrictPair

Safe Haskell Safe
Language Haskell2010

Description

A strict pair

data StrictPair a b Source

The same as a regular Haskell pair, but

(x :*: _|_) = (_|_ :*: y) = _|_

Constructors

!a :*: !b infixr 1

toPair :: StrictPair a b -> (a, b) Source

Convert a strict pair to a standard pair.
