Description

Sums, lifted to functors.

Since: base-4.9.0.0

data Sum (f :: k -> Type) (g :: k -> Type) (a :: k) Source

Lifted sum of functors.

Examples
Expand
>>> fmap (+1) (InL (Just 1))  :: Sum Maybe [] Int
InL (Just 2)

>>> fmap (+1) (InR [1, 2, 3]) :: Sum Maybe [] Int
InR [2,3,4]
