type family OrdCond o lt eq gt where ... Source

A case statement on Ordering.

OrdCond c l e g is l when c ~ LT, e when c ~ EQ, and g when c ~ GT.

Since: base-4.16.0.0

OrdCond 'EQ lt eq gt = eq
