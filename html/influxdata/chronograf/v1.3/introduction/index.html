The getting started guide gets you up and running with Chronograf with as little configuration and code as possible. By the end of the guide, you will have downloaded, installed, and configured all four packages of the TICK stack (Telegraf, InfluxDB, Chronograf, and Kapacitor), and you will be all set to monitor your infrastructure.
