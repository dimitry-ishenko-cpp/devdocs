Reference TICKscripts

The Kapacitor repository has a number of example TICKscripts. These scripts use common Telegraf plugins as the data source and show how to build common alerts. Telegraf plugins with example scripts include “cpu”, “disk”, “mem”, and “netstat” metrics from the system plugin.
