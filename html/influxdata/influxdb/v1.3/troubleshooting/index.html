Frequently Asked Questions

This page addresses frequent sources of confusion and places where InfluxDB behaves in an unexpected way relative to other database systems. Where applicable, it links to outstanding issues on GitHub.

System Monitoring

System Monitoring means all statistical and diagnostic information made available to the user of the InfluxDB system, about the system itself. Its purpose is to assist with troubleshooting and performance analysis of the database itself.
