Property Details

$isValid public property

Whether to continue rendering the view file. Event handlers of yii\base\View::EVENT_BEFORE_RENDER may set this property to decide whether to continue rendering the current view file.

public boolean $isValid = true

$output public property

The rendering result of yii\base\View::renderFile(). Event handlers may modify this property and the modified output will be returned by yii\base\View::renderFile(). This property is only used by yii\base\View::EVENT_AFTER_RENDER event.

public string $output = null

$params public property

The parameter array passed to the yii\base\View::render() method.

public array $params = null
