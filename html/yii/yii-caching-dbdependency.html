Class yii\caching\DbDependency

Implements yii\base\Configurable
Available since version 2.0
Source Code https://github.com/yiisoft/yii2/blob/master/framework/caching/DbDependency.php

DbDependency represents a dependency based on the query result of a SQL statement.

If the query result changes, the dependency is considered as changed. The query is specified via the $sql property.

For more details and usage information on Cache, see the guide article on caching.
