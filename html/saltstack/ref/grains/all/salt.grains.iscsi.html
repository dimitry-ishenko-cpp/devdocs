salt.grains.iscsi

Grains for iSCSI Qualified Names (IQN).

New in version 2018.3.0.

To enable these grains set iscsi_grains: True in the minion config.

iscsi_grains: True

salt.grains.iscsi.iscsi_iqn()

Return iSCSI IQN
