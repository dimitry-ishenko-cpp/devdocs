salt.grains.metadata

Grains from cloud metadata servers at 169.254.169.254

New in version 2017.7.0.

depends

requests

To enable these grains that pull from the http://169.254.169.254/latest metadata server set metadata_server_grains: True in the minion config.

metadata_server_grains: True
