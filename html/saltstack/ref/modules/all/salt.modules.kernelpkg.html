salt.modules.kernelpkg

kernelpkg is a virtual module that is fulfilled by one of the following modules:

Execution Module

Used for

kernelpkg_linux_apt

Debian/Ubuntu-based distros which use apt-get for package management

kernelpkg_linux_yum

RedHat-based distros and derivatives using yum or dnf
