salt.modules.redis

Module to provide redis functionality to Salt

New in version 2014.7.0.

configuration

This module requires the redis python module and uses the following defaults which may be overridden in the minion configuration:

redis.host: 'salt'
redis.port: 6379
redis.db: 0
redis.password: None
