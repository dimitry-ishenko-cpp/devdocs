salt.pillar.venafi

Venafi Pillar Certificates

This module will only return pillar data if the venafi runner module has already been used to create certificates.

To configure this module, set venafi to True in the ext_pillar section of your master configuration file:

ext_pillar:
  - venafi: True

salt.pillar.venafi.ext_pillar(minion_id, pillar, conf)

Return an existing set of certificates
