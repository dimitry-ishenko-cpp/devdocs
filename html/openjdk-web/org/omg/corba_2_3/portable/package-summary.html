Package org.omg.CORBA_2_3.portable

Provides methods for the input and output of value types, and contains other updates to the org/omg/CORBA/portable package.

See: Description

Class Description
Delegate

Delegate class provides the ORB vendor specific implementation of CORBA object.
InputStream

InputStream provides for the reading of all of the mapped IDL types from the stream.
ObjectImpl

ObjectImpl class is the base class for all stubs.
OutputStream

OutputStream provides interface for writing of all of the mapped IDL type to the stream.
