module ActiveSupport::DescendantsTracker

Active Support Descendants Tracker

This module provides an internal implementation to track descendants which is faster than iterating through ObjectSpace.

However Ruby 3.1 provide a fast native +Class#subclasses+ method, so if you know your code wonâ€™t be executed on older rubies, including ActiveSupport::DescendantsTracker does not provide any benefit.

def descendants
  subclasses = DescendantsTracker.reject!(self.subclasses)
  subclasses.concat(subclasses.flat_map(&:descendants))
end
