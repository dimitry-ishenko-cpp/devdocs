Active Record Connection Pool Queue

Remove the head of the queue.

If timeout is not given, remove and return the head of the queue if the number of available elements is strictly greater than the number of threads currently waiting (that is, donâ€™t jump ahead in line). Otherwise, return nil.

If timeout is given, block if there is no element available, waiting up to timeout seconds for an element to become available.

  • ActiveRecord::ConnectionTimeoutError if timeout is given and no element
