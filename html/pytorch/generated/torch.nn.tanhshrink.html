Tanhshrink

class torch.nn.Tanhshrink(*args, **kwargs) [source]

Applies the element-wise function:

Tanhshrink(x)=x−tanh⁡(x)\text{Tanhshrink}(x) = x - \tanh(x)
Shape:
  • Input: (∗)(*), where ∗* means any number of dimensions.
  • Output: (∗)(*), same shape as the input.

Examples:

>>> m = nn.Tanhshrink()
>>> input = torch.randn(2)
>>> output = m(input)
