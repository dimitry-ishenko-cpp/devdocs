std::flat_set<Key,Compare,KeyContainer>::size

size_type size() const noexcept;
 (since C++23)

Returns the number of elements in the underlying container, that is, std::distance(begin(), end()).

Parameters

(none)

Return value

The number of elements in the container adaptor.

Complexity

Constant.

Example

The following code uses size to display the number of elements in a std::flat_set<int>:

#include <iostream>
#include <flat_set>
 
int main()
{
    std::flat_set<int> nums{1, 3, 5, 7};
 
    std::cout << "nums contains " << nums.size() << " elements.\n";
}

Output:

nums contains 4 elements.
