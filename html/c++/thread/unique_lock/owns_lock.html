std::unique_lock<Mutex>::owns_lock

bool owns_lock() const noexcept;
 (since C++11)

Checks whether *this owns a locked mutex or not.

Parameters

(none)

Return value

true if *this has an associated mutex and has acquired ownership of it, false otherwise.
