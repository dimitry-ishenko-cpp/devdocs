class Net::HTTPNetworkAuthenticationRequired

Parent:
Net::HTTPServerError

Response class for Network Authentication Required responses (status code 511).

The client needs to authenticate to gain network access.

This class also includes (indirectly) module Net::HTTPHeader, which gives access to its methods for getting headers.
