class Net::HTTPUnsupportedMediaType

Parent:
Net::HTTPClientError

Response class for Unsupported Media Type responses (status code 415).

The request entity has a media type which the server or resource does not support.

This class also includes (indirectly) module Net::HTTPHeader, which gives access to its methods for getting headers.
