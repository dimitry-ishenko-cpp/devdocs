class Net::HTTPVersionNotSupported

Parent:
Net::HTTPServerError

Response class for HTTP Version Not Supported responses (status code 505).

The server does not support the HTTP version used in the request.

This class also includes (indirectly) module Net::HTTPHeader, which gives access to its methods for getting headers.
