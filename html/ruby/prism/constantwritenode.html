class Prism::ConstantWriteNode

Parent:
Prism::Node

Represents writing to a constant.

Foo = 1
^^^^^^^

Attributes

name [R]

The name of the [constant](github.com/ruby/prism/blob/main/docs/parsing_rules.md#constants).

Foo = :bar # name `:Foo`

XYZ = 1    # name `:XYZ`
value [R]

The value to write to the constant. It can be any [non-void expression](github.com/ruby/prism/blob/main/docs/parsing_rules.md#non-void-expression).

FOO = :bar
      ^^^^

MyClass = Class.new
          ^^^^^^^^^

def ===(other)
  other.is_a?(ConstantWriteNode) &&
    (name === other.name) &&
    (name_loc.nil? == other.name_loc.nil?) &&
    (value === other.value) &&
    (operator_loc.nil? == other.operator_loc.nil?)
end

Implements case-equality for the node. This is effectively == but without comparing the value of locations. Locations are checked only for presence.

def name_loc

  return location if location.is_a?(Location)
  @name_loc = Location.new(source, location >> 32, location & 0xFFFFFFFF)
