std::mersenne_twister_engine<UIntType,w,n,m,r,a,u,d,s,b,t,c,l,f>::max

static constexpr result_type max();
 (since C++11)

Returns the maximum value potentially generated by the random-number engine. This value is equal to 2w - 1, where w is the template parameter also accessible as static member word_size.

Parameters

(none)

Return value

The maximum potentially generated value.

Complexity

Constant.
