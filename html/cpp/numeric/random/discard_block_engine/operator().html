std::discard_block_engine<Engine,P,R>::operator()

result_type operator()();
 (since C++11)

Generates a random value. The state of the underlying engine is advanced one or more times.

Parameters

(none)

Return value

A pseudo-random number in [min(), max()].

Exceptions

Throws nothing.
