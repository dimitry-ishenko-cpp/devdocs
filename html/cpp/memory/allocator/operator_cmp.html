Compares two default allocators. Since default allocators are stateless, two default allocators are always equal.

1) Returns true.
2) Returns false.

The != operator is synthesized from operator==.

 (since C++20)

Parameters

lhs, rhs - default allocators to compare

Return value

1) true
2) false
