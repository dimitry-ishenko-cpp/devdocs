Symfony\Component\Routing\Generator\Dumper

Classes

GeneratorDumper GeneratorDumper is the base class for all built-in generator dumpers.
PhpGeneratorDumper PhpGeneratorDumper creates a PHP class able to generate URLs for a given set of routes.

Interfaces

GeneratorDumperInterface GeneratorDumperInterface is the interface that all generator dumper classes must implement.
