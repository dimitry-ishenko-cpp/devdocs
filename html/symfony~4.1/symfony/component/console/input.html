Symfony\Component\Console\Input

Classes

ArgvInput ArgvInput represents an input coming from the CLI arguments.
ArrayInput ArrayInput represents an input provided as an array.
Input Input is the base class for all concrete Input classes.
InputArgument Represents a command line argument.
InputDefinition A InputDefinition represents a set of valid command line arguments and options.
InputOption Represents a command line option.
StringInput StringInput represents an input provided as a string.

Interfaces

InputAwareInterface InputAwareInterface should be implemented by classes that depends on the Console Input.
InputInterface InputInterface is the interface implemented by all input classes.
StreamableInputInterface StreamableInputInterface is the interface implemented by all input classes that have an input stream.
