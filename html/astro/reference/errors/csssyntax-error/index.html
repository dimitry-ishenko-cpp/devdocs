CSS Syntax Error.

Example error messages:
CSSSyntaxError: Missed semicolon
CSSSyntaxError: Unclosed string

What went wrong?

Astro encountered an error while parsing your CSS, due to a syntax error. This is often caused by a missing semicolon.
