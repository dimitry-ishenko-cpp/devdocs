[Java] Class ManagedIdentityConcurrentMap<K, V>

  • org.apache.groovy.util.concurrent.ManagedIdentityConcurrentMap 
public class ManagedIdentityConcurrentMap<K, V>
extends ConcurrentReferenceHashMap

This is a basic implementation of a map able to forget its keys could be weak/soft/strong references. This bases on ConcurrentReferenceHashMap, thus it is safe for concurrency. This map compares keys through references.

Type Parameters:
K - the key type
V - the value type
