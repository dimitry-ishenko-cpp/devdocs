ReQL command: timezone

Command syntax

time.timezone() → string

Description

Return the timezone of the time object.

Example: Return all the users in the “-07:00” timezone.

r.table("users").filter(lambda user:
    user["subscriptionDate"].timezone() == "-07:00"
)
