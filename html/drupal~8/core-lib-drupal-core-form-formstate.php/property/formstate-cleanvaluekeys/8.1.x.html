protected property FormState::$cleanValueKeys

An associative array of form value keys to be removed by cleanValues().

Any values that are temporary but must still be displayed as values in the rendered form should be added to this array using addCleanValueKey(). Initialized with internal Form API values.

This property is uncacheable.
