protected function CachedStorage::findByPrefix

protected CachedStorage::findByPrefix($prefix)

Finds configuration object names starting with a given prefix.

Given the following configuration objects:

  • node.type.article
  • node.type.page

Passing the prefix 'node.type.' will return an array containing the above names.

Parameters

string $prefix: The prefix to search for

Return value

array An array containing matching configuration object names.

protected function findByPrefix($prefix) {
  $cache_key = $this->getCacheKey($prefix);
  if (!isset($this->findByPrefixCache[$cache_key])) {
    $this->findByPrefixCache[$cache_key] = $this->storage->listAll($prefix);
  }
  return $this->findByPrefixCache[$cache_key];
}
