public function FormStateInterface::hasValue

public FormStateInterface::hasValue($key)

Determines if a specific key is present in the submitted form values.

Parameters

string|array $key: Values are stored as a multi-dimensional associative array. If $key is a string, it will return isset($values[$key]). If $key is an array, each element of the array will be used as a nested key. If $key = array('foo', 'bar') it will return isset($values['foo']['bar']).

Return value

bool TRUE if the $key is set, FALSE otherwise.
