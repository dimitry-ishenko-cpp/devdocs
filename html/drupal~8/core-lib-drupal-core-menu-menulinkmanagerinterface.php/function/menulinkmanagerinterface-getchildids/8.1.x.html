public function MenuLinkManagerInterface::getChildIds

public MenuLinkManagerInterface::getChildIds($id)

Loads all child link IDs of a given menu link, regardless of visibility.

This method is mostly useful for testing.

Parameters

string $id: The menu link plugin ID.

Return value

array An unordered array of IDs representing the IDs of all children, or NULL if the ID is invalid.
