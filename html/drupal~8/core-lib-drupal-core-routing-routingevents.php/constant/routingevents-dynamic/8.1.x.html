constant RoutingEvents::DYNAMIC

Name of the event fired during route collection to allow new routes.

This event is used to add new routes based upon existing routes, giving modules the opportunity to dynamically generate additional routes. The event listener method receives a \Drupal\Core\Routing\RouteBuildEvent instance.
