constant RoutingEvents::ALTER

Name of the event fired during route collection to allow changes to routes.

This event is used to process new routes before they get saved, giving modules the opportunity to alter routes provided by any other module. The event listener method receives a \Drupal\Core\Routing\RouteBuildEvent instance.
