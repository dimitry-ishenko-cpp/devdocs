pub enum Utf8Pattern<'a> {
    StringPattern(&'a [u8]),
    CharPattern(char),
}

Result of calling Pattern::as_utf8_pattern(). Can be used for inspecting the contents of a Pattern in cases where the underlying representation can be represented as UTF-8.
