Creates a new pointer that is dangling, but well-aligned.

This is useful for initializing types which lazily allocate, like Vec::new does.

Note that the pointer value may potentially represent a valid pointer to a T, which means this must not be used as a ‚Äúnot yet initialized‚Äù sentinel value. Types that lazily allocate must track initialization by some other means.
