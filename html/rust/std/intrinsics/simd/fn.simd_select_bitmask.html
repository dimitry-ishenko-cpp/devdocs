Function simd_select_bitmask

pub unsafe extern "rust-intrinsic" fn simd_select_bitmask<M, T>(
    m: M,
    yes: T,
    no: T,
) -> T
ðŸ”¬This is a nightly-only experimental API. (core_intrinsics)

Selects elements from a bitmask.

M must be an unsigned integer or array of u8, matching simd_bitmask.

T must be a vector.

For each element, if the bit in mask is 1, select the element from if_true. If the corresponding bit in mask is 0, select the element from if_false.

The bitmask bit order matches simd_bitmask.

Safety

Padding bits must be all zero.
