Function simd_masked_store

pub unsafe extern "rust-intrinsic" fn simd_masked_store<V, U, T>(
    mask: V,
    ptr: U,
    val: T,
)
ðŸ”¬This is a nightly-only experimental API. (core_intrinsics)

Writes to a vector of pointers.

T must be a vector.

U must be a pointer to the element type of T

V must be a vector of integers with the same length as T (but any element size).

For each element, if the corresponding value in mask is !0, write the corresponding value in val to the pointer offset from ptr. The first element is written to ptr, the second to ptr.wrapping_offset(1) and so on. Otherwise if the corresponding value in mask is 0, do nothing.

Safety

Unmasked values in T must be writeable as if by <ptr>::write (e.g. aligned to the element type).

mask must only contain 0 or !0 values.
