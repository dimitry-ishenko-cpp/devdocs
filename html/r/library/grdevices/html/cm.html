cm Unit Transformation

Description

Translates from inches to cm (centimeters).

Usage

cm(x)

Arguments

x

numeric vector

Examples

cm(1)  # = 2.54

## Translate *from* cm *to* inches:

10 / cm(1) # -> 10cm  are 3.937 inches
