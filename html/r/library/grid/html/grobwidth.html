grobWidth Create a Unit Describing the Width of a Grob

Description

These functions create a unit object describing the width or height of a grob. They are generic.

Usage

grobWidth(x)
grobHeight(x)
grobAscent(x)
grobDescent(x)

Arguments

x

A grob object.

Value

A unit object.
