getYear

Get the year of the given date.

Description

Get the year of the given date.

Usage

// CommonJS
var getYear = require('date-fns/getYear')
// ES 2015
import getYear from 'date-fns/getYear'
// ESM
import { getYear } from 'date-fns'

Syntax

getYear(date)

Arguments

Name Description
date

the given date

Returns

Description

the year

Exceptions

Type Description
TypeError

1 argument required

Examples

// Which year is 2 July 2014?
const result = getYear(new Date(2014, 6, 2))
//=> 2014
