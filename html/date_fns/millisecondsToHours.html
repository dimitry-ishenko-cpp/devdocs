millisecondsToHours

Convert milliseconds to hours.

Description

Convert a number of milliseconds to a full number of hours.

Usage

// CommonJS
var millisecondsToHours = require('date-fns/millisecondsToHours')
// ES 2015
import millisecondsToHours from 'date-fns/millisecondsToHours'
// ESM
import { millisecondsToHours } from 'date-fns'

Syntax

millisecondsToHours(milliseconds)

Arguments

Name Description
milliseconds

number of milliseconds to be converted

Returns

Description

the number of milliseconds converted in hours

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 7200000 milliseconds to hours:
const result = millisecondsToHours(7200000)
//=> 2
// It uses floor rounding:
const result = millisecondsToHours(7199999)
//=> 1
