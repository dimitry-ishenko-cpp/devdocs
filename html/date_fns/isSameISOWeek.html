isSameISOWeek

Are the given dates in the same ISO week (and year)?

Description

Are the given dates in the same ISO week (and year)?

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var isSameISOWeek = require('date-fns/isSameISOWeek')
// ES 2015
import isSameISOWeek from 'date-fns/isSameISOWeek'
// ESM
import { isSameISOWeek } from 'date-fns'

Syntax

isSameISOWeek(dateLeft, dateRight)

Arguments

Name Description
dateLeft

the first date to check
dateRight

the second date to check

Returns

Description

the dates are in the same ISO week (and year)

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Are 1 September 2014 and 7 September 2014 in the same ISO week?
const result = isSameISOWeek(new Date(2014, 8, 1), new Date(2014, 8, 7))
//=> true
// Are 1 September 2014 and 1 September 2015 in the same ISO week?
const result = isSameISOWeek(new Date(2014, 8, 1), new Date(2015, 8, 1))
//=> false
