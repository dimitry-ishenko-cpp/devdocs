isSameQuarter

Are the given dates in the same quarter (and year)?

Description

Are the given dates in the same quarter (and year)?

Usage

// CommonJS
var isSameQuarter = require('date-fns/isSameQuarter')
// ES 2015
import isSameQuarter from 'date-fns/isSameQuarter'
// ESM
import { isSameQuarter } from 'date-fns'

Syntax

isSameQuarter(dateLeft, dateRight)

Arguments

Name Description
dateLeft

the first date to check
dateRight

the second date to check

Returns

Description

the dates are in the same quarter (and year)

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Are 1 January 2014 and 8 March 2014 in the same quarter?
const result = isSameQuarter(new Date(2014, 0, 1), new Date(2014, 2, 8))
//=> true
// Are 1 January 2014 and 1 January 2015 in the same quarter?
const result = isSameQuarter(new Date(2014, 0, 1), new Date(2015, 0, 1))
//=> false
