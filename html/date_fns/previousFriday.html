previousFriday

When is the previous Friday?

Description

When is the previous Friday?

Usage

// CommonJS
var previousFriday = require('date-fns/previousFriday')
// ES 2015
import previousFriday from 'date-fns/previousFriday'
// ESM
import { previousFriday } from 'date-fns'

Syntax

previousFriday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the previous Friday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the previous Friday before Jun, 19, 2021?
const result = previousFriday(new Date(2021, 5, 19))
//=> Fri June 18 2021 00:00:00
