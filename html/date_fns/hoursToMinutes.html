hoursToMinutes

Convert hours to minutes.

Description

Convert a number of hours to a full number of minutes.

Usage

// CommonJS
var hoursToMinutes = require('date-fns/hoursToMinutes')
// ES 2015
import hoursToMinutes from 'date-fns/hoursToMinutes'
// ESM
import { hoursToMinutes } from 'date-fns'

Syntax

hoursToMinutes(hours)

Arguments

Name Description
hours

number of hours to be converted

Returns

Description

the number of hours converted in minutes

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 2 hours to minutes:
const result = hoursToMinutes(2)
//=> 120
