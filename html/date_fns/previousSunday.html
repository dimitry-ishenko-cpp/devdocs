previousSunday

When is the previous Sunday?

Description

When is the previous Sunday?

Usage

// CommonJS
var previousSunday = require('date-fns/previousSunday')
// ES 2015
import previousSunday from 'date-fns/previousSunday'
// ESM
import { previousSunday } from 'date-fns'

Syntax

previousSunday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the previous Sunday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the previous Sunday before Jun, 21, 2021?
const result = previousSunday(new Date(2021, 5, 21))
//=> Sun June 20 2021 00:00:00
