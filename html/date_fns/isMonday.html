isMonday

Is the given date Monday?

Description

Is the given date Monday?

Usage

// CommonJS
var isMonday = require('date-fns/isMonday')
// ES 2015
import isMonday from 'date-fns/isMonday'
// ESM
import { isMonday } from 'date-fns'

Syntax

isMonday(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is Monday

Exceptions

Type Description
TypeError

1 argument required

Examples

// Is 22 September 2014 Monday?
const result = isMonday(new Date(2014, 8, 22))
//=> true
