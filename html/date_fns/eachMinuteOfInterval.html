eachMinuteOfInterval

Return the array of minutes within the specified time interval.

Description

Returns the array of minutes within the specified time interval.

Usage

// CommonJS
var eachMinuteOfInterval = require('date-fns/eachMinuteOfInterval')
// ES 2015
import eachMinuteOfInterval from 'date-fns/eachMinuteOfInterval'
// ESM
import { eachMinuteOfInterval } from 'date-fns'

Syntax

eachMinuteOfInterval(interval, [options])

Arguments

Name Description
interval

the interval. See [Interval]{@link https://date-fns.org/docs/Interval}
options

an object with options.
options.step

the step to increment by. The step must be equal to or greater than 1

Returns

Description

the array with starts of minutes from the minute of the interval start to the minute of the interval end

Exceptions

Type Description
TypeError

1 argument required
RangeError

options.step must be a number equal to or greater than 1
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// Each minute between 14 October 2020, 13:00 and 14 October 2020, 13:03
const result = eachMinuteOfInterval({
  start: new Date(2014, 9, 14, 13),
  end: new Date(2014, 9, 14, 13, 3)
})
//=> [
//   Wed Oct 14 2014 13:00:00,
//   Wed Oct 14 2014 13:01:00,
//   Wed Oct 14 2014 13:02:00,
//   Wed Oct 14 2014 13:03:00
// ]
