subQuarters

Subtract the specified number of year quarters from the given date.

Description

Subtract the specified number of year quarters from the given date.

Usage

// CommonJS
var subQuarters = require('date-fns/subQuarters')
// ES 2015
import subQuarters from 'date-fns/subQuarters'
// ESM
import { subQuarters } from 'date-fns'

Syntax

subQuarters(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of quarters to be subtracted. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the quarters subtracted

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Subtract 3 quarters from 1 September 2014:
const result = subQuarters(new Date(2014, 8, 1), 3)
//=> Sun Dec 01 2013 00:00:00
