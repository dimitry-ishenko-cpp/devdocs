roundToNearestMinutes

Rounds the given date to the nearest minute

Description

Rounds the given date to the nearest minute (or number of minutes). Rounds up when the given date is exactly between the nearest round minutes.

Usage

// CommonJS
var roundToNearestMinutes = require('date-fns/roundToNearestMinutes')
// ES 2015
import roundToNearestMinutes from 'date-fns/roundToNearestMinutes'
// ESM
import { roundToNearestMinutes } from 'date-fns'

Syntax

roundToNearestMinutes(date, [options])

Arguments

Name Description
date

the date to round
options

an object with options.
options.nearestTo

nearest number of minutes to round to. E.g. 15 to round to quarter hours.
options.roundingMethod

a rounding method (ceil, floor, round or trunc)

Returns

Description

the new date rounded to the closest minute

Exceptions

Type Description
TypeError

1 argument required
RangeError

options.nearestTo must be between 1 and 30

Examples

// Round 10 July 2014 12:12:34 to nearest minute:
const result = roundToNearestMinutes(new Date(2014, 6, 10, 12, 12, 34))
//=> Thu Jul 10 2014 12:13:00
// Round 10 July 2014 12:07:30 to nearest quarter hour:
const result = roundToNearestMinutes(new Date(2014, 6, 10, 12, 12, 34), { nearestTo: 15 })
// rounds up because given date is exactly between 12:00:00 and 12:15:00
//=> Thu Jul 10 2014 12:15:00
