isThursday

Is the given date Thursday?

Description

Is the given date Thursday?

Usage

// CommonJS
var isThursday = require('date-fns/isThursday')
// ES 2015
import isThursday from 'date-fns/isThursday'
// ESM
import { isThursday } from 'date-fns'

Syntax

isThursday(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is Thursday

Exceptions

Type Description
TypeError

1 argument required

Examples

// Is 25 September 2014 Thursday?
const result = isThursday(new Date(2014, 8, 25))
//=> true
