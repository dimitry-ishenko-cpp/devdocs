isTuesday

Is the given date Tuesday?

Description

Is the given date Tuesday?

Usage

// CommonJS
var isTuesday = require('date-fns/isTuesday')
// ES 2015
import isTuesday from 'date-fns/isTuesday'
// ESM
import { isTuesday } from 'date-fns'

Syntax

isTuesday(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is Tuesday

Exceptions

Type Description
TypeError

1 argument required

Examples

// Is 23 September 2014 Tuesday?
const result = isTuesday(new Date(2014, 8, 23))
//=> true
