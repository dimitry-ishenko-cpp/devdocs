addISOWeekYears

Add the specified number of ISO week-numbering years to the given date.

Description

Add the specified number of ISO week-numbering years to the given date.

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var addISOWeekYears = require('date-fns/addISOWeekYears')
// ES 2015
import addISOWeekYears from 'date-fns/addISOWeekYears'
// ESM
import { addISOWeekYears } from 'date-fns'

Syntax

addISOWeekYears(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of ISO week-numbering years to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the ISO week-numbering years added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 5 ISO week-numbering years to 2 July 2010:
const result = addISOWeekYears(new Date(2010, 6, 2), 5)
//=> Fri Jun 26 2015 00:00:00
