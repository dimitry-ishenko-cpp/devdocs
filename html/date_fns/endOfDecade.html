endOfDecade

Return the end of a decade for the given date.

Description

Return the end of a decade for the given date.

Usage

// CommonJS
var endOfDecade = require('date-fns/endOfDecade')
// ES 2015
import endOfDecade from 'date-fns/endOfDecade'
// ESM
import { endOfDecade } from 'date-fns'

Syntax

endOfDecade(date, [options])

Arguments

Name Description
date

the original date
options

an object with options.
options.additionalDigits

passed to toDate. See [toDate]{@link https://date-fns.org/docs/toDate}

Returns

Description

the end of a decade

Exceptions

Type Description
TypeError

1 argument required
RangeError

options.additionalDigits must be 0, 1 or 2

Examples

// The end of a decade for 12 May 1984 00:00:00:
const result = endOfDecade(new Date(1984, 4, 12, 00, 00, 00))
//=> Dec 31 1989 23:59:59.999
