isWednesday

Is the given date Wednesday?

Description

Is the given date Wednesday?

Usage

// CommonJS
var isWednesday = require('date-fns/isWednesday')
// ES 2015
import isWednesday from 'date-fns/isWednesday'
// ESM
import { isWednesday } from 'date-fns'

Syntax

isWednesday(date)

Arguments

Name Description
date

the date to check

Returns

Description

the date is Wednesday

Exceptions

Type Description
TypeError

1 argument required

Examples

// Is 24 September 2014 Wednesday?
const result = isWednesday(new Date(2014, 8, 24))
//=> true
