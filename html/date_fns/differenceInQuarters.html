differenceInQuarters

Get the number of quarters between the given dates.

Description

Get the number of quarters between the given dates.

Usage

// CommonJS
var differenceInQuarters = require('date-fns/differenceInQuarters')
// ES 2015
import differenceInQuarters from 'date-fns/differenceInQuarters'
// ESM
import { differenceInQuarters } from 'date-fns'

Syntax

differenceInQuarters(dateLeft, dateRight, [options])

Arguments

Name Description
dateLeft

the later date
dateRight

the earlier date
options

an object with options.
options.roundingMethod

a rounding method (ceil, floor, round or trunc)

Returns

Description

the number of full quarters

Exceptions

Type Description
TypeError

2 arguments required

Examples

// How many full quarters are between 31 December 2013 and 2 July 2014?
const result = differenceInQuarters(new Date(2014, 6, 2), new Date(2013, 11, 31))
//=> 2
