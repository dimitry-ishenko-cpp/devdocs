lastDayOfDecade

Return the last day of a decade for the given date.

Description

Return the last day of a decade for the given date.

Usage

// CommonJS
var lastDayOfDecade = require('date-fns/lastDayOfDecade')
// ES 2015
import lastDayOfDecade from 'date-fns/lastDayOfDecade'
// ESM
import { lastDayOfDecade } from 'date-fns'

Syntax

lastDayOfDecade(date)

Arguments

Name Description
date

the original date

Returns

Description

the last day of a decade

Exceptions

Type Description
TypeError

1 argument required

Examples

// The last day of a decade for 21 December 2012 21:12:00:
const result = lastDayOfDecade(new Date(2012, 11, 21, 21, 12, 00))
//=> Wed Dec 31 2019 00:00:00
