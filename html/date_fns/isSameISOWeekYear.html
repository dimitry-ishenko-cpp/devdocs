isSameISOWeekYear

Are the given dates in the same ISO week-numbering year?

Description

Are the given dates in the same ISO week-numbering year?

ISO week-numbering year: http://en.wikipedia.org/wiki/ISO_week_date

Usage

// CommonJS
var isSameISOWeekYear = require('date-fns/isSameISOWeekYear')
// ES 2015
import isSameISOWeekYear from 'date-fns/isSameISOWeekYear'
// ESM
import { isSameISOWeekYear } from 'date-fns'

Syntax

isSameISOWeekYear(dateLeft, dateRight)

Arguments

Name Description
dateLeft

the first date to check
dateRight

the second date to check

Returns

Description

the dates are in the same ISO week-numbering year

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Are 29 December 2003 and 2 January 2005 in the same ISO week-numbering year?
const result = isSameISOWeekYear(new Date(2003, 11, 29), new Date(2005, 0, 2))
//=> true
