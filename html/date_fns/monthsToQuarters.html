monthsToQuarters

Convert number of months to quarters.

Description

Convert a number of months to a full number of quarters.

Usage

// CommonJS
var monthsToQuarters = require('date-fns/monthsToQuarters')
// ES 2015
import monthsToQuarters from 'date-fns/monthsToQuarters'
// ESM
import { monthsToQuarters } from 'date-fns'

Syntax

monthsToQuarters(months)

Arguments

Name Description
months

number of months to be converted.

Returns

Description

the number of months converted in quarters

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 6 months to quarters:
const result = monthsToQuarters(6)
//=> 2
// It uses floor rounding:
const result = monthsToQuarters(7)
//=> 2
