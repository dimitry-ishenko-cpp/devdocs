monthsToYears

Convert number of months to years.

Description

Convert a number of months to a full number of years.

Usage

// CommonJS
var monthsToYears = require('date-fns/monthsToYears')
// ES 2015
import monthsToYears from 'date-fns/monthsToYears'
// ESM
import { monthsToYears } from 'date-fns'

Syntax

monthsToYears(months)

Arguments

Name Description
months

number of months to be converted

Returns

Description

the number of months converted in years

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 36 months to years:
const result = monthsToYears(36)
//=> 3

// It uses floor rounding:
const result = monthsToYears(40)
//=> 3
