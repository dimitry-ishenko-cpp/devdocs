eachMonthOfInterval

Return the array of months within the specified time interval.

Description

Return the array of months within the specified time interval.

Usage

// CommonJS
var eachMonthOfInterval = require('date-fns/eachMonthOfInterval')
// ES 2015
import eachMonthOfInterval from 'date-fns/eachMonthOfInterval'
// ESM
import { eachMonthOfInterval } from 'date-fns'

Syntax

eachMonthOfInterval(interval)

Arguments

Name Description
interval

the interval. See [Interval]{@link https://date-fns.org/docs/Interval}

Returns

Description

the array with starts of months from the month of the interval start to the month of the interval end

Exceptions

Type Description
TypeError

1 argument required
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// Each month between 6 February 2014 and 10 August 2014:
const result = eachMonthOfInterval({
  start: new Date(2014, 1, 6),
  end: new Date(2014, 7, 10)
})
//=> [
//   Sat Feb 01 2014 00:00:00,
//   Sat Mar 01 2014 00:00:00,
//   Tue Apr 01 2014 00:00:00,
//   Thu May 01 2014 00:00:00,
//   Sun Jun 01 2014 00:00:00,
//   Tue Jul 01 2014 00:00:00,
//   Fri Aug 01 2014 00:00:00
// ]
