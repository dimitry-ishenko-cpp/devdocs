eachWeekOfInterval

Return the array of weeks within the specified time interval.

Description

Return the array of weeks within the specified time interval.

Usage

// CommonJS
var eachWeekOfInterval = require('date-fns/eachWeekOfInterval')
// ES 2015
import eachWeekOfInterval from 'date-fns/eachWeekOfInterval'
// ESM
import { eachWeekOfInterval } from 'date-fns'

Syntax

eachWeekOfInterval(interval, [options])

Arguments

Name Description
interval

the interval. See [Interval]{@link https://date-fns.org/docs/Interval}
options

an object with options.
options.locale

the locale object. See [Locale]{@link https://date-fns.org/docs/Locale}
options.weekStartsOn

the index of the first day of the week (0 - Sunday)

Returns

Description

the array with starts of weeks from the week of the interval start to the week of the interval end

Exceptions

Type Description
TypeError

1 argument required
RangeError

options.weekStartsOn must be 0, 1, ..., 6
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// Each week within interval 6 October 2014 - 23 November 2014:
const result = eachWeekOfInterval({
  start: new Date(2014, 9, 6),
  end: new Date(2014, 10, 23)
})
//=> [
//   Sun Oct 05 2014 00:00:00,
//   Sun Oct 12 2014 00:00:00,
//   Sun Oct 19 2014 00:00:00,
//   Sun Oct 26 2014 00:00:00,
//   Sun Nov 02 2014 00:00:00,
//   Sun Nov 09 2014 00:00:00,
//   Sun Nov 16 2014 00:00:00,
//   Sun Nov 23 2014 00:00:00
// ]
