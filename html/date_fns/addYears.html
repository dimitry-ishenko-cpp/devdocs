addYears

Add the specified number of years to the given date.

Description

Add the specified number of years to the given date.

Usage

// CommonJS
var addYears = require('date-fns/addYears')
// ES 2015
import addYears from 'date-fns/addYears'
// ESM
import { addYears } from 'date-fns'

Syntax

addYears(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of years to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the years added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 5 years to 1 September 2014:
const result = addYears(new Date(2014, 8, 1), 5)
//=> Sun Sep 01 2019 00:00:00
