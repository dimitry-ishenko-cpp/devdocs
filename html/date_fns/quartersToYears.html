quartersToYears

Convert number of quarters to years.

Description

Convert a number of quarters to a full number of years.

Usage

// CommonJS
var quartersToYears = require('date-fns/quartersToYears')
// ES 2015
import quartersToYears from 'date-fns/quartersToYears'
// ESM
import { quartersToYears } from 'date-fns'

Syntax

quartersToYears(quarters)

Arguments

Name Description
quarters

number of quarters to be converted

Returns

Description

the number of quarters converted in years

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 8 quarters to years
const result = quartersToYears(8)
//=> 2
// It uses floor rounding:
const result = quartersToYears(11)
//=> 2
