setDay

Set the day of the week to the given date.

Description

Set the day of the week to the given date.

Usage

// CommonJS
var setDay = require('date-fns/setDay')
// ES 2015
import setDay from 'date-fns/setDay'
// ESM
import { setDay } from 'date-fns'

Syntax

setDay(date, day, [options])

Arguments

Name Description
date

the date to be changed
day

the day of the week of the new date
options

an object with options.
options.locale

the locale object. See [Locale]{@link https://date-fns.org/docs/Locale}
options.weekStartsOn

the index of the first day of the week (0 - Sunday)

Returns

Description

the new date with the day of the week set

Exceptions

Type Description
TypeError

2 arguments required
RangeError

options.weekStartsOn must be between 0 and 6

Examples

// Set week day to Sunday, with the default weekStartsOn of Sunday:
const result = setDay(new Date(2014, 8, 1), 0)
//=> Sun Aug 31 2014 00:00:00
// Set week day to Sunday, with a weekStartsOn of Monday:
const result = setDay(new Date(2014, 8, 1), 0, { weekStartsOn: 1 })
//=> Sun Sep 07 2014 00:00:00
