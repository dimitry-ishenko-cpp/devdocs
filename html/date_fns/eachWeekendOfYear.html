eachWeekendOfYear

List all the Saturdays and Sundays in the year.

Description

Get all the Saturdays and Sundays in the year.

Usage

// CommonJS
var eachWeekendOfYear = require('date-fns/eachWeekendOfYear')
// ES 2015
import eachWeekendOfYear from 'date-fns/eachWeekendOfYear'
// ESM
import { eachWeekendOfYear } from 'date-fns'

Syntax

eachWeekendOfYear(date)

Arguments

Name Description
date

the given year

Returns

Description

an array containing all the Saturdays and Sundays

Exceptions

Type Description
TypeError

1 argument required
RangeError

The passed date is invalid

Examples

// Lists all Saturdays and Sundays in the year
const result = eachWeekendOfYear(new Date(2020, 1, 1))
//=> [
//   Sat Jan 03 2020 00:00:00,
//   Sun Jan 04 2020 00:00:00,
//   ...
//   Sun Dec 27 2020 00:00:00
// ]
]
