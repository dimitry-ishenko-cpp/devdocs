eachDayOfInterval

Return the array of dates within the specified time interval.

Description

Return the array of dates within the specified time interval.

Usage

// CommonJS
var eachDayOfInterval = require('date-fns/eachDayOfInterval')
// ES 2015
import eachDayOfInterval from 'date-fns/eachDayOfInterval'
// ESM
import { eachDayOfInterval } from 'date-fns'

Syntax

eachDayOfInterval(interval, [options])

Arguments

Name Description
interval

the interval. See [Interval]{@link https://date-fns.org/docs/Interval}
options

an object with options.
options.step

the step to increment by. The value should be more than 1.

Returns

Description

the array with starts of days from the day of the interval start to the day of the interval end

Exceptions

Type Description
TypeError

1 argument required
RangeError

options.step must be a number greater than 1
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// Each day between 6 October 2014 and 10 October 2014:
const result = eachDayOfInterval({
  start: new Date(2014, 9, 6),
  end: new Date(2014, 9, 10)
})
//=> [
//   Mon Oct 06 2014 00:00:00,
//   Tue Oct 07 2014 00:00:00,
//   Wed Oct 08 2014 00:00:00,
//   Thu Oct 09 2014 00:00:00,
//   Fri Oct 10 2014 00:00:00
// ]
