eachWeekendOfInterval

List all the Saturdays and Sundays in the given date interval.

Description

Get all the Saturdays and Sundays in the given date interval.

Usage

// CommonJS
var eachWeekendOfInterval = require('date-fns/eachWeekendOfInterval')
// ES 2015
import eachWeekendOfInterval from 'date-fns/eachWeekendOfInterval'
// ESM
import { eachWeekendOfInterval } from 'date-fns'

Syntax

eachWeekendOfInterval(interval)

Arguments

Name Description
interval

the given interval. See [Interval]{@link https://date-fns.org/docs/Interval}

Returns

Description

an array containing all the Saturdays and Sundays

Exceptions

Type Description
TypeError

1 argument required
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// Lists all Saturdays and Sundays in the given date interval
const result = eachWeekendOfInterval({
  start: new Date(2018, 8, 17),
  end: new Date(2018, 8, 30)
})
//=> [
//   Sat Sep 22 2018 00:00:00,
//   Sun Sep 23 2018 00:00:00,
//   Sat Sep 29 2018 00:00:00,
//   Sun Sep 30 2018 00:00:00
// ]
