getDecade

Get the decade of the given date.

Description

Get the decade of the given date.

Usage

// CommonJS
var getDecade = require('date-fns/getDecade')
// ES 2015
import getDecade from 'date-fns/getDecade'
// ESM
import { getDecade } from 'date-fns'

Syntax

getDecade(date)

Arguments

Name Description
date

the given date

Returns

Description

the year of decade

Exceptions

Type Description
TypeError

1 argument required

Examples

// Which decade belongs 27 November 1942?
const result = getDecade(new Date(1942, 10, 27))
//=> 1940
