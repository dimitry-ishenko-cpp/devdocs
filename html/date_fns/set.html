set

Set date values to a given date.

Description

Set date values to a given date.

Sets time values to date from object values. A value is not set if it is undefined or null or doesn't exist in values.

Note about bundle size: set does not internally use setX functions from date-fns but instead opts to use native Date#setX methods. If you use this function, you may not want to include the other setX functions that date-fns provides if you are concerned about the bundle size.

Usage

// CommonJS
var set = require('date-fns/set')
// ES 2015
import set from 'date-fns/set'
// ESM
import { set } from 'date-fns'

Syntax

set(date, values)

Arguments

Name Description
date

the date to be changed
values

an object with options
values.year

the number of years to be set
values.month

the number of months to be set
values.date

the number of days to be set
values.hours

the number of hours to be set
values.minutes

the number of minutes to be set
values.seconds

the number of seconds to be set
values.milliseconds

the number of milliseconds to be set

Returns

Description

the new date with options set

Exceptions

Type Description
TypeError

2 arguments required
RangeError

values must be an object

Examples

// Transform 1 September 2014 into 20 October 2015 in a single line:
const result = set(new Date(2014, 8, 20), { year: 2015, month: 9, date: 20 })
//=> Tue Oct 20 2015 00:00:00
// Set 12 PM to 1 September 2014 01:23:45 to 1 September 2014 12:00:00:
const result = set(new Date(2014, 8, 1, 1, 23, 45), { hours: 12 })
//=> Mon Sep 01 2014 12:23:45
