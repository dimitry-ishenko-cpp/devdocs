addSeconds

Add the specified number of seconds to the given date.

Description

Add the specified number of seconds to the given date.

Usage

// CommonJS
var addSeconds = require('date-fns/addSeconds')
// ES 2015
import addSeconds from 'date-fns/addSeconds'
// ESM
import { addSeconds } from 'date-fns'

Syntax

addSeconds(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of seconds to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the seconds added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 30 seconds to 10 July 2014 12:45:00:
const result = addSeconds(new Date(2014, 6, 10, 12, 45, 0), 30)
//=> Thu Jul 10 2014 12:45:30
