hoursToSeconds

Convert hours to seconds.

Description

Convert a number of hours to a full number of seconds.

Usage

// CommonJS
var hoursToSeconds = require('date-fns/hoursToSeconds')
// ES 2015
import hoursToSeconds from 'date-fns/hoursToSeconds'
// ESM
import { hoursToSeconds } from 'date-fns'

Syntax

hoursToSeconds(hours)

Arguments

Name Description
hours

number of hours to be converted

Returns

Description

the number of hours converted in seconds

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 2 hours to seconds:
const result = hoursToSeconds(2)
//=> 7200
