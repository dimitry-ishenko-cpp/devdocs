addWeeks

Add the specified number of weeks to the given date.

Description

Add the specified number of week to the given date.

Usage

// CommonJS
var addWeeks = require('date-fns/addWeeks')
// ES 2015
import addWeeks from 'date-fns/addWeeks'
// ESM
import { addWeeks } from 'date-fns'

Syntax

addWeeks(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of weeks to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the weeks added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 4 weeks to 1 September 2014:
const result = addWeeks(new Date(2014, 8, 1), 4)
//=> Mon Sep 29 2014 00:00:00
