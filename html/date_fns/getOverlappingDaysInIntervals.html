getOverlappingDaysInIntervals

Get the number of days that overlap in two time intervals

Description

Get the number of days that overlap in two time intervals

Usage

// CommonJS
var getOverlappingDaysInIntervals = require('date-fns/getOverlappingDaysInIntervals')
// ES 2015
import getOverlappingDaysInIntervals from 'date-fns/getOverlappingDaysInIntervals'
// ESM
import { getOverlappingDaysInIntervals } from 'date-fns'

Syntax

getOverlappingDaysInIntervals(intervalLeft, intervalRight)

Arguments

Name Description
intervalLeft

the first interval to compare. See [Interval]{@link docs/Interval}
intervalRight

the second interval to compare. See [Interval]{@link docs/Interval}

Returns

Description

the number of days that overlap in two time intervals

Exceptions

Type Description
TypeError

2 arguments required
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// For overlapping time intervals adds 1 for each started overlapping day:
getOverlappingDaysInIntervals(
  { start: new Date(2014, 0, 10), end: new Date(2014, 0, 20) },
  { start: new Date(2014, 0, 17), end: new Date(2014, 0, 21) }
)
//=> 3
// For non-overlapping time intervals returns 0:
getOverlappingDaysInIntervals(
  { start: new Date(2014, 0, 10), end: new Date(2014, 0, 20) },
  { start: new Date(2014, 0, 21), end: new Date(2014, 0, 22) }
)
//=> 0
