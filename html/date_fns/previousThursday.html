previousThursday

When is the previous Thursday?

Description

When is the previous Thursday?

Usage

// CommonJS
var previousThursday = require('date-fns/previousThursday')
// ES 2015
import previousThursday from 'date-fns/previousThursday'
// ESM
import { previousThursday } from 'date-fns'

Syntax

previousThursday(date)

Arguments

Name Description
date

the date to start counting from

Returns

Description

the previous Thursday

Exceptions

Type Description
TypeError

1 argument required

Examples

// When is the previous Thursday before Jun, 18, 2021?
const result = previousThursday(new Date(2021, 5, 18))
//=> Thu June 17 2021 00:00:00
