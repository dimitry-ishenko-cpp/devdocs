isDate

Is the given value a date?

Description

Returns true if the given value is an instance of Date. The function works for dates transferred across iframes.

Usage

// CommonJS
var isDate = require('date-fns/isDate')
// ES 2015
import isDate from 'date-fns/isDate'
// ESM
import { isDate } from 'date-fns'

Syntax

isDate(value)

Arguments

Name Description
value

the value to check

Returns

Description

true if the given value is a date

Exceptions

Type Description
TypeError

1 arguments required

Examples

// For a valid date:
const result = isDate(new Date())
//=> true
// For an invalid date:
const result = isDate(new Date(NaN))
//=> true
// For some value:
const result = isDate('2014-02-31')
//=> false
// For an object:
const result = isDate({})
//=> false
