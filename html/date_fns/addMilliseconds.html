addMilliseconds

Add the specified number of milliseconds to the given date.

Description

Add the specified number of milliseconds to the given date.

Usage

// CommonJS
var addMilliseconds = require('date-fns/addMilliseconds')
// ES 2015
import addMilliseconds from 'date-fns/addMilliseconds'
// ESM
import { addMilliseconds } from 'date-fns'

Syntax

addMilliseconds(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of milliseconds to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the milliseconds added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 750 milliseconds to 10 July 2014 12:45:30.000:
const result = addMilliseconds(new Date(2014, 6, 10, 12, 45, 30, 0), 750)
//=> Thu Jul 10 2014 12:45:30.750
