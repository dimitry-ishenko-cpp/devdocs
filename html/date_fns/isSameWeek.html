isSameWeek

Are the given dates in the same week (and month and year)?

Description

Are the given dates in the same week (and month and year)?

Usage

// CommonJS
var isSameWeek = require('date-fns/isSameWeek')
// ES 2015
import isSameWeek from 'date-fns/isSameWeek'
// ESM
import { isSameWeek } from 'date-fns'

Syntax

isSameWeek(dateLeft, dateRight, [options])

Arguments

Name Description
dateLeft

the first date to check
dateRight

the second date to check
options

an object with options.
options.locale

the locale object. See [Locale]{@link https://date-fns.org/docs/Locale}
options.weekStartsOn

the index of the first day of the week (0 - Sunday)

Returns

Description

the dates are in the same week (and month and year)

Exceptions

Type Description
TypeError

2 arguments required
RangeError

options.weekStartsOn must be between 0 and 6

Examples

// Are 31 August 2014 and 4 September 2014 in the same week?
const result = isSameWeek(new Date(2014, 7, 31), new Date(2014, 8, 4))
//=> true
// If week starts with Monday,
// are 31 August 2014 and 4 September 2014 in the same week?
const result = isSameWeek(new Date(2014, 7, 31), new Date(2014, 8, 4), {
  weekStartsOn: 1
})
//=> false
// Are 1 January 2014 and 1 January 2015 in the same week?
const result = isSameWeek(new Date(2014, 0, 1), new Date(2015, 0, 1))
//=> false
