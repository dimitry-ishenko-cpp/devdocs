subMinutes

Subtract the specified number of minutes from the given date.

Description

Subtract the specified number of minutes from the given date.

Usage

// CommonJS
var subMinutes = require('date-fns/subMinutes')
// ES 2015
import subMinutes from 'date-fns/subMinutes'
// ESM
import { subMinutes } from 'date-fns'

Syntax

subMinutes(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of minutes to be subtracted. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the minutes subtracted

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Subtract 30 minutes from 10 July 2014 12:00:00:
const result = subMinutes(new Date(2014, 6, 10, 12, 0), 30)
//=> Thu Jul 10 2014 11:30:00
