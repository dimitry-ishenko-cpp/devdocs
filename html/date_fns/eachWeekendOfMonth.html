eachWeekendOfMonth

List all the Saturdays and Sundays in the given month.

Description

Get all the Saturdays and Sundays in the given month.

Usage

// CommonJS
var eachWeekendOfMonth = require('date-fns/eachWeekendOfMonth')
// ES 2015
import eachWeekendOfMonth from 'date-fns/eachWeekendOfMonth'
// ESM
import { eachWeekendOfMonth } from 'date-fns'

Syntax

eachWeekendOfMonth(date)

Arguments

Name Description
date

the given month

Returns

Description

an array containing all the Saturdays and Sundays

Exceptions

Type Description
TypeError

1 argument required
RangeError

The passed date is invalid

Examples

// Lists all Saturdays and Sundays in the given month
const result = eachWeekendOfMonth(new Date(2022, 1, 1))
//=> [
//   Sat Feb 05 2022 00:00:00,
//   Sun Feb 06 2022 00:00:00,
//   Sat Feb 12 2022 00:00:00,
//   Sun Feb 13 2022 00:00:00,
//   Sat Feb 19 2022 00:00:00,
//   Sun Feb 20 2022 00:00:00,
//   Sat Feb 26 2022 00:00:00,
//   Sun Feb 27 2022 00:00:00
// ]
