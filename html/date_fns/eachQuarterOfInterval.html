eachQuarterOfInterval

Return the array of quarters within the specified time interval.

Description

Return the array of quarters within the specified time interval.

Usage

// CommonJS
var eachQuarterOfInterval = require('date-fns/eachQuarterOfInterval')
// ES 2015
import eachQuarterOfInterval from 'date-fns/eachQuarterOfInterval'
// ESM
import { eachQuarterOfInterval } from 'date-fns'

Syntax

eachQuarterOfInterval(interval)

Arguments

Name Description
interval

the interval. See [Interval]{@link https://date-fns.org/docs/Interval}

Returns

Description

the array with starts of quarters from the quarter of the interval start to the quarter of the interval end

Exceptions

Type Description
TypeError

1 argument required
RangeError

The start of an interval cannot be after its end
RangeError

Date in interval cannot be Invalid Date

Examples

// Each quarter within interval 6 February 2014 - 10 August 2014:
const result = eachQuarterOfInterval({
  start: new Date(2014, 1, 6),
  end: new Date(2014, 7, 10)
})
//=> [
//   Wed Jan 01 2014 00:00:00,
//   Tue Apr 01 2014 00:00:00,
//   Tue Jul 01 2014 00:00:00,
// ]
