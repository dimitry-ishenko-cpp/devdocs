addHours

Add the specified number of hours to the given date.

Description

Add the specified number of hours to the given date.

Usage

// CommonJS
var addHours = require('date-fns/addHours')
// ES 2015
import addHours from 'date-fns/addHours'
// ESM
import { addHours } from 'date-fns'

Syntax

addHours(date, amount)

Arguments

Name Description
date

the date to be changed
amount

the amount of hours to be added. Positive decimals will be rounded using Math.floor, decimals less than zero will be rounded using Math.ceil.

Returns

Description

the new date with the hours added

Exceptions

Type Description
TypeError

2 arguments required

Examples

// Add 2 hours to 10 July 2014 23:00:00:
const result = addHours(new Date(2014, 6, 10, 23, 0), 2)
//=> Fri Jul 11 2014 01:00:00
