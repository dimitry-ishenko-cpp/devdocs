clamp

Return a date bounded by the start and the end of the given interval

Description

Clamps a date to the lower bound with the start of the interval and the upper bound with the end of the interval.

  • When the date is less than the start of the interval, the start is returned.
  • When the date is greater than the end of the interval, the end is returned.
  • Otherwise the date is returned.

Usage

// CommonJS
var clamp = require('date-fns/clamp')
// ES 2015
import clamp from 'date-fns/clamp'
// ESM
import { clamp } from 'date-fns'

Syntax

clamp(date, interval)

Arguments

Name Description
date

the date to be bounded
interval

the interval to bound to

Returns

Description

the date bounded by the start and the end of the interval

Exceptions

Type Description
TypeError

2 arguments required

Examples

// What is Mar, 21, 2021 bounded to an interval starting at Mar, 22, 2021 and ending at Apr, 01, 2021
const result = clamp(new Date(2021, 2, 21), {
  start: new Date(2021, 2, 22),
  end: new Date(2021, 3, 1),
})
//=> Mon Mar 22 2021 00:00:00
