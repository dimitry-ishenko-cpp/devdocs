minutesToSeconds

Convert minutes to seconds.

Description

Convert a number of minutes to a full number of seconds.

Usage

// CommonJS
var minutesToSeconds = require('date-fns/minutesToSeconds')
// ES 2015
import minutesToSeconds from 'date-fns/minutesToSeconds'
// ESM
import { minutesToSeconds } from 'date-fns'

Syntax

minutesToSeconds(minutes)

Arguments

Name Description
minutes

number of minutes to be converted

Returns

Description

the number of minutes converted in seconds

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 2 minutes to seconds
const result = minutesToSeconds(2)
//=> 120
