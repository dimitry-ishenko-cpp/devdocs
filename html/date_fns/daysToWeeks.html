daysToWeeks

Convert days to weeks.

Description

Convert a number of days to a full number of weeks.

Usage

// CommonJS
var daysToWeeks = require('date-fns/daysToWeeks')
// ES 2015
import daysToWeeks from 'date-fns/daysToWeeks'
// ESM
import { daysToWeeks } from 'date-fns'

Syntax

daysToWeeks(days)

Arguments

Name Description
days

number of days to be converted

Returns

Description

the number of days converted in weeks

Exceptions

Type Description
TypeError

1 argument required

Examples

// Convert 14 days to weeks:
const result = daysToWeeks(14)
//=> 2
// It uses floor rounding:
const result = daysToWeeks(13)
//=> 1
