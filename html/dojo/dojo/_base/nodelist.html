dojo/_base/NodeList

Summary

This module extends dojo/NodeList with the legacy connect(), coords(), blur(), focus(), change(), click(), error(), keydown(), keypress(), keyup(), load(), mousedown(), mouseenter(), mouseleave(), mousemove(), mouseout(), mouseover(), mouseup(), and submit() methods.

See the dojo/_base/NodeList reference documentation for more information.
