dojo/request/iframe

Summary

Sends a request using an iframe element with the given URL and options.

Usage

iframe(url,options);
Parameter Type Description
url String

URL to request
options dojo/request/iframe.__Options
Optional

Options for the request.

Returns: dojo/request.__Promise
