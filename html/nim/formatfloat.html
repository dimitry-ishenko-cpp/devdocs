This module implements formatting floats as strings.

Imports

dragonbox, schubfach

Procs

proc addFloat(result: var string; x: float | float32) {.inline.}
Converts float to its string representation and appends it to result.

Example:

var
  s = "foo:"
  b = 45.67
s.addFloat(45.67)
assert s == "foo:45.67"
