k1

The k1 attribute defines one of the values to be used within the arithmetic operation of the <feComposite> filter primitive.

The pixel composition is computed using the following formula:

js
result = k1 * i1 * i2 + k2 * i1 + k3 * i2 + k4;

You can use this attribute with the following SVG elements:

  • <feComposite>

Example

html
<svg viewBox="0 0 420 200" xmlns="http://www.w3.org/2000/svg">
  <filter id="composite1" x="0" y="0" width="100%" height="100%">
    <feComposite
      in2="SourceGraphic"
      operator="arithmetic"
      k1="1"
      k2="0"
      k3="0"
      k4="0" />
  </filter>
  <filter id="composite2" x="0" y="0" width="100%" height="100%">
    <feComposite
      in2="SourceGraphic"
      operator="arithmetic"
      k1="10"
      k2="0"
      k3="0"
      k4="0" />
  </filter>

  <image
    href="mdn.svg"
    x="0"
    y="0"
    width="200"
    height="200"
    style="filter: url(#composite1);" />
  <image
    href="mdn.svg"
    x="0"
    y="0"
    width="200"
    height="200"
    style="filter: url(#composite2); transform: translateX(220px);" />
</svg>
