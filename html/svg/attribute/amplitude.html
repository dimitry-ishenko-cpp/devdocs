amplitude

The amplitude attribute controls the amplitude of the gamma function of a component transfer element when its type attribute is gamma.

You can use this attribute with the following SVG elements:

  • <feFuncA>
  • <feFuncB>
  • <feFuncG>
  • <feFuncR>
