The widths attribute indicates a list of range values, each followed by one or more glyph widths.

You can use this attribute with the following SVG elements:

  • <font-face>

Usage notes

<number>

This value is a comma-separated list of UCS range values as defined in ISO 10646, each followed by one or more glyph widths.

If this attribute is used, the units-per-em attribute must also be used.

If the range is omitted, a range of U+0-7FFFFFFF is assumed which covers all characters and their glyphs. If not enough glyph widths are given, the last in the list is replicated to cover that range. If too many widths are provided, the extras are ignored.
