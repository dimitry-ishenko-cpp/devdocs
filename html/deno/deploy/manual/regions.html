Regions

Deno Deploy deploys your code throughout the world. Each new request is served from the closest region to your user. Deploy is presently located in the following regions:

  1. Tokyo (asia-northeast1)
  2. Mumbai (asia-south1)
  3. Singapore (asia-southeast1)
  4. Sydney (australia-southeast1)
  5. Tel Aviv (me-west1)
  6. London (europe-west2)
  7. Frankfurt (europe-west3)
  8. Netherlands (europe-west4)
  9. Sao Paolo (southamerica-east1)
  10. North Virginia (us-east4)
  11. Texas (us-south1)
  12. California (us-west1)

We will update the list as we add more regions.

Code is deployed to all regions and is served from the region closest to the end user to minimize latency. It is not currently possible to restrict the regions in which your code is deployed.
