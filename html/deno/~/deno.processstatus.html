Deno.ProcessStatus

The status resolved from the .status() method of a Deno.Process instance.

If success is true, then code will be 0, but if success is false, the sub-process exit code will be set in code.

type ProcessStatus = {
success: true; 


code: 0; 


signal?: undefined; 
} | {
success: false; 


code: number; 


signal?: number; 
};

Type

success: true;
