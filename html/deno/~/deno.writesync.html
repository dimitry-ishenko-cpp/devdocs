Deno.writeSync

Synchronously write to the resource ID (rid) the contents of the array buffer (data).

Returns the number of bytes written. This function is one of the lowest level APIs and most users should not work with this directly, but rather use writeAllSync() from std/streams/conversion.ts instead.

It is not guaranteed that the full buffer will be written in a single call.

const encoder = new TextEncoder();
const data = encoder.encode("Hello world");
const file = Deno.openSync("/foo/bar.txt", { write: true });
const bytesWritten = Deno.writeSync(file.rid, data); // 11
Deno.close(file.rid);
