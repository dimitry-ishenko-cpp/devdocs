Deno.StartTlsOptions

interface StartTlsOptions {
caCerts?: string[];

hostname?: string;

}

Properties

caCerts?: string[]

A list of root certificates that will be used in addition to the default root certificates to verify the peer's certificate.

Must be in PEM format.

hostname?: string

A literal IP address or host name that can be resolved to an IP address. If not specified, defaults to 127.0.0.1.
