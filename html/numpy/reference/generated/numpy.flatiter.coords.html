numpy.flatiter.coords

attribute

flatiter.coords

An N-dimensional tuple of current coordinates.

Examples

>>> x = np.arange(6).reshape(2, 3)
>>> fl = x.flat
>>> fl.coords
(0, 0)
>>> next(fl)
0
>>> fl.coords
(0, 1)
