SparseQR module

Provides QR decomposition for sparse matrices.

This module provides a simplicial version of the left-looking Sparse QR decomposition. The columns of the input matrix should be reordered to limit the fill-in during the decomposition. Built-in methods (COLAMD, AMD) or external methods (METIS) can be used to this end. See the OrderingMethods module for the list of built-in and external ordering methods.
