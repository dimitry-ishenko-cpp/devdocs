Backing Up and Restoring Databases

There are a number of ways to backup a MariaDB server.

Title Description
Backup and Restore Overview Backing up and restoring MariaDB.
Replication as a Backup Solution Replication can be used to support the backup strategy.
mariadb-dump Dump a database or a collection of databases in a portable format.
Mariabackup Physical backups, supports Data-at-Rest and InnoDB compression.
Backup and Restore via dbForge Studio The fastest and easiest way to perform these operations with MariaDB databases.
mariadb-hotcopy Fast backup program on local machine. Deprecated.
