Random Query Generator Tests

PQuery is currently in development that does random query generation.

The documentation for the Random Query Generator is available here.

The latest RQG used by MariaDB can be found at:

  • https://github.com/elenst/rqg/tree/master
  • https://github.com/MariaDB/randgen/tree/master

The list of automatic tests is located in the buildbot.conf file in the mariadb.org-tools repository.
