Roles

Roles bundle privileges together to ease account management

Title Description
Roles Overview Bundling privileges together.
CREATE ROLE Add new roles.
DROP ROLE Drop a role.
CURRENT_ROLE Current role name.
SET ROLE Enable a role.
SET DEFAULT ROLE Sets a default role for a specified (or current) user.
GRANT Create accounts and set privileges or roles.
REVOKE Remove privileges or roles.
mysql.roles_mapping Table MariaDB roles information.
Information Schema APPLICABLE_ROLES Table Roles available to be used.
Information Schema ENABLED_ROLES Table Enabled roles for the current session.
SecuRich Library of security-related stored procedures.
