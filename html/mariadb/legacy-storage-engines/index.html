Legacy Storage Engines

The following storage engines are no longer maintained.

Title Description
BDB BDB storage engine
Cassandra Storage Engine A storage engine interface to Cassandra.
EXAMPLE Storage Engine Legacy storage engine designed to assist developers with writing new storage engines
FEDERATED Storage Engine A legacy storage engine no longer in active development.
IBMDB2I Storage Engine Legacy storage engine designed to allow storage on DB2 tables
NDB in MariaDB MySQL Cluster has been removed from MariaDB
PBXT PBXT is a general purpose transactional storage engine. It's however not maintained anymore.
TokuDB For use in high-performance and write-intensive environments.
