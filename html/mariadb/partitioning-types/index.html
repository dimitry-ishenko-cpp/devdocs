Partitioning Types

Title Description
Partitioning Types Overview A partition type determines how a partitioned table rows are distributed across partitions
LIST Partitioning Type LIST partitioning is used to assign each partition a list of values
RANGE Partitioning Type The RANGE partitioning type is used to assign each partition a range of values.
HASH Partitioning Type Form of partitioning in which the server takes care of the partition in which to place the data.
KEY Partitioning Type Used to have the server assign the distribution of rows across partitions.
RANGE COLUMNS and LIST COLUMNS Partitioning Types Used to assign each partition a range or a list of values
