Virtual Machine to Test the Cassandra Storage Engine

CassandraSE is no longer actively being developed and has been removed in MariaDB 10.6. See MDEV-23024.

Julien Duponchelle has made a virtual machine available for testing the Cassandra storage engine. Find out more at Cassandra MariaDB VirtualBox.

The virtual machine is based on:

  • Ubuntu 12.04
  • DataStax Cassandra
  • MariaDB 5.5.27 + Cassandra

Vagrant is used for setup. Full instructions are at Julien's website.
