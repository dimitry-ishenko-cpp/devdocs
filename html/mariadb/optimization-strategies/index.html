Optimization Strategies

Various optimization strategies used by the query optimizer.

Title Description
DuplicateWeedout Strategy DuplicateWeedout is an execution strategy for Semi-join subqueries.
FirstMatch Strategy FirstMatch is an execution strategy for Semi-join subqueries
LooseScan Strategy LooseScan is an execution strategy for Semi-join subqueries
Semi-join Materialization Strategy Semi-join Materialization is a subquery materialization used for Semi-join subqueries.
Improvements to ORDER BY Optimization Several Improvements to the ORDER BY Optimizer in Version 10.1 of MariaDB.
