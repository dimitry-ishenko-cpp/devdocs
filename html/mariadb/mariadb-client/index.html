mariadb Client

The mariadb command-line client.

Previously, the client was called mysql which, from MariaDB 10.5, is still a symlink.

Title Description
mariadb Command-Line Client mysql is a simple SQL shell with GNU readline capabilities.
Delimiters How to change the delimiter for the mariadb client.
mysql Command-line Client Symlink or old name for mariadb, the command-line client.
