Backup, Restore and Import Clients

Clients for taking backups or importing/restoring data

Title Description
Mariabackup Physical backups, supports Data-at-Rest and InnoDB compression.
mariadb-dump Dump a database or a collection of databases in a portable format.
mariadb-hotcopy Fast backup program on local machine. Deprecated.
mariadb-import Loads tables from text files in various formats.
Backup/Restore + Data Export/Import via dbForge Studio The fastest and easiest way to perform these operations with MariaDB databases.
