>>

Syntax

value1 >> value2

Description

Converts a longlong (BIGINT) number (value1) to binary and shifts value2 units to the right.

Examples

SELECT 4 >> 2;
+--------+
| 4 >> 2 |
+--------+
|      1 |
+--------+
