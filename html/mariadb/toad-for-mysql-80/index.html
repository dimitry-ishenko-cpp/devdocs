TOAD for MySQL

Features

  • Version control integration.
  • Macro record and playback.
  • Database browser.
  • Code snippet editor.
  • Security manager.
  • SQL editor.
  • Fast, multi-tabbed schema browser.
  • DB extract, compare-and-search utility.
  • Import/export utility.
