Implemented Interfaces

GtkMediaControls implements GtkAccessible, GtkBuildable and GtkConstraintTarget.

Includes

#include <gtk/gtk.h>

Description

GtkMediaControls is a widget to show controls for a GtkMediaStream and giving users a way to use it.
