dart:ffi

nullptr top-level property

Pointer<Never> nullptr
final

Represents a pointer into the native C memory corresponding to 'NULL', e.g. a pointer with address 0.

Implementation

final Pointer<Never> nullptr = Pointer.fromAddress(0);
