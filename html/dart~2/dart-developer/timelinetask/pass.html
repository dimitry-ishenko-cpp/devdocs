pass method

int pass()

Retrieve the TimelineTask's task id. Will throw an exception if the stack is not empty.

Implementation

int pass() {
  if (_stack.length > 0) {
    throw new StateError(
        'You cannot pass a TimelineTask without finishing all started '
        'operations');
  }
  int r = _taskId;
  return r;
}
