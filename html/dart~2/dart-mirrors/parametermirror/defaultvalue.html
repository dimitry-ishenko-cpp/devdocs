dart:mirrors

defaultValue property

InstanceMirror? defaultValue

Returns the default value of an optional parameter.

Returns an InstanceMirror on the (compile-time constant) default value for an optional parameter. If no default value is declared, it defaults to null and a mirror of null is returned.

Returns null for a required parameter.
