dart:indexed_db

supported property

bool supported

Checks to see if Indexed DB is supported on the current platform.

Implementation

static bool get supported {
  return JS(
      'bool',
      '!!(window.indexedDB || '
          'window.webkitIndexedDB || '
          'window.mozIndexedDB)');
}
