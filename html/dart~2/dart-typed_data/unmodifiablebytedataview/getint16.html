int getInt16(
  1. int byteOffset,
  2. [Endian endian = Endian.big]
)
override

Returns the (possibly negative) integer represented by the two bytes at the specified byteOffset in this object, in two's complement binary form.

The return value will be between -215 and 215 - 1, inclusive.

The byteOffset must be non-negative, and byteOffset + 2 must be less than or equal to the length of this object.

Implementation

int getInt16(int byteOffset, [Endian endian = Endian.big]) =>
    _data.getInt16(byteOffset, endian);
