void forEach(
  1. void action(
    1. E element

Invokes action on each element of this iterable in iteration order.

Example:

final numbers = <int>[1, 2, 6, 7];
numbers.forEach(print);
// 1
// 2
// 6
// 7

Implementation

void forEach(void action(E element)) {
  for (E element in this) action(element);
}
