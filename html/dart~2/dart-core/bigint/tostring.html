toString method

String toString()
override

Returns a String-representation of this integer.

The returned string is parsable by parse. For any BigInt i, it is guaranteed that i == BigInt.parse(i.toString()).

Example:

var bigNumber = BigInt.parse('100000000000000000000000');
print(bigNumber.toString()); // "100000000000000000000000"
