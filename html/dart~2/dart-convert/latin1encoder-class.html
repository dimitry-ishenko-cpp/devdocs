dart:convert

Latin1Encoder class

This class converts strings of only ISO Latin-1 characters to bytes.

Example:

final latin1Encoder = latin1.encoder;

const sample = 'àáâãäå';
final encoded = latin1Encoder.convert(sample);
print(encoded); // [224, 225, 226, 227, 228, 229]
