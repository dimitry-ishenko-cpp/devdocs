remove method

E remove()

Removes this entry from any chain of entries it is part of.

Returns its element value.

Implementation

E remove() {
  _previousLink?._nextLink = _nextLink;
  _nextLink?._previousLink = _previousLink;
  _nextLink = null;
  _previousLink = null;
  return element;
}
