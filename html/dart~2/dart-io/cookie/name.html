dart:io

name property

String name
read / write

The name of the cookie.

Must be a token as specified in RFC 6265.

The allowed characters in a token are the visible ASCII characters, U+0021 (!) through U+007E (~), except the separator characters: (, ), <, >, @, ,, ;, :, \, ", /, [, ], ?, =, {, and }.
