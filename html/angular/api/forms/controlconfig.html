ControlConfig

type-alias

ControlConfig is a tuple containing a value of type T, plus optional validators and async validators.

type ControlConfig<T> = [
    T | FormControlState<T>,
    (ValidatorFn | (ValidatorFn[]))?,
    (AsyncValidatorFn | AsyncValidatorFn[])?
];
