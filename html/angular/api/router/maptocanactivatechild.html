mapToCanActivateChild

function

Maps an array of injectable classes with canActivateChild functions to an array of equivalent CanActivateChildFn for use in a Route definition.

See more...

mapToCanActivateChild(providers: Type<{ canActivateChild: CanActivateChildFn; }>[]): CanActivateChildFn[]

Parameters
providers Type<{ canActivateChild: CanActivateChildFn; }>[]
Returns

CanActivateChildFn[]

See also

  • Route

Description

Usage 

@Injectable({providedIn: 'root'})
export class AdminGuard {
  canActivate() {
    return true;
  }
}

const route: Route = {
  path: 'admin',
  canActivate: mapToCanActivate([AdminGuard]),
};
