mapToCanActivate

function

Maps an array of injectable classes with canActivate functions to an array of equivalent CanActivateFn for use in a Route definition.

See more...

mapToCanActivate(providers: Type<{ canActivate: CanActivateFn; }>[]): CanActivateFn[]

Parameters
providers Type<{ canActivate: CanActivateFn; }>[]
Returns

CanActivateFn[]

See also

  • Route

Description

Usage 

@Injectable({providedIn: 'root'})
export class AdminGuard {
  canActivate() {
    return true;
  }
}

const route: Route = {
  path: 'admin',
  canActivate: mapToCanActivate([AdminGuard]),
};
