VersionEvent

type-alias

A union of all event types that can be emitted by api/service-worker/SwUpdate#versionUpdates.

type VersionEvent = VersionDetectedEvent | VersionInstallationFailedEvent | VersionReadyEvent | NoNewVersionDetectedEvent;
