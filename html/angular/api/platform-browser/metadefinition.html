MetaDefinition

type-alias

Represents the attributes of an HTML <meta> element. The element itself is represented by the internal HTMLMetaElement.

type MetaDefinition = {
    charset?: string;
    content?: string;
    httpEquiv?: string;
    id?: string;
    itemprop?: string;
    name?: string;
    property?: string;
    scheme?: string;
    url?: string;
} & {
    [prop: string]: string;
};
