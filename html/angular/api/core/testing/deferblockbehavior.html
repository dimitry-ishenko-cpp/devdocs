DeferBlockBehavior

enum developer preview

Options for configuring defer blocks behavior.

enum DeferBlockBehavior {
  Manual
  Playthrough
}

Members

Member Description
Manual

Manual triggering mode for defer blocks. Provides control over when defer blocks render and which state they render. This is the default behavior in test environments.
Playthrough

Playthrough mode for defer blocks. This mode behaves like defer blocks would in a browser.
