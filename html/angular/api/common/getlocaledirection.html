getLocaleDirection

function

Retrieves the writing direction of a specified locale

getLocaleDirection(locale: string): 'ltr' | 'rtl'

Parameters
locale string

A locale code for the locale format rules to use.
Returns

'ltr' | 'rtl': 'rtl' or 'ltr'
