5.1.27 TYPE as an alias for PRINT

For compatibility, GNU Fortran will interpret TYPE statements as PRINT statements with the flag -fdec. With this flag asserted, the following two examples are equivalent: 

TYPE *, 'hello world'
PRINT *, 'hello world'
