Description:

TINY(X) returns the smallest positive (non zero) number in the model of the type of X.

Standard:

Fortran 90 and later

Class:

Inquiry function

Syntax:

RESULT = TINY(X)

Arguments:
X Shall be of type REAL.
Return value:

The return value is of the same type and kind as X

Example:

See HUGE for an example.
