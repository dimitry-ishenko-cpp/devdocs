8.217 PRECISION — Decimal precision of a real kind

Description:

PRECISION(X) returns the decimal precision in the model of the type of X.

Standard:

Fortran 90 and later

Class:

Inquiry function

Syntax:

RESULT = PRECISION(X)

Arguments:
X Shall be of type REAL or COMPLEX. It may be scalar or valued.
Return value:

The return value is of type INTEGER and of the default integer kind.

Example:
program prec_and_range
  real(kind=4) :: x(2)
  complex(kind=8) :: y

  print *, precision(x), range(x)
  print *, precision(y), range(y)
end program prec_and_range
See also:

SELECTED_REAL_KIND — Choose real kind, RANGE — Decimal exponent range
