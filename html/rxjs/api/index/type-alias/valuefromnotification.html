ValueFromNotification

type-alias stable

Gets the value type from an ObservableNotification, if possible.

type ValueFromNotification<T> = T extends {
    kind: 'N' | 'E' | 'C';
} ? T extends NextNotification<any> ? T extends {
    value: infer V;
} ? V : undefined : never : never;
