Static Properties

Property Type Description
frameTimeFactor 10

The number of virtual time units each character in a marble diagram represents. If the test scheduler is being used in "run mode", via the run method, this is temporarily set to 1 for the duration of the run block, then set back to whatever value it was.

run<T>(callback: (helpers: RunHelpers) => T): T

The run method performs the test in 'run mode' - in which schedulers used within the test automatically delegate to the TestScheduler. That is, in 'run mode' there is no need to explicitly pass a TestScheduler instance to observable creators or operators.
