pandas.Timestamp.date

Timestamp.date()

Return date object with same year, month and day.

Examples

>>> ts = pd.Timestamp('2023-01-01 10:00:00.00')
>>> ts
Timestamp('2023-01-01 10:00:00')
>>> ts.date()
datetime.date(2023, 1, 1)
