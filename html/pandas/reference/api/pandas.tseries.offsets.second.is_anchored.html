pandas.tseries.offsets.Second.is_anchored

Second.is_anchored()

Return False.

Deprecated since version 2.2.0: is_anchored is deprecated and will be removed in a future version. Use False instead.

Examples

>>> pd.offsets.Hour().is_anchored()
False
>>> pd.offsets.Hour(2).is_anchored()
False
