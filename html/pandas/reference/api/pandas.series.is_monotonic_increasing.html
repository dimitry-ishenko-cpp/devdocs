pandas.Series.is_monotonic_increasing

propertySeries.is_monotonic_increasing[source]

Return boolean if values in the object are monotonically increasing.

Returns:
bool

Examples

>>> s = pd.Series([1, 2, 2])
>>> s.is_monotonic_increasing
True

>>> s = pd.Series([3, 2, 1])
>>> s.is_monotonic_increasing
False
