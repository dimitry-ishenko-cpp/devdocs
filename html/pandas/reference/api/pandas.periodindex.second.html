pandas.PeriodIndex.second

propertyPeriodIndex.second[source]

The second of the period.

Examples

>>> idx = pd.PeriodIndex(["2023-01-01 10:00:30",
...                       "2023-01-01 10:00:31"], freq='s')
>>> idx.second
Index([30, 31], dtype='int64')
