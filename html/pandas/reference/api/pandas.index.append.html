pandas.Index.append

Index.append(other)[source]

Append a collection of Index options together.

Parameters:
other:Index or list/tuple of indices
Returns:
Index

Examples

>>> idx = pd.Index([1, 2, 3])
>>> idx.append(pd.Index([4]))
Index([1, 2, 3, 4], dtype='int64')
