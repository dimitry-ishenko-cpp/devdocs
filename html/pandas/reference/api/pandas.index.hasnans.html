pandas.Index.hasnans

Index.hasnans[source]

Return True if there are any NaNs.

Enables various performance speedups.

Returns:
bool

Examples

>>> s = pd.Series([1, 2, 3], index=['a', 'b', None])
>>> s
a    1
b    2
None 3
dtype: int64
>>> s.index.hasnans
True
