pandas.Series.sparse.sp_values

Series.sparse.sp_values[source]

An ndarray containing the non- fill_value values.

Examples

>>> from pandas.arrays import SparseArray
>>> s = SparseArray([0, 0, 1, 0, 2], fill_value=0)
>>> s.sp_values
array([1, 2])
