pandas.Timedelta.view

Timedelta.view(dtype)

Array view compatibility.

Parameters:
dtype:str or dtype

The dtype to view the underlying data as.

Examples

>>> td = pd.Timedelta('3D')
>>> td
Timedelta('3 days 00:00:00')
>>> td.view(int)
259200000000000
