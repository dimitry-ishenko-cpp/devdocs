pandas.Timedelta.days

Timedelta.days

Returns the days of the timedelta.

Returns:
int

Examples

>>> td = pd.Timedelta(1, "d")
>>> td.days
1

>>> td = pd.Timedelta('4 min 3 us 42 ns')
>>> td.days
0
