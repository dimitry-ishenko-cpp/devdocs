pandas.Index.dtype

Index.dtype[source]

Return the dtype object of the underlying data.

Examples

>>> idx = pd.Index([1, 2, 3])
>>> idx
Index([1, 2, 3], dtype='int64')
>>> idx.dtype
dtype('int64')
