pandas.PeriodIndex.hour

propertyPeriodIndex.hour[source]

The hour of the period.

Examples

>>> idx = pd.PeriodIndex(["2023-01-01 10:00", "2023-01-01 11:00"], freq='h')
>>> idx.hour
Index([10, 11], dtype='int64')
