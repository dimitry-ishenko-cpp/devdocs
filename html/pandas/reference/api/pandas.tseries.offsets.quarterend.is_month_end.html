pandas.tseries.offsets.QuarterEnd.is_month_end

QuarterEnd.is_month_end(ts)

Return boolean whether a timestamp occurs on the month end.

Examples

>>> ts = pd.Timestamp(2022, 1, 1)
>>> freq = pd.offsets.Hour(5)
>>> freq.is_month_end(ts)
False
