pandas.Timestamp.ctime

Timestamp.ctime()

Return ctime() style string.

Examples

>>> ts = pd.Timestamp('2023-01-01 10:00:00.00')
>>> ts
Timestamp('2023-01-01 10:00:00')
>>> ts.ctime()
'Sun Jan  1 10:00:00 2023'
