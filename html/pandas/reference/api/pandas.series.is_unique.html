pandas.Series.is_unique

propertySeries.is_unique[source]

Return boolean if values in the object are unique.

Returns:
bool

Examples

>>> s = pd.Series([1, 2, 3])
>>> s.is_unique
True

>>> s = pd.Series([1, 2, 3, 1])
>>> s.is_unique
False
