pandas.DatetimeIndex.tz

propertyDatetimeIndex.tz[source]

Return the timezone.

Returns
datetime.tzinfo, pytz.tzinfo.BaseTZInfo, dateutil.tz.tz.tzfile, or None

Returns None when the array is tz-naive.
