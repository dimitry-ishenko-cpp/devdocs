pandas.Series.hasnans

propertySeries.hasnans[source]

Return True if there are any NaNs.

Enables various performance speedups.

Returns:
bool

Examples

>>> s = pd.Series([1, 2, 3, None])
>>> s
0    1.0
1    2.0
2    3.0
3    NaN
dtype: float64
>>> s.hasnans
True
