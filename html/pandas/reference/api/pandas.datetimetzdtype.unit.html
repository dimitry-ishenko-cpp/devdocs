pandas.DatetimeTZDtype.unit

propertyDatetimeTZDtype.unit[source]

The precision of the datetime data.

Examples

>>> from zoneinfo import ZoneInfo
>>> dtype = pd.DatetimeTZDtype(tz=ZoneInfo('America/Los_Angeles'))
>>> dtype.unit
'ns'
