pandas.Interval.closed

Interval.closed

String describing the inclusive side the intervals.

Either left, right, both or neither.

Examples

>>> interval = pd.Interval(left=1, right=2, closed='left')
>>> interval
Interval(1, 2, closed='left')
>>> interval.closed
'left'
