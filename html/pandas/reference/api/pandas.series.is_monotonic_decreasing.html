pandas.Series.is_monotonic_decreasing

propertySeries.is_monotonic_decreasing[source]

Return boolean if values in the object are monotonically decreasing.

Returns:
bool

Examples

>>> s = pd.Series([3, 2, 2, 1])
>>> s.is_monotonic_decreasing
True

>>> s = pd.Series([1, 2, 3])
>>> s.is_monotonic_decreasing
False
