pandas.IntervalDtype.subtype

propertyIntervalDtype.subtype[source]

The dtype of the Interval bounds.

Examples

>>> dtype = pd.IntervalDtype(subtype='int64', closed='both')
>>> dtype.subtype
dtype('int64')
