pandas.Index.shape

propertyIndex.shape[source]

Return a tuple of the shape of the underlying data.

Examples

>>> idx = pd.Index([1, 2, 3])
>>> idx
Index([1, 2, 3], dtype='int64')
>>> idx.shape
(3,)
