pandas.PeriodIndex.month

propertyPeriodIndex.month[source]

The month as January=1, December=12.

Examples

>>> idx = pd.PeriodIndex(["2023-01", "2023-02", "2023-03"], freq="M")
>>> idx.month
Index([1, 2, 3], dtype='int64')
