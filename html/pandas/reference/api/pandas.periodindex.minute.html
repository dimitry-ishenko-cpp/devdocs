pandas.PeriodIndex.minute

propertyPeriodIndex.minute[source]

The minute of the period.

Examples

>>> idx = pd.PeriodIndex(["2023-01-01 10:30:00",
...                       "2023-01-01 11:50:00"], freq='min')
>>> idx.minute
Index([30, 50], dtype='int64')
