pandas.Timedelta.total_seconds

Timedelta.total_seconds()

Total seconds in the duration.

Examples

>>> td = pd.Timedelta('1min')
>>> td
Timedelta('0 days 00:01:00')
>>> td.total_seconds()
60.0
