pandas.PeriodIndex.week

propertyPeriodIndex.week[source]

The week ordinal of the year.

Examples

>>> idx = pd.PeriodIndex(["2023-01", "2023-02", "2023-03"], freq="M")
>>> idx.week  # It can be written `weekofyear`
Index([5, 9, 13], dtype='int64')
