All Cookstyle Cops


The department is: Chef/Modernize

The full name of the cop is: Chef/Modernize/ExecuteTzUtil


Enabled by default Supports autocorrection Target Chef Version
Enabled No 14.6+

Instead of using the execute or powershell_script resources to run the tzutil command, use Chef Infra Clientâ€™s built-in timezone resource which is available in Chef Infra Client 14.6 and later.

Examples

incorrect

execute 'set tz' do
  command 'tzutil.exe /s UTC'
end

execute 'tzutil /s UTC'

powershell_script 'set windows timezone' do
  code "tzutil.exe /s UTC"
  not_if { shell_out('tzutil.exe /g').stdout.include?('UTC') }
end

correct

timezone 'UTC'
