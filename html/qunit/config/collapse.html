QUnit.config.collapse

version added: 1.0.0

Description

In the HTML Reporter, collapse the details of failing tests after the first one.

type boolean
default true

By default, QUnitâ€™s HTML Reporter collapses consecutive failing tests showing only the details for the first failed test. The other tests can be expanded manually with a single click on the test title.

Set this option to false to expand the details for all failing tests.
