ACL

ACL USERS
Syntax
ACL USERS
Available since:
6.0.0
Time complexity:
O(N). Where N is the number of configured users.
ACL categories:
@admin, @slow, @dangerous,

The command shows a list of all the usernames of the currently configured users in the Redis ACL system.

Return

An array of strings.

Examples

> ACL USERS
1) "anna"
2) "antirez"
3) "default"
