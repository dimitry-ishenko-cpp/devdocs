CLIENT

CLIENT UNPAUSE
Syntax
CLIENT UNPAUSE
Available since:
6.2.0
Time complexity:
O(N) Where N is the number of paused clients
ACL categories:
@admin, @slow, @dangerous, @connection,

CLIENT UNPAUSE is used to resume command processing for all clients that were paused by CLIENT PAUSE.

Return

Simple string reply: The command returns OK
