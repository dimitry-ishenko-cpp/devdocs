TOUCH

TOUCH
Syntax
TOUCH key [key ...]
Available since:
3.2.1
Time complexity:
O(N) where N is the number of keys that will be touched.
ACL categories:
@keyspace, @read, @fast,

Alters the last access time of a key(s). A key is ignored if it does not exist.

Return

Integer reply: The number of keys that were touched.

Examples

SET key1 "Hello"
SET key2 "World"
TOUCH key1 key2
