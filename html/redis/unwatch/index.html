UNWATCH

UNWATCH
Syntax
UNWATCH
Available since:
2.2.0
Time complexity:
O(1)
ACL categories:
@fast, @transaction,

Flushes all the previously watched keys for a transaction.

If you call EXEC or DISCARD, there's no need to manually call UNWATCH.

Return

Simple string reply: always OK.
