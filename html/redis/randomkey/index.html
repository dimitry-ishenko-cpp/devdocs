RANDOMKEY
Syntax
RANDOMKEY
Available since:
1.0.0
Time complexity:
O(1)
ACL categories:
@keyspace, @read, @slow,

Return a random key from the currently selected database.

Return

Bulk string reply: the random key, or nil when the database is empty.
