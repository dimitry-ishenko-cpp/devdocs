COMMAND

COMMAND COUNT
Syntax
COMMAND COUNT
Available since:
2.8.13
Time complexity:
O(1)
ACL categories:
@slow, @connection,

Returns Integer reply of number of total commands in this Redis server.

Return

Integer reply: number of commands returned by COMMAND

Examples

COMMAND COUNT
