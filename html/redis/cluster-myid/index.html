CLUSTER

CLUSTER MYID
Syntax
CLUSTER MYID
Available since:
3.0.0
Time complexity:
O(1)
ACL categories:
@slow,

Returns the node's id.

The CLUSTER MYID command returns the unique, auto-generated identifier that is associated with the connected cluster node.

Return

Bulk string reply: The node id.
