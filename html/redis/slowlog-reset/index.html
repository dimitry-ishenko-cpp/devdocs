SLOWLOG

SLOWLOG RESET
Syntax
SLOWLOG RESET
Available since:
2.2.12
Time complexity:
O(N) where N is the number of entries in the slowlog
ACL categories:
@admin, @slow, @dangerous,

This command resets the slow log, clearing all entries in it.

Once deleted the information is lost forever.

@reply

Simple string reply: OK
