CMS.INFO

CMS.INFO
Syntax
CMS.INFO key
Available in:
Redis Stack / Bloom 2.0.0
Time complexity:
O(1)

Returns width, depth and total count of the sketch.

Parameters:

  • key: The name of the sketch.

Return

Array reply with information of the filter.

Examples

redis> CMS.INFO test
 1) width
 2) (integer) 2000
 3) depth
 4) (integer) 7
 5) count
 6) (integer) 0
