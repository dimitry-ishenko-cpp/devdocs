GRAPH.CONFIG

GRAPH.CONFIG GET
Syntax
GRAPH.CONFIG GET name
Available in:
Redis Stack / Graph 2.2.11
Time complexity:

Retrieves the current value of a RedisGraph configuration parameter.

RedisGraph configuration parameters are detailed here.

* can be used to retrieve the value of all RedisGraph configuration parameters.
