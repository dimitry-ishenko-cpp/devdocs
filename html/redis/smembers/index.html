SMEMBERS

SMEMBERS
Syntax
SMEMBERS key
Available since:
1.0.0
Time complexity:
O(N) where N is the set cardinality.
ACL categories:
@read, @set, @slow,

Returns all the members of the set value stored at key.

This has the same effect as running SINTER with one argument key.

Return

Array reply: all elements of the set.

Examples

SADD myset "Hello"
SADD myset "World"
SMEMBERS myset
