Module: @ember-data/store

RecordDataStoreWrapper provides encapsulated API access to the minimal subset of the Store's functionality that cache (RecordData) implementations should interact with. It is provided to the Store's createRecordDataFor hook.

Cache implementations should not need more than this API provides.

This class cannot be directly instantiated.
