The DataAdapter helps a data persistence library interface with tools that debug Ember such as the Ember Inspector for Chrome and Firefox.

This class will be extended by a persistence library which will override some of the methods with library-specific code.

The methods likely to be overridden are:

  • getFilters
  • detect
  • columnsForType
  • getRecords
  • getRecordColumnValues
  • getRecordKeywords
  • getRecordFilterValues
  • getRecordColor

The adapter will need to be registered in the application's container as dataAdapter:main.

Example:

Application.initializer({
  name: "data-adapter",

  initialize: function(application) {
    application.register('data-adapter:main', DS.DataAdapter);
  }
});
