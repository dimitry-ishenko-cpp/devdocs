Available since version 2.0.10
Source Code https://github.com/yiisoft/yii2/blob/master/framework/mutex/OracleMutex.php

OracleMutex implements mutex "lock" mechanism via Oracle locks.

Application configuration example:

[
    'components' => [
        'db' => [
            'class' => 'yii\db\Connection',
            'dsn' => 'oci:dbname=LOCAL_XE',
             ...
        ]
        'mutex' => [
            'class' => 'yii\mutex\OracleMutex',
            'lockMode' => 'NL_MODE',
            'releaseOnCommit' => true,
             ...
        ],
    ],
]

See also:

  • http://docs.oracle.com/cd/B19306_01/appdev.102/b14258/d_lock.htm
