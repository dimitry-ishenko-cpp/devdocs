Available since version 2.0
Source Code https://github.com/yiisoft/yii2/blob/master/framework/base/InlineAction.php

InlineAction represents an action that is defined as a controller method.

The name of the controller method is available via $actionMethod which is set by the $controller who creates this action.

For more details and usage information on InlineAction, see the guide article on actions.
