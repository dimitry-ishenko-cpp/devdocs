Available since version 2.0.8
Source Code https://github.com/yiisoft/yii2/blob/master/framework/mutex/PgsqlMutex.php

PgsqlMutex implements mutex "lock" mechanism via PgSQL locks.

Application configuration example:

[
    'components' => [
        'db' => [
            'class' => 'yii\db\Connection',
            'dsn' => 'pgsql:host=127.0.0.1;dbname=demo',
        ]
        'mutex' => [
            'class' => 'yii\mutex\PgsqlMutex',
        ],
    ],
]

See also yii\mutex\Mutex.
