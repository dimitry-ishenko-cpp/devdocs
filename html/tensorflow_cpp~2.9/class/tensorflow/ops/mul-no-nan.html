tensorflow::ops::MulNoNan

#include <math_ops.h>

Returns x * y element-wise.

Summary

Returns zero if y is zero, even if x if infinite or NaN.

NOTE: MulNoNan supports broadcasting. More about broadcasting here

Args:

  • scope: A Scope object

Returns:

  • Output: The z tensor.
