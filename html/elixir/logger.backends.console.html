Source Logger.Backends.Console

This module is deprecated. Use LoggerBackends.Console from :logger_backends dependency.

A logger backend that logs messages by printing them to the console.

This backend was typically configured as config :logger, :console, but it has been deprecated in favor of :default_handler and :default_formatter. However, for backwards compatibility, you can still add it as:

config :logger, :backends, [Logger.Backends.Console]

However, if you plan to continue using Logger backends in the long term, consider using the :logger_backends project.
