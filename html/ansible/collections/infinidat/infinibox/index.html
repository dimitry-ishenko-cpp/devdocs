Infinidat.Infinibox

Collection version 1.4.5

  • Description
  • Plugin Index

Description

A set of Ansible modules that allow simple, but powerful, idempotent interactions with Infinidat Infiniboxes.

Authors:

  • Infinidat <partners.infi@infinidat.com>
  • David Ohlemacher
  • Gregory Shulov

Supported ansible-core versions:

  • 2.14.0 or newer
  • Issue Tracker
  • Repository (Sources)

Plugin Index

These are the plugins in the infinidat.infinibox collection:

Modules

  • infini_certificate module – Create (present state) or clear (absent state) SSL certificates on Infinibox
  • infini_cluster module – Create, Delete and Modify Host Cluster on Infinibox
  • infini_config module – Modify config on Infinibox
  • infini_event module – Create custom events on Infinibox
  • infini_export module – Create, Delete or Modify NFS Exports on Infinibox
  • infini_export_client module – Create, Delete or Modify NFS Client(s) for existing exports on Infinibox
  • infini_fibre_channel_switch module – Manage Infinibox FC switch names
  • infini_fs module – Create, Delete or Modify filesystems on Infinibox
  • infini_host module – Create, Delete or Modify Hosts on Infinibox
  • infini_infinimetrics module – Create (present state) or remove (absent state) an Infinibox registration on an Infinimetrics.
  • infini_map module – Create and Delete mapping of a volume to a host or cluster on Infinibox
  • infini_metadata module – Create, Delete or Modify metadata on Infinibox
  • infini_network_space module – Create, Delete and Modify network spaces on Infinibox
  • infini_notification_rule module – Config notification rules
  • infini_notification_target module – Config notification target
  • infini_pool module – Create, Delete and Modify Pools on Infinibox
  • infini_port module – Add and Delete fiber channel and iSCSI ports to a host on Infinibox
  • infini_sso module – Configures or queries SSO on Infinibox
  • infini_user module – Create, Delete and Modify a User on Infinibox
  • infini_users_repository module – Create, Delete or Modify respositories of users that can log on to an Infinibox
  • infini_vol module – Create, Delete or Modify volumes on Infinibox
