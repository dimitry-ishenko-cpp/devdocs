Plugin Index

These are the plugins in the ovirt.ovirt collection:

Modules

  • ovirt_affinity_group module – Module to manage affinity groups in oVirt/RHV
  • ovirt_affinity_label module – Module to manage affinity labels in oVirt/RHV
  • ovirt_affinity_label_info module – Retrieve information about one or more oVirt/RHV affinity labels
  • ovirt_api_info module – Retrieve information about the oVirt/RHV API
  • ovirt_auth module – Module to manage authentication to oVirt/RHV
  • ovirt_cluster module – Module to manage clusters in oVirt/RHV
  • ovirt_cluster_info module – Retrieve information about one or more oVirt/RHV clusters
  • ovirt_datacenter module – Module to manage data centers in oVirt/RHV
  • ovirt_datacenter_info module – Retrieve information about one or more oVirt/RHV datacenters
  • ovirt_disk module – Module to manage Virtual Machine and floating disks in oVirt/RHV
  • ovirt_disk_info module – Retrieve information about one or more oVirt/RHV disks
  • ovirt_disk_profile module – Module to manage storage domain disk profiles in ovirt
  • ovirt_event module – Create or delete an event in oVirt/RHV
  • ovirt_event_info module – This module can be used to retrieve information about one or more oVirt/RHV events
  • ovirt_external_provider module – Module to manage external providers in oVirt/RHV
  • ovirt_external_provider_info module – Retrieve information about one or more oVirt/RHV external providers
  • ovirt_group module – Module to manage groups in oVirt/RHV
  • ovirt_group_info module – Retrieve information about one or more oVirt/RHV groups
  • ovirt_host module – Module to manage hosts in oVirt/RHV
  • ovirt_host_info module – Retrieve information about one or more oVirt/RHV hosts
  • ovirt_host_network module – Module to manage host networks in oVirt/RHV
  • ovirt_host_pm module – Module to manage power management of hosts in oVirt/RHV
  • ovirt_host_storage_info module – Retrieve information about one or more oVirt/RHV HostStorages (applicable only for block storage)
  • ovirt_instance_type module – Module to manage Instance Types in oVirt/RHV
  • ovirt_job module – Module to manage jobs in oVirt/RHV
  • ovirt_mac_pool module – Module to manage MAC pools in oVirt/RHV
  • ovirt_network module – Module to manage logical networks in oVirt/RHV
  • ovirt_network_info module – Retrieve information about one or more oVirt/RHV networks
  • ovirt_nic module – Module to manage network interfaces of Virtual Machines in oVirt/RHV
  • ovirt_nic_info module – Retrieve information about one or more oVirt/RHV virtual machine network interfaces
  • ovirt_permission module – Module to manage permissions of users/groups in oVirt/RHV
  • ovirt_permission_info module – Retrieve information about one or more oVirt/RHV permissions
  • ovirt_qos module – Module to manage QoS entries in ovirt
  • ovirt_quota module – Module to manage datacenter quotas in oVirt/RHV
  • ovirt_quota_info module – Retrieve information about one or more oVirt/RHV quotas
  • ovirt_role module – Module to manage roles in oVirt/RHV
  • ovirt_scheduling_policy_info module – Retrieve information about one or more oVirt scheduling policies
  • ovirt_snapshot module – Module to manage Virtual Machine Snapshots in oVirt/RHV
  • ovirt_snapshot_info module – Retrieve information about one or more oVirt/RHV virtual machine snapshots
  • ovirt_storage_connection module – Module to manage storage connections in oVirt
  • ovirt_storage_domain module – Module to manage storage domains in oVirt/RHV
  • ovirt_storage_domain_info module – Retrieve information about one or more oVirt/RHV storage domains
  • ovirt_storage_template_info module – Retrieve information about one or more oVirt/RHV templates relate to a storage domain.
  • ovirt_storage_vm_info module – Retrieve information about one or more oVirt/RHV virtual machines relate to a storage domain.
  • ovirt_system_option_info module – Retrieve information about one oVirt/RHV system options.
  • ovirt_tag module – Module to manage tags in oVirt/RHV
  • ovirt_tag_info module – Retrieve information about one or more oVirt/RHV tags
  • ovirt_template module – Module to manage virtual machine templates in oVirt/RHV
  • ovirt_template_info module – Retrieve information about one or more oVirt/RHV templates
  • ovirt_user module – Module to manage users in oVirt/RHV
  • ovirt_user_info module – Retrieve information about one or more oVirt/RHV users
  • ovirt_vm module – Module to manage Virtual Machines in oVirt/RHV
  • ovirt_vm_info module – Retrieve information about one or more oVirt/RHV virtual machines
  • ovirt_vm_os_info module – Retrieve information on all supported oVirt/RHV operating systems
  • ovirt_vmpool module – Module to manage VM pools in oVirt/RHV
  • ovirt_vmpool_info module – Retrieve information about one or more oVirt/RHV vmpools
  • ovirt_vnic_profile module – Module to manage vNIC profile of network in oVirt/RHV
  • ovirt_vnic_profile_info module – Retrieve information about one or more oVirt/RHV vnic profiles
