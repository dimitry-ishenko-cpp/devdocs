Plugin Index

These are the plugins in the vmware.vmware collection:

Modules

  • appliance_info module – Gather appliance information
  • cluster module – Manage VMware vSphere clusters
  • cluster_drs module – Manage Distributed Resource Scheduler (DRS) on VMware vSphere clusters
  • cluster_vcls module – Manage the vCLS (vSphere Cluster Services) VM disk placement for this cluster.
  • content_template module – Manage template in content library from virtual machine.
  • folder_template_from_vm module – Create a template in a local VCenter folder from an existing VM
  • guest_info module – Gather guest information
  • license_info module – Fetch VMware vCenter license keys
  • vcsa_settings module – Configure vCenter Server Appliance settings
  • vm_list_group_by_clusters_info module – Returns information about the virtual machines grouping by clusters and folders
  • vm_portgroup_info module – Returns information about the portgroups of virtual machines
