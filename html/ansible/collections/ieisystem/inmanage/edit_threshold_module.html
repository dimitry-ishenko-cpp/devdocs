Synopsis

  • Set threshold information on ieisystem Server.

Requirements

The below requirements are needed on the host that executes this module.

  • Python 3.7+
  • inManage

Examples

- name: Threshold test
  hosts: inmanage
  connection: local
  gather_facts: false
  vars:
    inmanage:
      host: "{{ ansible_ssh_host }}"
      username: "{{ username }}"
      password: "{{ password }}"

  tasks:

  - name: "Set threshold information"
    ieisystem.inmanage.edit_threshold:
      name: "GPU1_Temp"
      uc: 94
      unc: 92
      provider: "{{ inmanage }}"
