Plugin Index

These are the plugins in the vultr.cloud collection:

Modules

  • account_info module – Get information about the Vultr account
  • bare_metal module – Manages bare metal machines on Vultr.
  • block_storage module – Manages block storage volumes on Vultr
  • block_storage_info module – Get information about the Vultr block storage
  • dns_domain module – Manages DNS domains on Vultr
  • dns_domain_info module – Gather information about the Vultr DNS domains
  • dns_record module – Manages DNS records on Vultr
  • firewall_group module – Manages firewall groups on Vultr
  • firewall_group_info module – Gather information about the Vultr firewall groups
  • firewall_rule module – Manages firewall rules on Vultr
  • firewall_rule_info module – Gather information about the Vultr firewall rules
  • instance module – Manages server instances on Vultr.
  • instance_info module – Get information about the Vultr instances
  • object_storage module – Manages object storages on Vultr
  • os_info module – Get information about the Vultr operation systems
  • plan_info module – Gather information about the Vultr plans
  • plan_metal_info module – Gather information about the Vultr bare metal plans
  • region_info module – Gather information about the Vultr regions
  • reserved_ip module – Manages reserved IPs on Vultr
  • snapshot module – Manages snapshots on Vultr
  • snapshot_info module – Gather information about the Vultr snapshots
  • ssh_key module – Manages ssh keys on Vultr.
  • ssh_key_info module – Get information about the Vultr SSH keys
  • startup_script module – Manages startup scripts on Vultr
  • startup_script_info module – Gather information about the Vultr startup scripts
  • user module – Manages users on Vultr
  • user_info module – Get information about the Vultr users
  • vpc module – Manages VPCs on Vultr
  • vpc2 module – Manages VPCs 2.0 on Vultr
  • vpc2_info module – Gather information about the Vultr VPCs 2.0
  • vpc_info module – Gather information about the Vultr VPCs

Inventory Plugins

  • vultr inventory – Retrieves list of instances via Vultr v2 API
