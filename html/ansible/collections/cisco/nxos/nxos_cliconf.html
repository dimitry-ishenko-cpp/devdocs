Synopsis

  • This nxos plugin provides low level abstraction apis for sending and receiving CLI commands from Cisco NX-OS network devices.

Parameters

Parameter

Comments

config_commands

list / elements=string

added in cisco.nxos 2.0.0

Specifies a list of commands that can make configuration changes to the target device.

When `ansible_network_single_user_mode` is enabled, if a command sent to the device is present in this list, the existing cache is invalidated.

Default: []

Configuration:

  • Variable: ansible_nxos_config_commands

Authors

  • Ansible Networking Team (@ansible-network)

Hint

Configuration entries for each entry type have a low to high priority order. For example, a variable that is lower in the list will override a variable that is higher up.
