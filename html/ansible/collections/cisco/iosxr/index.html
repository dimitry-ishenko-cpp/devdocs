Description

Ansible Network Collection for Cisco IOSXR devices.

Author:

  • Ansible Network Community (ansible-network)

Supported ansible-core versions:

  • 2.14.0 or newer
  • Issue Tracker
  • Repository (Sources)

Plugin Index

These are the plugins in the cisco.iosxr collection:

Modules

  • iosxr_acl_interfaces module – Resource module to configure ACL interfaces.
  • iosxr_acls module – Resource module to configure ACLs.
  • iosxr_banner module – Module to configure multiline banners.
  • iosxr_bgp_address_family module – Resource module to configure BGP Address family.
  • iosxr_bgp_global module – Resource module to configure BGP.
  • iosxr_bgp_neighbor_address_family module – Resource module to configure BGP Neighbor Address family.
  • iosxr_bgp_templates module – Manages BGP templates resource module.
  • iosxr_command module – Module to run commands on remote devices.
  • iosxr_config module – Module to manage configuration sections.
  • iosxr_facts module – Module to collect facts from remote devices.
  • iosxr_hostname module – Resource module to configure hostname.
  • iosxr_interfaces module – Resource module to configure interfaces.
  • iosxr_l2_interfaces module – Resource Module to configure L2 interfaces.
  • iosxr_l3_interfaces module – Resource module to configure L3 interfaces.
  • iosxr_lacp module – Resource module to configure LACP.
  • iosxr_lacp_interfaces module – Resource module to configure LACP interfaces.
  • iosxr_lag_interfaces module – Resource module to configure LAG interfaces.
  • iosxr_lldp_global module – Resource module to configure LLDP.
  • iosxr_lldp_interfaces module – Resource module to configure LLDP interfaces.
  • iosxr_logging_global module – Resource module to configure logging.
  • iosxr_netconf module – Configures NetConf sub-system service on Cisco IOS-XR devices
  • iosxr_ntp_global module – Resource module to configure NTP.
  • iosxr_ospf_interfaces module – Resource module to configure OSPF interfaces.
  • iosxr_ospfv2 module – Resource module to configure OSPFv2.
  • iosxr_ospfv3 module – Resource module to configure OSPFv3.
  • iosxr_ping module – Tests reachability using ping from IOSXR switch.
  • iosxr_prefix_lists module – Resource module to configure prefix lists.
  • iosxr_snmp_server module – Resource module to configure snmp server.
  • iosxr_static_routes module – Resource module to configure static routes.
  • iosxr_system module – Module to manage the system attributes.
  • iosxr_user module – Module to manage the aggregates of local users.

Cliconf Plugins

  • iosxr cliconf – Use iosxr cliconf to run command on Cisco IOS XR platform

Netconf Plugins

  • iosxr netconf – Use iosxr netconf plugin to run netconf commands on Cisco IOSXR platform
