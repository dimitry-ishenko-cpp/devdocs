kaytus.ksmanage.add_ldisk module – Create logical disk

Note

This module is part of the kaytus.ksmanage collection (version 1.2.2).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install kaytus.ksmanage. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: kaytus.ksmanage.add_ldisk.

New in kaytus.ksmanage 1.0.0

Synopsis

  • Create logical disk on kaytus Server.

Requirements

The below requirements are needed on the host that executes this module.

  • Python 3.7+
  • inManage

Examples

- name: Add ldisk test
  hosts: ksmanage
  connection: local
  gather_facts: false
  vars:
    ksmanage:
      host: "{{ ansible_ssh_host }}"
      username: "{{ username }}"
      password: "{{ password }}"

  tasks:

  - name: "Add LSI ldisk"
    kaytus.ksmanage.add_ldisk:
      ctrl_id: 0
      level: 1
      size: 1
      access: 1
      r: 1
      w: 1
      io: 1
      cache: 1
      init: 2
      select: 10
      slot: 0, 1
      provider: "{{ ksmanage }}"

  - name: "Add PMC ldisk"
    kaytus.ksmanage.add_ldisk:
      ctrl_id: 0
      level: 1
      size: 1
      accelerator: 1
      slot: 0, 1
      vname: "test"
      provider: "{{ ksmanage }}"

  - name: "Add MV ldisk"
    kaytus.ksmanage.add_ldisk:
      ctrl_id: 0
      size: 1
      vname: "test"
      provider: "{{ ksmanage }}"
