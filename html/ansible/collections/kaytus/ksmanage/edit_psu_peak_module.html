Synopsis

  • Set psu peak information on kaytus Server.

Requirements

The below requirements are needed on the host that executes this module.

  • Python 3.7+
  • inManage

Examples

- name: Psu peak test
  hosts: ksmanage
  connection: local
  gather_facts: false
  vars:
    ksmanage:
      host: "{{ ansible_ssh_host }}"
      username: "{{ username }}"
      password: "{{ password }}"

  tasks:

  - name: "Set psu peak information"
    kaytus.ksmanage.edit_psu_peak:
      status: "disable"
      provider: "{{ ksmanage }}"

  - name: "Set psu peak information"
    kaytus.ksmanage.edit_psu_peak:
      status: "enable"
      time: 10
      provider: "{{ ksmanage }}"
