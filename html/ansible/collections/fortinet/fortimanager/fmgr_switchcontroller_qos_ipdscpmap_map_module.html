fortinet.fortimanager.fmgr_switchcontroller_qos_ipdscpmap_map module – Maps between IP-DSCP value to COS queue.

Note

This module is part of the fortinet.fortimanager collection (version 2.7.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install fortinet.fortimanager.

To use it in a playbook, specify: fortinet.fortimanager.fmgr_switchcontroller_qos_ipdscpmap_map.

New in fortinet.fortimanager 2.0.0
