Plugin Index

These are the plugins in the community.hrobot collection:

Modules

  • boot module – Set boot configuration
  • failover_ip module – Manage Hetzner’s failover IPs
  • failover_ip_info module – Retrieve information on Hetzner’s failover IPs
  • firewall module – Manage Hetzner’s dedicated server firewall
  • firewall_info module – Manage Hetzner’s dedicated server firewall
  • reset module – Reset a dedicated server
  • reverse_dns module – Set or remove reverse DNS entry for IP
  • server module – Update server information
  • server_info module – Query information on one or more servers
  • ssh_key module – Add, remove or update SSH key
  • ssh_key_info module – Query information on SSH keys
  • v_switch module – Manage Hetzner’s vSwitch

Inventory Plugins

  • robot inventory – Hetzner Robot inventory source
