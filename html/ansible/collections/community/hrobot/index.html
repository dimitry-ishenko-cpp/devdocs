Community.Hrobot

Collection version 1.2.0

Plugin Index

These are the plugins in the community.hrobot collection

Inventory Plugins

  • robot – Hetzner Robot inventory source

Modules

  • boot – Set boot configuration
  • failover_ip – Manage Hetzner’s failover IPs
  • failover_ip_info – Retrieve information on Hetzner’s failover IPs
  • firewall – Manage Hetzner’s dedicated server firewall
  • firewall_info – Manage Hetzner’s dedicated server firewall
  • reset – Reset a dedicated server
  • reverse_dns – Set or remove reverse DNS entry for IP
  • server – Update server information
  • server_info – Query information on one or more servers
  • ssh_key – Add, remove or update SSH key
  • ssh_key_info – Query information on SSH keys
