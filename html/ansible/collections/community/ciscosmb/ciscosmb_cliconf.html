Note

This cliconf plugin is part of the community.ciscosmb collection (version 1.0.9).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install community.ciscosmb.

To use it in a playbook, specify: community.ciscosmb.ciscosmb.

Synopsis

  • This ciscosmb plugin provides low level abstraction apis for sending and receiving CLI commands from Cisco SMB network devices.
