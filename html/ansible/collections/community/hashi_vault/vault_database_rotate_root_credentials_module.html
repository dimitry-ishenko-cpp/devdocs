community.hashi_vault.vault_database_rotate_root_credentials module – Rotates the root credentials stored for the database connection. This user must have permissions to update its own password.

Note

This module is part of the community.hashi_vault collection (version 6.2.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install community.hashi_vault. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: community.hashi_vault.vault_database_rotate_root_credentials.

New in community.hashi_vault 6.2.0

Synopsis

  • Trigger root credential rotation of a Database Connection identified by its connection_name.

Requirements

The below requirements are needed on the host that executes this module.

  • hvac (Python library) >= 2.0.0
  • For detailed requirements, see the collection requirements page.
