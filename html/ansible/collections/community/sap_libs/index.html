Plugin Index

These are the plugins in the community.sap_libs collection:

Modules

  • sap_company module – This module will manage a company entities in a SAP S4HANA environment
  • sap_control_exec module – Ansible Module to execute SAPCONTROL
  • sap_hdbsql module – Ansible Module to execute SQL on SAP HANA
  • sap_pyrfc module – Ansible Module for use of SAP PyRFC to execute SAP RFCs (Remote Function Calls) to SAP remote-enabled function modules
  • sap_snote module – This module will upload and (de)implements SNOTES in a SAP S4HANA environment.
  • sap_system_facts module – Gathers SAP facts in a host
  • sap_task_list_execute module – Perform SAP Task list execution
  • sap_user module – This module will manage a user entities in a SAP S4/HANA environment
  • sapcar_extract module – Manages SAP SAPCAR archives
