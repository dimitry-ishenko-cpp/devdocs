Note

This connection plugin is part of the community.general collection (version 9.5.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install community.general.

To use it in a playbook, specify: community.general.qubes.

Synopsis

  • Run commands or put/fetch files to an existing Qubes AppVM using qubes tools.

Parameters

Parameter

Comments

remote_addr

string

VM name.

Default: "inventory_hostname"

Configuration:

  • Variable: ansible_host

remote_user

string

The user to execute as inside the VM.

Default: "The I(user) account as default in Qubes OS."

Configuration:

  • Variable: ansible_user
