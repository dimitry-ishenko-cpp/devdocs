Plugin Index

These are the plugins in the community.okd collection:

Modules

  • k8s module – Manage OpenShift objects
  • openshift_adm_groups_sync module – Sync OpenShift Groups with records from an external provider.
  • openshift_adm_migrate_template_instances module – Update TemplateInstances to point to the latest group-version-kinds
  • openshift_adm_prune_auth module – Removes references to the specified roles, clusterroles, users, and groups
  • openshift_adm_prune_builds module – Prune old completed and failed builds
  • openshift_adm_prune_deployments module – Remove old completed and failed deployment configs
  • openshift_adm_prune_images module – Remove unreferenced images
  • openshift_auth module – Authenticate to OpenShift clusters which require an explicit login step
  • openshift_build module – Start a new build or Cancel running, pending, or new builds.
  • openshift_import_image module – Import the latest image information from a tag in a container image registry.
  • openshift_process module – Process an OpenShift template.openshift.io/v1 Template
  • openshift_registry_info module – Display information about the integrated registry.
  • openshift_route module – Expose a Service as an OpenShift Route.

Connection Plugins

  • oc connection – Execute tasks in pods running on OpenShift.

Inventory Plugins

  • openshift inventory – OpenShift inventory source DEPRECATED
