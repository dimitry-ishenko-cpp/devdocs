Plugin Index

These are the plugins in the community.rabbitmq collection:

Modules

  • rabbitmq_binding module – Manage rabbitMQ bindings
  • rabbitmq_exchange module – Manage rabbitMQ exchange
  • rabbitmq_feature_flag module – Enables feature flag
  • rabbitmq_global_parameter module – Manage RabbitMQ global parameters
  • rabbitmq_parameter module – Manage RabbitMQ parameters
  • rabbitmq_plugin module – Manage RabbitMQ plugins
  • rabbitmq_policy module – Manage the state of policies in RabbitMQ
  • rabbitmq_publish module – Publish a message to a RabbitMQ queue.
  • rabbitmq_queue module – Manage rabbitMQ queues
  • rabbitmq_upgrade module – Execute rabbitmq-upgrade commands
  • rabbitmq_user module – Manage RabbitMQ users
  • rabbitmq_user_limits module – Manage RabbitMQ user limits
  • rabbitmq_vhost module – Manage the state of a virtual host in RabbitMQ
  • rabbitmq_vhost_limits module – Manage the state of virtual host limits in RabbitMQ

Lookup Plugins

  • rabbitmq lookup – Retrieve messages from an AMQP/AMQPS RabbitMQ queue.
