Containers.Podman

Collection version 1.8.1

Plugin Index

These are the plugins in the containers.podman collection

Connection Plugins

  • buildah – Interact with an existing buildah container
  • podman – Interact with an existing podman container

Modules

  • podman_container – Manage podman containers
  • podman_container_info – Gather facts about containers using podman
  • podman_containers – Manage podman containers in a batch
  • podman_export – Export a podman container
  • podman_image – Pull images for use by podman
  • podman_image_info – Gather info about images using podman
  • podman_import – Import Podman container from a tar file.
  • podman_load – Load image from a tar file.
  • podman_login – Login to a container registry using podman
  • podman_login_info – Return the logged-in user if any for a given registry
  • podman_logout – Log out of a container registry using podman
  • podman_network – Manage podman networks
  • podman_network_info – Gather info about podman networks
  • podman_play – Play kubernetes YAML file using podman
  • podman_pod – Manage Podman pods
  • podman_pod_info – Gather info about podman pods
  • podman_save – Saves podman image to tar file
  • podman_secret – Manage podman secrets
  • podman_volume – Manage Podman volumes
  • podman_volume_info – Gather info about podman volumes
