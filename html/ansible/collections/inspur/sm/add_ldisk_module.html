inspur.sm.add_ldisk module – Create logical disk.

Note

This module is part of the inspur.sm collection (version 2.3.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install inspur.sm.

To use it in a playbook, specify: inspur.sm.add_ldisk.

New in inspur.sm 0.1.0

Examples

- name: Add ldisk test
  hosts: ism
  connection: local
  gather_facts: no
  vars:
    ism:
      host: "{{ ansible_ssh_host }}"
      username: "{{ username }}"
      password: "{{ password }}"

  tasks:

  - name: "Show pdisk information"
    inspur.sm.add_ldisk:
      info: "show"
      provider: "{{ ism }}"

  - name: "Add ldisk"
    inspur.sm.add_ldisk:
      ctrl_id: 0
      level: 1
      size: 1
      access: 1
      r: 1
      w: 1
      io: 1
      cache: 1
      init: 2
      select: 10
      slot: 0,1
      provider: "{{ ism }}"

  - name: "Add PMC  ldisk"
    inspur.sm.add_ldisk:
      ctrl_id: 0
      level: 1
      size: 1
      accelerator: 1
      vname: "test"
      provider: "{{ ism }}"
