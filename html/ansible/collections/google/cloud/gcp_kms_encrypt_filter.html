google.cloud.gcp_kms_encrypt filter – Support auth tokens as a Filter plugin

Note

This filter plugin is part of the google.cloud collection (version 1.4.1).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install google.cloud.

To use it in a playbook, specify: google.cloud.gcp_kms_encrypt.

Synopsis

  • Enables the ‘accesstoken’ authentication choice.

Authors

  • Eric Anderson

Hint

Configuration entries for each entry type have a low to high priority order. For example, a variable that is lower in the list will override a variable that is higher up.
