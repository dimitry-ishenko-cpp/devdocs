Description

This collection contains Ansible modules to change objects in Icinga 2 using the director API.

Authors:

  • Sebastian Gumprich <sebastian.gumprich@telekom.de>
  • Lars Krahl <lars.krahl@telekom.de>

Supported ansible-core versions:

  • 2.14.0 or newer
  • Issue Tracker
  • Repository (Sources)
