Dellemc.Os10

Collection version 1.1.1

Plugin Index

These are the plugins in the dellemc.os10 collection

Cliconf Plugins

  • os10 – Use os10 cliconf to run command on Dell OS10 platform

Modules

  • base_xml_to_dict – Operations for show command output convertion from xml to json format.
  • bgp_validate – Validate the bgp neighbor state,raise error if it is not in established state
  • mtu_validate – Validate the MTU value for lldp neighbors
  • os10_command – Run commands on devices running Dell EMC SmartFabric OS10
  • os10_config – Manage Dell EMC SmartFabric OS10 configuration sections
  • os10_facts – Collect facts from devices running Dell EMC SmartFabric OS10
  • show_system_network_summary – Operations for show_system_network output in json/yaml format.
  • vlt_validate – Validate the vlt info, raise an error if peer is not in up state
  • wiring_validate – Validate the wiring based on the planned wiring details
