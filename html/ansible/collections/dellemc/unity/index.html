Plugin Index

These are the plugins in the dellemc.unity collection:

Modules

  • cifsserver module – Manage CIFS server on Unity storage system
  • consistencygroup module – Manage consistency groups on Unity storage system
  • filesystem module – Manage filesystem on Unity storage system
  • filesystem_snapshot module – Manage filesystem snapshot on the Unity storage system
  • host module – Manage Host operations on Unity
  • info module – Gathering information about Unity
  • interface module – Manage Interfaces on Unity storage system
  • nasserver module – Manage NAS servers on Unity storage system
  • nfs module – Manage NFS export on Unity storage system
  • nfsserver module – Manage NFS server on Unity storage system
  • replication_session module – Manage replication session on Unity storage system
  • smbshare module – Manage SMB shares on Unity storage system
  • snapshot module – Manage snapshots on the Unity storage system
  • snapshotschedule module – Manage snapshot schedules on Unity storage system
  • storagepool module – Manage storage pool on Unity
  • tree_quota module – Manage quota tree on the Unity storage system
  • user_quota module – Manage user quota on the Unity storage system
  • volume module – Manage volume on Unity storage system
