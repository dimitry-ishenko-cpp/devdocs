Description

Dell OpenManage Ansible Modules allows data center and IT administrators to use RedHat Ansible to automate and orchestrate the configuration, deployment, and update of Dell PowerEdge Servers and modular infrastructure by leveraging the management automation capabilities in-built into the Integrated Dell Remote Access Controller (iDRAC), OpenManage Enterprise and OpenManage Enterprise Modular.

Role Index

  • idrac_bios role – Modify and clear BIOS attributes, and reset BIOS settings
  • idrac_boot role – Configure the boot order settings
  • idrac_certificate role – This role allows to generate certificate signing request, import, and export certificates on iDRAC
  • idrac_export_server_config_profile role – Role to export iDRAC Server Configuration Profile (SCP)
  • idrac_firmware role – Firmware update from a repository on a network share (CIFS, NFS, HTTP, HTTPS, FTP)
  • idrac_gather_facts role – Role to get the facts from the iDRAC Server
  • idrac_import_server_config_profile role – Import iDRAC Server Configuration Profile (SCP)
  • idrac_job_queue role – Role to manage the iDRAC lifecycle controller job queue.
  • idrac_os_deployment role – Role to deploy operating system on the iDRAC servers

  • idrac_storage_controller role – Configures the physical disk, virtual disk, and storage controller settings
  • idrac_user role – Role to manage local users for iDRAC.
