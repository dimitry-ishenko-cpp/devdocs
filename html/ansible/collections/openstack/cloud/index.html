Openstack.Cloud

Collection version 1.5.1

Plugin Index

These are the plugins in the openstack.cloud collection

Inventory Plugins

  • openstack – OpenStack inventory source

Modules

  • address_scope – Create or delete address scopes from OpenStack
  • auth – Retrieve an auth token
  • baremetal_inspect – Explicitly triggers baremetal node introspection in ironic.
  • baremetal_node – Create/Delete Bare Metal Resources from OpenStack
  • baremetal_node_action – Activate/Deactivate Bare Metal Resources from OpenStack
  • catalog_service – Manage OpenStack Identity services
  • coe_cluster – Add/Remove COE cluster from OpenStack Cloud
  • coe_cluster_template – Add/Remove COE cluster template from OpenStack Cloud
  • compute_flavor – Manage OpenStack compute flavors
  • compute_flavor_info – Retrieve information about one or more flavors
  • config – Get OpenStack Client config
  • dns_zone – Manage OpenStack DNS zones
  • dns_zone_info – Getting information about dns zones
  • endpoint – Manage OpenStack Identity service endpoints
  • federation_idp – manage a federation Identity Provider
  • federation_idp_info – Get the information about the available federation identity providers
  • federation_mapping – Manage a federation mapping
  • federation_mapping_info – Get the information about the available federation mappings
  • floating_ip – Add/Remove floating IP from an instance
  • floating_ip_info – Get information about floating ips
  • group_assignment – Associate OpenStack Identity users and groups
  • host_aggregate – Manage OpenStack host aggregates
  • identity_domain – Manage OpenStack Identity Domains
  • identity_domain_info – Retrieve information about one or more OpenStack domains
  • identity_group – Manage OpenStack Identity Groups
  • identity_group_info – Retrieve info about one or more OpenStack groups
  • identity_role – Manage OpenStack Identity Roles
  • identity_role_info – Retrive information about roles
  • identity_user – Manage OpenStack Identity Users
  • identity_user_info – Retrieve information about one or more OpenStack users
  • image – Add/Delete images from OpenStack Cloud
  • image_info – Retrieve information about an image within OpenStack.
  • keypair – Add/Delete a keypair from OpenStack
  • keypair_info – Get information about keypairs from OpenStack
  • keystone_federation_protocol – manage a federation Protocol
  • keystone_federation_protocol_info – get information about federation Protocols
  • lb_health_monitor – Add/Delete a health m nonitor to a pool in the load balancing service from OpenStack Cloud
  • lb_listener – Add/Delete a listener for a load balancer from OpenStack Cloud
  • lb_member – Add/Delete a member for a pool in load balancer from OpenStack Cloud
  • lb_pool – Add/Delete a pool in the load balancing service from OpenStack Cloud
  • loadbalancer – Add/Delete load balancer from OpenStack Cloud
  • network – Creates/removes networks from OpenStack
  • networks_info – Retrieve information about one or more OpenStack networks.
  • object – Create or Delete objects and containers from OpenStack
  • object_container – Manage Swift container.
  • port – Add/Update/Delete ports from an OpenStack cloud.
  • port_info – Retrieve information about ports within OpenStack.
  • project – Manage OpenStack Projects
  • project_access – Manage OpenStack compute flavors access
  • project_info – Retrieve information about one or more OpenStack projects
  • quota – Manage OpenStack Quotas
  • recordset – Manage OpenStack DNS recordsets
  • role_assignment – Associate OpenStack Identity users and roles
  • router – Create or delete routers from OpenStack
  • routers_info – Retrieve information about one or more OpenStack routers.
  • security_group – Add/Delete security groups from an OpenStack cloud.
  • security_group_info – Lists security groups
  • security_group_rule – Add/Delete rule from an existing security group
  • security_group_rule_info – Querying security group rules
  • server – Create/Delete Compute Instances from OpenStack
  • server_action – Perform actions on Compute Instances from OpenStack
  • server_group – Manage OpenStack server groups
  • server_info – Retrieve information about one or more compute instances
  • server_metadata – Add/Update/Delete Metadata in Compute Instances from OpenStack
  • server_volume – Attach/Detach Volumes from OpenStack VM’s
  • stack – Add/Remove Heat Stack
  • stack_info – Retrive information about Heat stacks
  • subnet – Add/Remove subnet to an OpenStack network
  • subnets_info – Retrieve information about one or more OpenStack subnets.
  • volume – Create/Delete Cinder Volumes
  • volume_backup – Add/Delete Volume backup
  • volume_backup_info – Get Backups
  • volume_info – Retrive information about volumes
  • volume_snapshot – Create/Delete Cinder Volume Snapshots
  • volume_snapshot_info – Get volume snapshots
