6.62.24 Nvidia PTX Built-in Functions

These built-in functions are available for the Nvidia PTX target:

Built-in Function: unsigned int __builtin_nvptx_brev (unsigned int x)

Reverse the bit order of a 32-bit unsigned integer.

Built-in Function: unsigned long long __builtin_nvptx_brevll (unsigned long long x)

Reverse the bit order of a 64-bit unsigned integer.
