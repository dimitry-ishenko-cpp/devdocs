ReQL command: append

Command syntax

array.append(value) → array

Description

Append a value to an array.

Example: Retrieve Iron Man’s equipment list with the addition of some new boots.

r.table('marvel').get('IronMan')('equipment').append('newBoots').run(conn, callback)
