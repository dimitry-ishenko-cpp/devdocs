\nocite

Synopsis: 

\nocite{keys}

Produces no output but writes keys to the auxiliary file jobname.aux (see Jobname).

The mandatory argument keys is a comma-separated list of one or more citation keys (see \bibitem). This information is used by BibTeX to include these works in your reference list even though you have not explicitly cited them (see \cite).
